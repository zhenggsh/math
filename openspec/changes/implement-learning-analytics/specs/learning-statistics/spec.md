## ADDED Requirements

### Requirement: Learning count statistics
The system SHALL provide learning session counts by time period.

#### Scenario: Total sessions
- **WHEN** viewing analytics
- **THEN** the system SHALL show total number of learning sessions

#### Scenario: Sessions by date
- **WHEN** viewing analytics
- **THEN** the system SHALL show session count grouped by date

### Requirement: Learning duration statistics
The system SHALL provide total learning time statistics.

#### Scenario: Total duration
- **WHEN** viewing analytics
- **THEN** the system SHALL show cumulative learning time

#### Scenario: Duration by knowledge point
- **WHEN** viewing analytics
- **THEN** the system SHALL show time spent on each knowledge point

#### Scenario: Duration trend
- **WHEN** viewing analytics
- **THEN** the system SHALL show daily/weekly learning time trends
