#### 知识点 1.1.1：集合的含义

**1. 定义**

一般地，把研究对象统称为元素，把一些元素组成的总体叫做集合。集合是数学中最基本的概念之一，是不定义的原概念。集合论由德国数学家康托尔（Georg Cantor）于 19 世纪末创立，是现代数学的基石。

**2. 重要性级别**：A

**3. 特性**

元素用小写字母表示，集合用大写字母表示；集合与元素的关系用 $\in$（属于）或 $\notin$（不属于）表示；常见数集有 $\mathbb{N}$（自然数集）、$\mathbb{Z}$（整数集）、$\mathbb{Q}$（有理数集）、$\mathbb{R}$（实数集）。

**4. 运算方式**

判断对象 $a$ 是否属于集合 $A$，结果为一个命题：$a \in A$（真）或 $a \notin A$（真）。集合的确定性要求对于任意对象，属于或不属于集合 $A$ 二者必居其一且仅居其一。

**5. 定理**

- **集合确定性的逻辑表述**：设 $A$ 是一个集合，$x$ 为任意对象，则 $x \in A$ 与 $x \notin A$ 有且仅有一个成立。
- 前提：集合 $A$ 已明确定义。
- 结论：元素与集合的关系是二值的。
- 使用场合：判断元素归属、证明集合相等。

**6. 公式**

$$a \in A \quad \text{或} \quad a \notin A$$

常见数集符号：

$$\mathbb{N}^* = \mathbb{N}_+ = \{1, 2, 3, \dots\}$$
$$\mathbb{Z} = \{\dots, -2, -1, 0, 1, 2, \dots\}$$
$$\mathbb{Q} = \left\{x \mid x = \frac{p}{q},\ p \in \mathbb{Z},\ q \in \mathbb{N}^*\right\}$$
$$\mathbb{R} = \text{全体实数}$$

**7. 二级结论**

- 空集 $\varnothing$ 是不含任何元素的集合，它是任何集合的子集。
- 数集之间有包含关系：$\mathbb{N} \subsetneqq \mathbb{Z} \subsetneqq \mathbb{Q} \subsetneqq \mathbb{R}$。

#### 知识点 1.1.2：元素的基本特性

**1. 定义**

集合中的元素具有三大特性：确定性（任何一个对象是否属于集合是明确的）、互异性（任意两个元素都不同）、无序性（元素没有顺序之分）。

**2. 重要性级别**：A

**3. 特性**

确定性：$a \in A$ 与 $a \notin A$ 必居其一；互异性：$\{a, b, c\}$ 隐含 $a \neq b$ 且 $b \neq c$ 且 $a \neq c$；无序性：$\{1, 2, 3\} = \{3, 2, 1\}$。

**4. 运算方式**

利用互异性检验集合表示的正确性：若集合表示中出现重复元素，则必须化简为唯一形式。例如 $\{1, 1, 2\} = \{1, 2\}$。利用确定性判断边界：给定不等式解集时，端点是否包含需明确判断。

**5. 定理**

- **互异性定理**：若集合 $A$ 用列举法表示为 $A = \{a_1, a_2, \dots, a_n\}$，则当 $i \neq j$ 时必有 $a_i \neq a_j$。
- 前提：集合采用列举法表示。
- 结论：元素两两不同。
- 使用场合：根据集合元素个数求参数范围（如 $\{1, a, a^2\}$ 要求 $a \neq 1$ 且 $a^2 \neq 1$ 且 $a^2 \neq a$）。

**6. 公式**

$$\{a, b\} = \{b, a\} \quad \text{（无序性）}$$
$$\{a, a, b\} = \{a, b\} \quad \text{（互异性）}$$

**7. 二级结论**

- 已知集合 $A = \{x, x^2, 1\}$ 含 3 个元素，则需满足 $x \neq 1$，$x^2 \neq 1$ 且 $x \neq x^2$，即 $x \neq \pm 1$ 且 $x \neq 0$。
- 两个集合相等的充要条件是元素完全相同（与顺序无关）。

#### 知识点 1.1.3：集合的表示方法

**1. 定义**

集合的表示方法有列举法（把元素一一列举并用花括号括起来）、描述法（用元素的共同特征表示为 $\{x \in D \mid P(x)\}$）、Venn 图法和区间表示法。

**2. 重要性级别**：A

**3. 特性**

列举法适用于有限集；描述法适用于无限集或复杂集合；区间表示法是实数集合的简写方式。

**4. 运算方式**

- **列举法**：直接列出集合中所有元素，如 $A = \{1, 2, 3, 4, 5\}$。
- **描述法**：通过特征性质描述，格式为 $\{代表元素 \mid 元素满足的条件\}$，如 $B = \{x \in \mathbb{R} \mid x^2 - 3x + 2 = 0\}$。
- **Venn 图法**：用平面上的封闭曲线内部表示集合，直观展示集合关系与运算。
- **区间表示法**：设 $a < b$，则：
  - 开区间：$(a, b) = \{x \in \mathbb{R} \mid a < x < b\}$
  - 闭区间：$[a, b] = \{x \in \mathbb{R} \mid a \leq x \leq b\}$
  - 半开半闭区间：$[a, b)$ 或 $(a, b]$
  - 无穷区间：$(a, +\infty)$，$[a, +\infty)$，$(-\infty, b)$，$(-\infty, b]$，$(-\infty, +\infty) = \mathbb{R}$

**5. 定理**

- **区间运算的集合还原定理**：任何区间表示都可以唯一还原为描述法表示，且反之在实数连续条件下成立。
- 前提：讨论范围为实数集 $\mathbb{R}$。
- 结论：区间与描述法一一对应。
- 使用场合：定义域、值域的区间书写，不等式解集的规范化表达。

**6. 公式**

$$\{x \in \mathbb{R} \mid a < x < b\} = (a, b)$$
$$\{x \in \mathbb{R} \mid a \leq x \leq b\} = [a, b]$$
$$\{x \in \mathbb{R} \mid x > a\} = (a, +\infty)$$
$$\{x \in \mathbb{R} \mid x \leq b\} = (-\infty, b]$$

**7. 二级结论**

- 描述法中竖线左侧的代表元素符号不影响集合本质，即 $\{x \mid x > 1\} = \{t \mid t > 1\}$。
- 用描述法表示偶数集：$\{x \in \mathbb{Z} \mid x = 2k,\ k \in \mathbb{Z}\}$。
- 点集与数集的区别：$\{(x, y) \mid y = x\}$ 表示直线上的点集，而 $\{x \mid y = x\}$ 表示数集 $\mathbb{R}$。

#### 知识点 1.1.4：集合间的基本关系

**1. 定义**

子集：$A$ 的每个元素都是 $B$ 的元素则 $A \subseteq B$；集合相等：$A \subseteq B$ 且 $B \subseteq A$ 则 $A = B$；真子集：$A \subseteq B$ 且 $A \neq B$ 则 $A \subsetneqq B$。空集是任何集合的子集。

**2. 重要性级别**：A

**3. 特性**

含有 $n$ 个元素的集合有 $2^n$ 个子集、$2^n - 1$ 个真子集；子集具有自反性和传递性；空集是任何非空集合的真子集。

**4. 运算方式**

- **包含关系判断**：逐一验证 $A$ 中每个元素是否都属于 $B$。
- **集合相等证明**：需双向证明 $A \subseteq B$ 且 $B \subseteq A$。
- **子集个数计算**：由集合元素个数直接计算。含 $n$ 个元素的集合，其子集总数为 $2^n$，真子集数为 $2^n - 1$，非空子集数为 $2^n - 1$，非空真子集数为 $2^n - 2$。

**5. 定理**

- **子集计数定理**：若集合 $A$ 含有 $n$（$n \in \mathbb{N}$）个元素，则 $A$ 的子集个数为 $2^n$。
- 前提：$A$ 为有限集，$|A| = n$。
- 结论：$|\mathcal{P}(A)| = 2^n$，其中 $\mathcal{P}(A)$ 表示 $A$ 的幂集。
- 使用场合：快速计算子集个数、求参数范围使得某集合的子集个数满足条件。

- **包含关系的传递性**：若 $A \subseteq B$ 且 $B \subseteq C$，则 $A \subseteq C$。
- 前提：三个集合均有明确定义。
- 结论：包含关系具有传递性。
- 使用场合：多层次集合包含问题的推导。

**6. 公式**

$$A \subseteq B \Leftrightarrow \forall x \in A,\ x \in B$$
$$A = B \Leftrightarrow A \subseteq B \text{ 且 } B \subseteq A$$
$$|A| = n \Rightarrow \text{子集个数} = 2^n$$
$$\text{真子集个数} = 2^n - 1$$
$$\text{非空真子集个数} = 2^n - 2$$

**7. 二级结论**

- 若 $A \subseteq B$ 且 $A \subseteq C$，则 $A \subseteq B \cap C$。
- 空集 $\varnothing$ 是任何集合的子集，是任何非空集合的真子集，在解题中常作为易漏解的边界情况。
- 若 $A \subsetneqq B$ 且 $B \subsetneqq C$，则 $A \subsetneqq C$，但 $A$ 可能等于 $C$ 仅当传递链中出现等号。

#### 知识点 1.1.5：集合的基本运算

**1. 定义**

并集 $A \cup B = \{x \mid x \in A \text{ 或 } x \in B\}$；交集 $A \cap B = \{x \mid x \in A \text{ 且 } x \in B\}$；补集 $\complement_U A = \{x \mid x \in U \text{ 且 } x \notin A\}$。德摩根定律：$\complement_U(A \cup B) = (\complement_U A) \cap (\complement_U B)$。

**2. 重要性级别**：A

**3. 特性**

并集和交集都满足交换律和结合律；并集与交集满足分配律；补集满足双重补集律；德摩根定律将并和交与补联系起来。

**4. 运算方式**

- **并集**：取两集合全部元素，合并后去除重复。
- **交集**：仅取两集合的公共元素。
- **补集**：在全集 $U$ 中去除 $A$ 的元素。
- **混合运算**：遵循先括号内、后括号外的原则；德摩根定律可简化含补集的混合运算。

**5. 定理**

- **德摩根定律（De Morgan's Laws）**：
  $$\complement_U(A \cup B) = (\complement_U A) \cap (\complement_U B)$$
  $$\complement_U(A \cap B) = (\complement_U A) \cup (\complement_U B)$$
- 前提：$A$，$B$ 均为全集 $U$ 的子集。
- 结论：并的补等于补的交，交的补等于补的并。
- 使用场合：简化集合运算、逻辑命题的否定转换。

- **分配律**：
  $$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$$
  $$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$$
- 前提：任意三个集合 $A$，$B$，$C$。
- 结论：并对交可分配，交对并可分配。
- 使用场合：复杂集合表达式的化简与证明。

**6. 公式**


$$A \cup B = \{x \mid x \in A \lor x \in B\}$$
$$A \cap B = \{x \mid x \in A \land x \in B\}$$
$$\complement_U A = \{x \mid x \in U \land x \notin A\}$$

**7. 二级结论**

- 若 $A \subseteq B$，则 $A \cup B = B$ 且 $A \cap B = A$。
- $A \cap B = A \Leftrightarrow A \subseteq B \Leftrightarrow A \cup B = B$。
- 容斥原理（两集合版）：$|A \cup B| = |A| + |B| - |A \cap B|$。
- 对于有限个集合，德摩根定律可推广：$\complement_U\left(\bigcup_{i=1}^n A_i\right) = \bigcap_{i=1}^n (\complement_U A_i)$。

#### 知识点 1.2.1：映射的概念

**1. 定义**

设 $A$、$B$ 是两个非空集合，如果按照某种确定的对应关系 $f$，使 $A$ 中的任意一个元素 $x$ 在 $B$ 中都有唯一确定的元素 $y$ 与之对应，则称 $f: A \to B$ 为从集合 $A$ 到集合 $B$ 的映射。其中 $y$ 称为 $x$ 在映射 $f$ 下的像，$x$ 称为 $y$ 的原像。

**2. 重要性级别**：A

**3. 特性**

映射的三要素：原像集合 $A$、像的集合、对应法则 $f$；$A$ 中每个元素都要有对应且对应必须唯一；函数是一种特殊的映射。

**4. 运算方式**

- **映射的判定**：验证集合 $A$ 中每个元素在 $B$ 中是否有像（不允许无像），且像是否唯一（不允许一元素对应多像）。
- **像的计算**：给定 $x \in A$，通过对应法则 $f$ 计算 $y = f(x) \in B$。
- **原像的寻找**：给定 $y \in B$，求解方程 $f(x) = y$ 在 $A$ 中的解集，该解集即为 $y$ 的原像集。

**5. 定理**

- **映射的存在性判定**：对应关系 $f: A \to B$ 构成映射的充要条件是：$\forall x \in A$，$\exists! y \in B$ 使得 $y = f(x)$。
- 前提：集合 $A$、$B$ 非空，对应法则 $f$ 已给出。
- 结论：每个原像恰有一个像。
- 使用场合：判断给定对应关系是否为映射、分析函数图像是否满足垂直线检验。

**6. 公式**

$$f: A \to B, \quad x \mapsto y = f(x)$$
$$\text{像集} = f(A) = \{f(x) \mid x \in A\} \subseteq B$$

**7. 二级结论**

- 映射 $f: A \to B$ 中，$A$ 中元素个数 $|A|$ 与像集 $f(A)$ 中元素个数 $|f(A)|$ 满足 $|f(A)| \leq |A|$，等号成立当且仅当 $f$ 为单射。
- 若 $f: A \to B$ 且 $g: B \to C$ 都是映射，则复合映射 $g \circ f: A \to C$ 仍为映射，定义为 $(g \circ f)(x) = g(f(x))$。

#### 知识点 1.2.2：函数的概念

**1. 定义**

设 $A$、$B$ 是非空数集，如果按照某种确定的对应关系 $f$，使 $A$ 中的任意一个数 $x$ 在 $B$ 中都有唯一确定的数 $f(x)$ 与之对应，则称 $f: A \to B$ 为从 $A$ 到 $B$ 的函数，记作 $y = f(x)$，$x \in A$。

**2. 重要性级别**：A

**3. 特性**

函数是特殊的映射，要求 $A$、$B$ 必须是数集；自变量 $x$ 的取值范围称为定义域，函数值的集合称为值域；对应关系 $f$ 是核心。

**4. 运算方式**

- **函数求值**：给定 $x_0 \in A$，代入对应法则计算 $y_0 = f(x_0)$。
- **函数值的代数运算**：已知 $f(x)$ 与 $g(x)$，可构造新函数 $(f+g)(x) = f(x) + g(x)$，$(f \cdot g)(x) = f(x) \cdot g(x)$，$\left(\frac{f}{g}\right)(x) = \frac{f(x)}{g(x)}$（要求 $g(x) \neq 0$）。

**5. 定理**

- **函数的垂直线检验**：平面直角坐标系中，一个图像表示函数的充要条件是：任意垂直于 $x$ 轴的直线与该图像至多有一个交点。
- 前提：图像在坐标系中给出。
- 结论：垂直线至多交图像于一点。
- 使用场合：判断给定曲线是否为函数图像、理解函数的单值性。

**6. 公式**

$$y = f(x), \quad x \in A$$
$$\text{定义域: } D_f = A$$
$$\text{值域: } R_f = \{f(x) \mid x \in A\} \subseteq B$$

**7. 二级结论**

- 函数是一种特殊的映射（数集到数集的映射），因此映射的所有性质函数都具备。
- 函数 $y = f(x)$ 中，$f$ 表示对应法则，$f(x)$ 表示函数在 $x$ 处的函数值，两者不可混淆。例如 $f(x) = x^2$ 与 $f(t) = t^2$ 表示同一函数。
- 若函数 $f(x)$ 满足 $f(x+T) = f(x)$ 对定义域内所有 $x$ 成立，则 $f(x)$ 为周期函数，$T$ 为周期（此为后续拓展内容）。

#### 知识点 1.2.3：函数的三要素

**1. 定义**

函数的三要素是定义域（自变量 $x$ 的取值范围）、值域（函数值 $f(x)$ 的集合）和对应关系（$x$ 与 $y$ 之间的映射规则）。

**2. 重要性级别**：A

**3. 特性**

值域由定义域和对应关系共同决定；判断两个函数相等只需看定义域和对应关系是否相同；$f(x) = x$ 与 $g(x) = \frac{x^2}{x}$ 不相等，因为定义域不同。

**4. 运算方式**

- **判断函数相等**：两个函数 $f$ 与 $g$ 相等，当且仅当定义域相同且对定义域内每个 $x$ 都有 $f(x) = g(x)$。
- **由三要素反推函数**：已知定义域和对应关系，值域自然确定；反之，已知对应关系和值域，未必能唯一确定定义域。

**5. 定理**

- **函数相等的判定定理**：函数 $f$ 与函数 $g$ 相等 $\Leftrightarrow$ $D_f = D_g$ 且 $\forall x \in D_f$，$f(x) = g(x)$。
- 前提：两个函数均有明确定义。
- 结论：定义域相同且对应关系完全一致。
- 使用场合：判断两个解析式是否表示同一函数、求参数使两函数相等。

**6. 公式**

$$f = g \Leftrightarrow \begin{cases} D_f = D_g \ f(x) = g(x),\ \forall x \in D_f \end{cases}$$
$$R_f = \{y \mid y = f(x),\ x \in D_f\}$$

**7. 二级结论**

- 对应关系相同但定义域不同的函数是不同函数。例如 $f(x) = x$（$x \in \mathbb{R}$）与 $g(x) = x$（$x \in [0, 1]$）是不同函数，值域分别为 $\mathbb{R}$ 与 $[0, 1]$。
- 若两个函数的值域和对应关系相同，但定义域不同，则它们仍不相等。
- 实际问题中的函数，定义域还受实际意义限制（如人数必须为正整数，长度必须为正数）。

#### 知识点 1.2.4：函数定义域的求解

**1. 定义**

求函数定义域的原则：分母不为零、偶次根号下非负、对数的真数大于零、零次幂的底数不为零、实际问题中考虑自变量的实际意义。

**2. 重要性级别**：A

**3. 特性**

定义域必须用集合或区间表示；多个限制条件取交集；复合函数定义域要注意内层函数的值域与外层函数定义域的对应关系。

**4. 运算方式**

- **分式函数**：分母 $\neq 0$。
- **偶次根式**：被开方数 $\geq 0$。
- **对数函数**：真数 $> 0$，底数 $> 0$ 且 $\neq 1$。
- **零次幂**：底数 $\neq 0$（即 $x^0$ 要求 $x \neq 0$）。
- **正切函数**：$x \neq k\pi + \frac{\pi}{2}$，$k \in \mathbb{Z}$。
- **复合函数**：设 $y = f(g(x))$，则定义域由 $g(x)$ 的定义域与 $f$ 对 $g(x)$ 值的要求共同决定。

**5. 定理**

- **定义域的交集原则**：若函数 $f(x)$ 由多个基本初等函数通过四则运算或复合构成，则其定义域为各部分定义域的交集。
- 前提：函数表达式由有限个基本运算组合而成。
- 结论：$D_f = D_1 \cap D_2 \cap \dots \cap D_n$。
- 使用场合：分式、根式、对数混合函数的定义域求解。

**6. 公式**

$$f(x) = \frac{1}{g(x)} \Rightarrow D_f = \{x \mid g(x) \neq 0\}$$
$$f(x) = \sqrt[2n]{g(x)} \Rightarrow D_f = \{x \mid g(x) \geq 0\}$$
$$f(x) = \log_a g(x) \Rightarrow D_f = \{x \mid g(x) > 0,\ a > 0,\ a \neq 1\}$$
$$f(x) = [g(x)]^0 \Rightarrow D_f = \{x \mid g(x) \neq 0\}$$

**7. 二级结论**

- 已知 $f(x)$ 定义域为 $[a, b]$，则 $f(g(x))$ 的定义域由 $a \leq g(x) \leq b$ 解出。
- 已知 $f(g(x))$ 定义域为 $[a, b]$（即 $x \in [a, b]$），则 $f(x)$ 的定义域为 $g(x)$ 在 $[a, b]$ 上的值域。
- 实际问题中常见的定义域限制：时间 $t \geq 0$，人数 $n \in \mathbb{N}^*$，长度 $l > 0$ 等。

#### 知识点 1.2.5：函数值域的求解

**1. 定义**

求函数值域的常用方法有观察法、配方法、判别式法、换元法、分离常数法、单调性法和图像法。不同函数类型适用不同方法。

**2. 重要性级别**：A

**3. 特性**

一次函数值域为 $\mathbb{R}$；二次函数值域用配方法；分式函数可尝试分离常数法；含根式可考虑换元法；利用单调性可确定函数在区间上的最值。

**4. 运算方式**

- **观察法**：适用于简单函数，如 $y = \sqrt{x}$ 的值域为 $[0, +\infty)$。
- **配方法**：适用于二次函数，如 $y = x^2 - 2x + 3 = (x-1)^2 + 2$，值域为 $[2, +\infty)$。
- **分离常数法**：适用于一次分式函数，如 $y = \frac{2x+1}{x-1} = 2 + \frac{3}{x-1}$，值域为 $\{y \mid y \neq 2\}$。
- **判别式法**：适用于分子分母均为二次的分式函数，将 $y = \frac{a_1x^2+b_1x+c_1}{a_2x^2+b_2x+c_2}$ 整理为关于 $x$ 的二次方程，令判别式 $\Delta \geq 0$ 求 $y$ 的范围。
- **换元法**：适用于含根式的函数，如令 $t = \sqrt{x+1}$（$t \geq 0$），转化为关于 $t$ 的函数求值域。
- **单调性法**：若函数在区间上单调，则值域为区间端点函数值构成的区间。

**5. 定理**

- **闭区间连续函数的最值定理**：若函数 $f(x)$ 在闭区间 $[a, b]$ 上连续，则 $f(x)$ 在 $[a, b]$ 上必有最大值和最小值。
- 前提：$f(x)$ 在 $[a, b]$ 上连续。
- 结论：$\exists x_1, x_2 \in [a, b]$，使得 $f(x_1) \leq f(x) \leq f(x_2)$ 对 $\forall x \in [a, b]$ 成立。
- 使用场合：闭区间上多项式函数、初等函数的最值与值域求解。

**6. 公式**

$$y = ax^2 + bx + c \quad (a \neq 0) \Rightarrow R_f = \begin{cases} \left[\frac{4ac-b^2}{4a}, +\infty\right), & a > 0 \ \left(-\infty, \frac{4ac-b^2}{4a}\right], & a < 0 \end{cases}$$

$$y = \frac{ax+b}{cx+d} \quad (c \neq 0) \Rightarrow R_f = \left\{y \mid y \neq \frac{a}{c}\right\}$$

$$y = a^x \quad (a > 0, a \neq 1) \Rightarrow R_f = (0, +\infty)$$

$$y = \log_a x \quad (a > 0, a \neq 1) \Rightarrow R_f = \mathbb{R}$$

**7. 二级结论**

- 若 $f(x)$ 在区间 $D$ 上单调递增，则 $R_f = [f(x_{\min}), f(x_{\max})]$（端点可取时）。
- 若 $f(x)$ 在 $D$ 上连续且存在最值，则值域必为闭区间 $[m, M]$。
- 对于分式函数 $y = \frac{ax+b}{cx+d}$，对称中心为 $\left(-\frac{d}{c}, \frac{a}{c}\right)$，值域中去掉的就是对称中心的纵坐标。

#### 知识点 1.2.6：函数的表示法

**1. 定义**

函数的表示方法主要有解析法（用数学表达式表示）、列表法（列出 $x$ 与 $y$ 的对应值表）和图像法（用平面直角坐标系中的图形表示）。

**2. 重要性级别**：A

**3. 特性**

解析法精确但不够直观；列表法直接但不完整；图像法直观但不精确；三种方法各有优缺点，常结合使用；分段函数需要特别注意。

**4. 运算方式**

- **解析法**：给出函数表达式 $y = f(x)$，可精确计算任意自变量对应的函数值，便于代数运算和理论分析。
- **列表法**：列出有限组 $(x, y)$ 对应值，适用于离散型变量或实验数据。
- **图像法**：在坐标系中绘制 $(x, f(x))$ 的轨迹，直观反映函数的变化趋势、单调性、奇偶性、最值等性质。
- **三种方法的互化**：由解析式可描点画图，由图像可观察性质并推测解析式，由列表数据可尝试拟合解析式。

**5. 定理**

- **描点画图的基本原理**：函数 $y = f(x)$ 的图像是点集 $\{(x, y) \mid y = f(x), x \in D_f\}$。因此，在坐标系中描出足够多的点并用光滑曲线连接，即可近似得到函数图像。
- 前提：函数在所作图区间内连续。
- 结论：有限个点可逼近连续曲线。
- 使用场合：手工绘制函数图像草图、利用图像分析函数性质。

**6. 公式**

$$\text{解析法: } y = f(x)$$
$$\text{列表法: } \begin{array}{c|c|c|c} x & x_1 & x_2 & \dots \ \hline y & y_1 & y_2 & \dots \end{array}$$
$$\text{图像法: } G_f = \{(x, y) \in \mathbb{R}^2 \mid y = f(x),\ x \in D_f\}$$

**7. 二级结论**

- 解析法是三种表示法中唯一能进行精确代数运算和微积分分析的表示。
- 图像法是判断函数单调性、奇偶性、零点个数的最直观工具，也是数形结合思想的核心载体。
- 在信息时代，列表法常表现为数据库中的数据表，通过插值或拟合算法可转化为解析近似。

#### 知识点 1.2.7：分段函数

**1. 定义**

在函数定义域内，对于自变量 $x$ 的不同取值范围有着不同的对应关系的函数叫做分段函数。分段函数是一个函数而不是多个函数。

**2. 重要性级别**：A

**3. 特性**

分段函数的图像由几段曲线组成；求分段函数值要先确定 $x$ 所在的区间；分段函数的定义域是各段定义域的并集；分段函数可以描述复杂的对应关系。

**4. 运算方式**

- **求函数值**：给定 $x_0$，先判断 $x_0$ 属于哪个子区间，再代入该区间的对应表达式计算。
- **作图像**：在各区间的定义域范围内分别作出对应函数的图像，注意区间端点的开闭情况（实心点表示包含，空心点表示不包含）。
- **求最值**：分别在每段上求最值，再比较各段最值得到全局最值。
- **求值域**：分别求出每段的值域，再取并集。

**5. 定理**

- **分段函数的连续性判定**：分段函数 $f(x)$ 在分段点 $x_0$ 处连续的充要条件是：$\lim_{x \to x_0^-} f(x) = \lim_{x \to x_0^+} f(x) = f(x_0)$。
- 前提：分段点左右两侧表达式已知。
- 结论：左右极限存在、相等且等于函数值。
- 使用场合：判断分段函数在分段点处是否连续、作图时判断分段点是否为实心点。

**6. 公式**

$$f(x) = \begin{cases} f_1(x), & x \in D_1 \ f_2(x), & x \in D_2 \ \vdots \ f_n(x), & x \in D_n \end{cases}$$

其中 $D_1 \cup D_2 \cup \dots \cup D_n = D_f$，且 $D_i \cap D_j = \varnothing$（$i \neq j$）。

**7. 二级结论**

- 绝对值函数 $y = |x|$ 是最经典的分段函数：$y = \begin{cases} x, & x \geq 0 \ -x, & x < 0 \end{cases}$。
- 取整函数（高斯函数）$y = [x]$ 表示不超过 $x$ 的最大整数，其图像是阶梯形。
- 分段函数在分段点处可能出现间断，这是分析分段函数性质的关键节点。
- 实际问题中（如出租车计费、阶梯电价、个人所得税计算），分段函数是天然的数学模型。

#### 知识点 1.3.1：函数的单调性

**1. 定义**

设函数 $f(x)$ 的定义域为 $I$，如果对于 $I$ 内某个区间 $D$ 上的任意两个自变量值 $x_1$、$x_2$，当 $x_1 < x_2$ 时都有 $f(x_1) < f(x_2)$，则称 $f(x)$ 在 $D$ 上是增函数；若当 $x_1 < x_2$ 时都有 $f(x_1) > f(x_2)$，则称 $f(x)$ 在 $D$ 上是减函数。

**2. 重要性级别**：A

**3. 特性**

增函数的图像从左向右上升；减函数的图像从左向右下降；单调性是针对某个区间而言的局部性质；写单调区间时不能用并集连接不同区间。

**4. 运算方式**

- **定义法判断单调性**：
  1. 在区间 $D$ 上任取 $x_1 < x_2$；
  2. 计算差值 $f(x_1) - f(x_2)$；
  3. 对差值进行变形（因式分解、配方、有理化等）；
  4. 判断符号：若 $f(x_1) - f(x_2) < 0$ 则为增函数，若 $> 0$ 则为减函数。

- **图像法**：观察函数图像在区间上的升降趋势。

**5. 定理**

- **单调性的局部传递**：若 $f(x)$ 在区间 $D_1$ 上单调递增，在区间 $D_2$ 上单调递增，且 $D_1 \cap D_2 \neq \varnothing$，则 $f(x)$ 在 $D_1 \cup D_2$ 的某个包含交区间的子区间上保持递增趋势，但整个并集不一定单调递增。
- 前提：两区间有交集。
- 结论：并集整体不一定保持单调性。
- 使用场合：理解为何写单调区间时不能用并集符号（如 $y = \frac{1}{x}$ 在 $(-\infty, 0)$ 和 $(0, +\infty)$ 上都递减，但不能说在 $(-\infty, 0) \cup (0, +\infty)$ 上递减）。

**6. 公式**

$$\forall x_1, x_2 \in D,\ x_1 < x_2:$$
$$f(x_1) < f(x_2) \Leftrightarrow f(x) \text{ 在 } D \text{ 上单调递增}$$
$$f(x_1) > f(x_2) \Leftrightarrow f(x) \text{ 在 } D \text{ 上单调递减}$$

等价表述（作差法）：
$$\frac{f(x_1) - f(x_2)}{x_1 - x_2} > 0 \Leftrightarrow f(x) \text{ 在 } D \text{ 上单调递增}$$
$$\frac{f(x_1) - f(x_2)}{x_1 - x_2} < 0 \Leftrightarrow f(x) \text{ 在 } D \text{ 上单调递减}$$

**7. 二级结论**

- 单调性是函数的局部性质，一个函数在不同区间上可以有不同的单调性。
- 写单调区间时，若函数在两个不连续区间上单调性相同，不能用 "$\cup$" 连接，应分别写出或用 "和" 连接。例如：$y = \frac{1}{x}$ 的递减区间为 $(-\infty, 0)$ 和 $(0, +\infty)$。
- 后续可用导数法判断单调性：若 $f'(x) > 0$，则 $f(x)$ 在该区间上单调递增。

#### 知识点 1.3.2：单调性的判断

**1. 定义**

判断函数单调性的方法有定义法（任取 $x_1 < x_2$ 作差 $f(x_1) - f(x_2)$ 判断符号）、图像法（观察图像走向）和性质法（增 $+$ 增 $=$ 增，减 $+$ 减 $=$ 减）。

**2. 重要性级别**：A

**3. 特性**

定义法是根本方法；性质法可以快速判断复合函数的单调性；实际问题中常结合图像观察；对于可导函数后续可用导数法判断。

**4. 运算方式**

- **定义法**：最严格的方法，适用于任何函数，但计算量较大。
- **图像法**：适用于已知图像或易作图的函数，直观快速。
- **性质法**：利用已知函数的单调性推导组合函数的单调性：
  - 增 $+$ 增 $=$ 增
  - 减 $+$ 减 $=$ 减
  - 增 $-$ 减 $=$ 增
  - 减 $-$ 增 $=$ 减
  - $-f(x)$ 的单调性与 $f(x)$ 相反
- **复合函数单调性**（同增异减）：若 $y = f(u)$ 与 $u = g(x)$ 单调性相同（同为增或同为减），则 $y = f(g(x))$ 为增函数；若单调性相反，则复合函数为减函数。

**5. 定理**

- **复合函数单调性定理（同增异减）**：设函数 $u = g(x)$ 在区间 $D$ 上单调，函数 $y = f(u)$ 在对应区间 $g(D)$ 上单调，则复合函数 $y = f(g(x))$ 的单调性遵循 "同增异减" 原则。
- 前提：内外层函数各自的单调性已确定，且内层函数的值域落在外层函数的定义域内。
- 结论：内外单调性相同 $\Rightarrow$ 复合函数递增；内外单调性相反 $\Rightarrow$ 复合函数递减。
- 使用场合：快速判断复合函数（如对数函数、指数函数复合）的单调区间。

**6. 公式**

$$f(x) \nearrow,\ g(x) \nearrow \Rightarrow (f+g)(x) \nearrow$$
$$f(x) \searrow,\ g(x) \searrow \Rightarrow (f+g)(x) \searrow$$
$$f(x) \nearrow \Rightarrow -f(x) \searrow$$
$$f(x) \nearrow,\ g(x) \searrow \Rightarrow (f-g)(x) \nearrow$$

复合函数：
$$g(x) \nearrow,\ f(u) \nearrow \Rightarrow f(g(x)) \nearrow$$
$$g(x) \nearrow,\ f(u) \searrow \Rightarrow f(g(x)) \searrow$$

**7. 二级结论**

- 若 $f(x)$ 单调递增且恒正，则 $y = \frac{1}{f(x)}$ 单调递减。
- 若 $f(x)$ 单调递增且 $f(x) > 0$，则 $y = \sqrt{f(x)}$ 单调递增。
- 单调函数必存在反函数（在对应区间上），且反函数的单调性与原函数相同。

#### 知识点 1.3.3：函数的最值

**1. 定义**

设函数 $y = f(x)$ 的定义域为 $I$，如果存在实数 $M$ 满足对任意 $x \in I$ 都有 $f(x) \leq M$ 且存在 $x_0 \in I$ 使 $f(x_0) = M$，则称 $M$ 是函数的最大值。类似地，若存在 $m$ 满足 $f(x) \geq m$ 且能达到，则 $m$ 为最小值。

**2. 重要性级别**：A

**3. 特性**

最小值类似定义；最值是函数值域的边界值；闭区间上的连续函数必有最大值和最小值；利用单调性可以求函数在区间上的最值。

**4. 运算方式**

- **利用单调性**：若 $f(x)$ 在 $[a, b]$ 上单调递增，则最小值为 $f(a)$，最大值为 $f(b)$。
- **配方法**：适用于二次函数，通过配方求顶点坐标得到最值。
- **判别式法**：适用于特定分式函数。
- **图像法**：通过观察图像的最高点和最低点。
- **基本不等式法**：适用于可化为 $y = x + \frac{k}{x}$（$k > 0$，$x > 0$）形式的函数，利用均值不等式 $x + \frac{k}{x} \geq 2\sqrt{k}$。

**5. 定理**

- **闭区间连续函数的最值定理**：若 $f(x)$ 在闭区间 $[a, b]$ 上连续，则 $f(x)$ 在 $[a, b]$ 上必能取到最大值和最小值。
- 前提：$f(x)$ 在 $[a, b]$ 上连续。
- 结论：$\exists x_{\max}, x_{\min} \in [a, b]$，使得 $f(x_{\max}) = M$，$f(x_{\min}) = m$，且 $m \leq f(x) \leq M$ 对所有 $x \in [a, b]$ 成立。
- 使用场合：闭区间上初等函数最值的保证、实际优化问题的求解。

- **二次函数在闭区间上的最值**：设 $f(x) = ax^2 + bx + c$（$a \neq 0$），对称轴 $x = -\frac{b}{2a}$。在区间 $[m, n]$ 上：
  - 若对称轴在区间内，则最值在顶点和端点处取得；
  - 若对称轴在区间左侧，函数在区间上单调，最值在两端点；
  - 若对称轴在区间右侧，同理。
- 前提：二次函数、闭区间。
- 结论：最值必在顶点或端点处取得。
- 使用场合：二次函数区间最值问题、含参数二次函数的讨论。

**6. 公式**

$$M = \max_{x \in I} f(x) \Leftrightarrow \forall x \in I,\ f(x) \leq M \text{ 且 } \exists x_0 \in I,\ f(x_0) = M$$
$$m = \min_{x \in I} f(x) \Leftrightarrow \forall x \in I,\ f(x) \geq m \text{ 且 } \exists x_0 \in I,\ f(x_0) = m$$

二次函数 $f(x) = ax^2 + bx + c$ 在对称轴 $x = -\frac{b}{2a}$ 处：
$$f_{\text{极值}} = \frac{4ac - b^2}{4a}$$

**7. 二级结论**

- 若 $f(x)$ 在区间 $D$ 上连续且只有一个极值点，则该极值点即为最值点。
- 对于二次函数 $f(x) = ax^2 + bx + c$ 在区间 $[m, n]$ 上，讨论最值时需要比较 $f(m)$、$f(n)$ 和 $f\left(-\frac{b}{2a}\right)$（若对称轴在区间内）。
- 实际问题中的最值问题，求得数学最值后必须检验是否在定义域内可达、是否符合实际意义。

#### 知识点 1.3.4：函数的奇偶性

**1. 定义**

如果对于函数 $f(x)$ 的定义域内任意一个 $x$ 都有 $f(-x) = f(x)$，则称 $f(x)$ 为偶函数；如果都有 $f(-x) = -f(x)$，则称 $f(x)$ 为奇函数。

**2. 重要性级别**：A

**3. 特性**

偶函数的图像关于 $y$ 轴对称；奇函数的图像关于原点对称；奇偶性的前提条件是定义域关于原点对称；奇函数在 $x = 0$ 处有定义则必有 $f(0) = 0$。

**4. 运算方式**

- **定义法判断**：
  1. 验证定义域是否关于原点对称；
  2. 计算 $f(-x)$；
  3. 比较 $f(-x)$ 与 $f(x)$ 及 $-f(x)$：
     - 若 $f(-x) = f(x)$，则为偶函数；
     - 若 $f(-x) = -f(x)$，则为奇函数；
     - 若都不满足，则为非奇非偶函数。

- **图像法**：观察图像对称性——关于 $y$ 轴对称则为偶函数，关于原点对称则为奇函数。

**5. 定理**

- **奇偶性的必要条件**：若 $f(x)$ 具有奇偶性（无论是奇函数还是偶函数），则其定义域 $D$ 必须关于原点对称，即 $x \in D \Rightarrow -x \in D$。
- 前提：$f(x)$ 为奇函数或偶函数。
- 结论：$D$ 关于原点对称。
- 使用场合：快速排除非奇非偶函数、求参数使定义域对称。

- **奇函数在原点处的性质**：若奇函数 $f(x)$ 在 $x = 0$ 处有定义，则必有 $f(0) = 0$。
- 前提：$f(x)$ 为奇函数，且 $0 \in D_f$。
- 结论：$f(0) = 0$。
- 使用场合：利用 $f(0) = 0$ 求参数、判断函数图像是否过原点。

**6. 公式**

$$f(-x) = f(x) \quad \forall x \in D_f \Leftrightarrow f(x) \text{ 为偶函数}$$
$$f(-x) = -f(x) \quad \forall x \in D_f \Leftrightarrow f(x) \text{ 为奇函数}$$

图像对称性：
$$\text{偶函数: } (x, y) \in G_f \Rightarrow (-x, y) \in G_f$$
$$\text{奇函数: } (x, y) \in G_f \Rightarrow (-x, -y) \in G_f$$

**7. 二级结论**

- 既是奇函数又是偶函数的函数只有 $f(x) = 0$（定义域关于原点对称时）。
- 偶函数满足 $f(x) = f(|x|)$，在对称区间上研究性质时可只考虑 $x \geq 0$。
- 奇函数在对称区间上的单调性相同；偶函数在对称区间上的单调性相反。
- 任何定义域关于原点对称的函数 $f(x)$ 都可以唯一分解为一个奇函数与一个偶函数之和：
  $$f(x) = \underbrace{\frac{f(x) - f(-x)}{2}}_{\text{奇函数}} + \underbrace{\frac{f(x) + f(-x)}{2}}_{\text{偶函数}}$$

#### 知识点 1.3.5：奇偶性的判断

**1. 定义**

判断函数奇偶性的步骤：首先判断定义域是否关于原点对称，然后计算 $f(-x)$ 并与 $f(x)$ 比较。若 $f(-x) = f(x)$ 则为偶函数，若 $f(-x) = -f(x)$ 则为奇函数。

**2. 重要性级别**：A

**3. 特性**

定义域不关于原点对称则非奇非偶；两者都不满足则为非奇非偶函数；判断时应先化简 $f(-x)$ 再与 $\pm f(x)$ 比较。

**4. 运算方式**

- **步骤一：检验定义域对称性**
  若 $D_f$ 不关于原点对称，直接判定为非奇非偶函数。

- **步骤二：计算 $f(-x)$**
  将 $-x$ 代入函数表达式，注意符号变化和化简。

- **步骤三：比较与判定**
  - $f(-x) = f(x)$：偶函数
  - $f(-x) = -f(x)$：奇函数
  - 均不满足：非奇非偶函数

- **特殊技巧**：
  - 对于多项式函数：只含偶次项为偶函数，只含奇次项为奇函数，同时含偶次项和奇次项为非奇非偶函数（常数项视为偶次项）。
  - 对于分式函数：通常先通分或化简再判断。

**5. 定理**

- **多项式函数奇偶性判定定理**：多项式函数 $f(x) = a_nx^n + a_{n-1}x^{n-1} + \dots + a_1x + a_0$ 为偶函数的充要条件是所有奇次项系数为零；为奇函数的充要条件是所有偶次项系数（含常数项）为零。
- 前提：$f(x)$ 为多项式函数，定义域为 $\mathbb{R}$（关于原点对称）。
- 结论：奇次项全零 $\Leftrightarrow$ 偶函数；偶次项全零 $\Leftrightarrow$ 奇函数。
- 使用场合：快速判断多项式函数的奇偶性、求参数使多项式为奇/偶函数。

**6. 公式**

$$f(x) = \sum_{k=0}^{n} a_k x^k$$
$$\text{偶函数} \Leftrightarrow a_1 = a_3 = a_5 = \dots = 0$$
$$\text{奇函数} \Leftrightarrow a_0 = a_2 = a_4 = \dots = 0$$

**7. 二级结论**

- 判断奇偶性时，定义域的对称性检验是第一关，可快速排除大量非奇非偶函数。
- 若 $f(x)$ 的定义域为 $[a, b]$ 且 $a + b \neq 0$，则 $f(x)$ 必为非奇非偶函数。
- 对抽象函数或复合函数，奇偶性判断可借助已知基本初等函数的奇偶性通过性质法推导。

#### 知识点 1.3.6：函数性质的运算规律

**1. 定义**

单调性运算规律：增 $+$ 增 $=$ 增，减 $+$ 减 $=$ 减，$-$ 增 $=$ 减，$-$ 减 $=$ 增；奇偶性运算规律：奇 $\pm$ 奇 $=$ 奇，偶 $\pm$ 偶 $=$ 偶，奇 $\times$ 偶 $=$ 奇，偶 $\times$ 偶 $=$ 偶，奇 $\times$ 奇 $=$ 偶。

**2. 重要性级别**：B

**3. 特性**

奇函数在关于原点对称的区间上单调性相同；偶函数在关于原点对称的区间上单调性相反；奇 $\times$ 奇 $=$ 偶可以推广为两个奇函数的复合函数是偶函数。

**4. 运算方式**

- **单调性的四则运算**：
  - $f \nearrow + g \nearrow \Rightarrow (f+g) \nearrow$
  - $f \searrow + g \searrow \Rightarrow (f+g) \searrow$
  - $-f(x)$ 的单调性与 $f(x)$ 相反
  - $f \nearrow \times g \nearrow$：不确定（取决于函数值正负）

- **奇偶性的四则运算**：设 $f$、$g$ 定义域均关于原点对称：
  - 奇 $\pm$ 奇 $=$ 奇
  - 偶 $\pm$ 偶 $=$ 偶
  - 奇 $\pm$ 偶 $=$ 非奇非偶（一般情况下）
  - 奇 $\times$ 奇 $=$ 偶
  - 偶 $\times$ 偶 $=$ 偶
  - 奇 $\times$ 偶 $=$ 奇
  - $\frac{\text{奇}}{\text{偶}} = \text{奇}$（分母不为零时）

**5. 定理**

- **奇偶性乘积定理**：设 $f(x)$、$g(x)$ 的定义域均关于原点对称，则：
  - 若 $f$、$g$ 同为奇函数或同为偶函数，则 $f \cdot g$ 为偶函数；
  - 若 $f$、$g$ 一奇一偶，则 $f \cdot g$ 为奇函数。
- 前提：$f$、$g$ 定义域对称且均在定义域上有奇偶性。
- 结论：同偶异奇。
- 使用场合：快速判断乘积函数的奇偶性、分析复杂函数的组成。

**6. 公式**

$$f, g \text{ 均为奇} \Rightarrow f+g \text{ 为奇},\ f \cdot g \text{ 为偶}$$
$$f, g \text{ 均为偶} \Rightarrow f+g \text{ 为偶},\ f \cdot g \text{ 为偶}$$
$$f \text{ 奇},\ g \text{ 偶} \Rightarrow f \cdot g \text{ 为奇}$$

**7. 二级结论**

- 若 $f(x)$ 为奇函数且 $g(x)$ 为偶函数，则复合函数 $f(g(x))$ 为偶函数（因为 $f(g(-x)) = f(g(x))$）。
- 若 $f(x)$ 为奇函数，$g(x)$ 为奇函数，则 $f(g(x))$ 为奇函数。
- 若 $f(x)$ 为偶函数，则无论 $g(x)$ 的奇偶性如何，$f(g(x))$ 必为偶函数（当复合有意义时）。

#### 知识点 2.1.1：指数幂的运算

**1. 定义**

如果 $x^n = a$（$n \in \mathbb{N}^*$ 且 $n > 1$），则称 $x$ 为 $a$ 的 $n$ 次方根。正分数指数幂 $a^{\frac{m}{n}} = \sqrt[n]{a^m}$（$a > 0$，$m, n \in \mathbb{N}^*$，$n > 1$），负分数指数幂 $a^{-\frac{m}{n}} = \frac{1}{a^{\frac{m}{n}}}$。指数幂满足 $a^r \cdot a^s = a^{r+s}$ 等运算性质。

**2. 重要性级别**：A

**3. 特性**

$n$ 为奇数时 $\sqrt[n]{a^n} = a$；$n$ 为偶数时 $\sqrt[n]{a^n} = |a|$；运算性质：$a^r \cdot a^s = a^{r+s}$，$(a^r)^s = a^{rs}$，$(ab)^r = a^r \cdot b^r$；底数必须为正数。

**4. 运算方式**

- **根式化简**：利用 $\sqrt[n]{a^m} = a^{\frac{m}{n}}$ 将根式转化为分数指数幂进行运算。
- **指数幂乘法**：同底数幂相乘，指数相加：$a^r \cdot a^s = a^{r+s}$。
- **指数幂除法**：同底数幂相除，指数相减：$\frac{a^r}{a^s} = a^{r-s}$。
- **幂的乘方**：底数不变，指数相乘：$(a^r)^s = a^{rs}$。
- **积的乘方**：$(ab)^r = a^r \cdot b^r$。
- **商的乘方**：$\left(\frac{a}{b}\right)^r = \frac{a^r}{b^r}$。

**5. 定理**

- **指数幂运算基本定理**：设 $a > 0$，$b > 0$，$r, s \in \mathbb{Q}$（可推广到 $\mathbb{R}$），则：
  1. $a^r \cdot a^s = a^{r+s}$
  2. $(a^r)^s = a^{rs}$
  3. $(ab)^r = a^r \cdot b^r$
- 前提：底数 $a > 0$，$b > 0$（保证对任意有理指数都有意义）。
- 结论：指数运算可像整数指数一样进行代数化简。
- 使用场合：根式化简、指数方程求解、指数式的大小比较。

**6. 公式**

$$a^r \cdot a^s = a^{r+s} \quad (a > 0,\ r, s \in \mathbb{R})$$
$$\frac{a^r}{a^s} = a^{r-s} \quad (a > 0,\ r, s \in \mathbb{R})$$
$$(a^r)^s = a^{rs} \quad (a > 0,\ r, s \in \mathbb{R})$$
$$(ab)^r = a^r \cdot b^r \quad (a > 0,\ b > 0,\ r \in \mathbb{R})$$
$$\sqrt[n]{a^m} = a^{\frac{m}{n}} \quad (a > 0,\ m, n \in \mathbb{N}^*)$$
$$a^{-\frac{m}{n}} = \frac{1}{a^{\frac{m}{n}}} = \frac{1}{\sqrt[n]{a^m}}$$

特殊情形：
$$a^0 = 1 \quad (a \neq 0)$$
$$\sqrt[n]{a^n} = |a| \quad (n \text{ 为偶数})$$
$$\sqrt[n]{a^n} = a \quad (n \text{ 为奇数})$$

**7. 二级结论**

- 指数运算中，底数化为同底是核心技巧。例如比较 $3^{\frac{1}{2}}$、$9^{\frac{1}{4}}$、$27^{\frac{1}{6}}$ 时，统一化为以 3 为底：$3^{\frac{1}{2}}$、$3^{\frac{1}{2}}$、$3^{\frac{1}{2}}$，三者相等。
- $a^{\log_a N} = N$（对数恒等式，见 2.2.1）与指数运算结合使用，可实现指数与对数的灵活转化。

#### 知识点 2.1.2：指数函数的概念

**1. 定义**

一般地，函数 $y = a^x$（$a > 0$ 且 $a \neq 1$，$x \in \mathbb{R}$）叫做指数函数，其中 $x$ 为自变量，定义域为 $\mathbb{R}$。底数 $a > 0$ 保证对任意实数 $x$ 都有意义，$a \neq 1$ 避免退化为常函数 $y = 1$。

**2. 重要性级别**：A

**3. 特性**

指数函数与幂函数的区别：指数函数底数为常数指数为变量，幂函数底数为变量指数为常数；$a = 1$ 时为常函数 $y = 1$；$a \leq 0$ 时定义域不完整。

**4. 运算方式**

- **指数函数的求值**：给定 $x_0$，计算 $y_0 = a^{x_0}$。
- **指数函数的复合**：若 $y = a^{g(x)}$，则定义域由 $g(x)$ 的定义域决定（因为指数函数定义域为 $\mathbb{R}$，无额外限制）。

**5. 定理**

- **指数函数定义域的完备性**：对于 $a > 0$ 且 $a \neq 1$，$a^x$ 对任意实数 $x$ 都有定义且值为正实数。
- 前提：$a > 0$ 且 $a \neq 1$。
- 结论：$D = \mathbb{R}$，$a^x > 0$ 恒成立。
- 使用场合：指数函数建模中解释为何任意实数时间/距离都有意义。

**6. 公式**

$$y = a^x \quad (a > 0,\ a \neq 1,\ x \in \mathbb{R})$$
$$D = \mathbb{R}$$
$$\text{当 } a > 1: \quad a^x \begin{cases} > 1, & x > 0 \\ = 1, & x = 0 \\ < 1, & x < 0 \end{cases}$$
$$\text{当 } 0 < a < 1: \quad a^x \begin{cases} < 1, & x > 0 \\ = 1, & x = 0 \\ > 1, & x < 0 \end{cases}$$

**7. 二级结论**

- 指数函数 $y = a^x$ 与 $y = \left(\frac{1}{a}\right)^x$ 的图像关于 $y$ 轴对称，因为 $\left(\frac{1}{a}\right)^x = a^{-x}$。
- 指数函数的值恒正，因此方程 $a^x = 0$ 无解，$a^x = -1$ 也无解。
- 若 $a > 1$，则 $a^x$ 是增函数；若 $0 < a < 1$，则 $a^x$ 是减函数（将在 2.1.4 中严格证明）。

#### 知识点 2.1.3：指数函数的图像

**1. 定义**

指数函数 $y = a^x$（$a > 0$ 且 $a \neq 1$）的图像：$a > 1$ 时从左下向右上单调上升；$0 < a < 1$ 时从左上向右下单调下降。图像都过定点 $(0, 1)$，$x$ 轴是渐近线。

**2. 重要性级别**：A

**3. 特性**

所有指数函数图像都在 $x$ 轴上方即 $y > 0$ 恒成立；$a > 1$ 时 $x > 0$ 则 $y > 1$，$x < 0$ 则 $0 < y < 1$；$0 < a < 1$ 时 $x > 0$ 则 $0 < y < 1$，$x < 0$ 则 $y > 1$；图像分布在一、二象限。

**4. 运算方式**

- **描点画图**：选取 $x = -2, -1, 0, 1, 2$ 等关键点，计算对应 $y$ 值，描点连线。
- **图像变换**：
  - $y = a^{x+b}$：图像左右平移 $|b|$ 个单位（$b > 0$ 向左，$b < 0$ 向右）。
  - $y = a^x + c$：图像上下平移 $|c|$ 个单位。
  - $y = k \cdot a^x$：纵向伸缩。

**5. 定理**

- **指数函数图像的定点定理**：无论底数 $a$ 取何值（$a > 0$ 且 $a \neq 1$），指数函数 $y = a^x$ 的图像必过定点 $(0, 1)$，即 $a^0 = 1$。
- 前提：$y = a^x$ 为标准指数函数形式。
- 结论：图像过 $(0, 1)$。
- 使用场合：求指数型函数过定点问题（如 $y = a^{x-1} + 2$ 过定点 $(1, 3)$）。

- **指数函数图像的渐近线定理**：指数函数 $y = a^x$ 的图像以 $x$ 轴（即直线 $y = 0$）为水平渐近线。
- 前提：$a > 0$ 且 $a \neq 1$。
- 结论：$\lim_{x \to -\infty} a^x = 0$（当 $a > 1$）；$\lim_{x \to +\infty} a^x = 0$（当 $0 < a < 1$）。
- 使用场合：分析函数图像趋势、求指数函数的值域边界。

**6. 公式**

$$y = a^x \text{ 过定点 } (0, 1)$$
$$a > 1: \quad \lim_{x \to -\infty} a^x = 0,\ \lim_{x \to +\infty} a^x = +\infty$$
$$0 < a < 1: \quad \lim_{x \to -\infty} a^x = +\infty,\ \lim_{x \to +\infty} a^x = 0$$

**7. 二级结论**

- 在第一象限内（$x > 0$），底数越大的指数函数图像越靠上方，即若 $a > b > 1$，则当 $x > 0$ 时 $a^x > b^x$。
- 直线 $x = 1$ 与指数函数图像的交点为 $(1, a)$，因此可以通过交点纵坐标快速判断底数大小。
- 记忆口诀：大底高升小底低降，都过 $(0, 1)$。

#### 知识点 2.1.4：指数函数的性质

**1. 定义**

指数函数 $y = a^x$ 的性质：定义域为 $\mathbb{R}$；值域为 $(0, +\infty)$；过定点 $(0, 1)$；$a > 1$ 时在 $\mathbb{R}$ 上单调递增，$0 < a < 1$ 时在 $\mathbb{R}$ 上单调递减；$x$ 轴是水平渐近线。

**2. 重要性级别**：A

**3. 特性**

值域 $y > 0$ 恒成立；指数函数既不是奇函数也不是偶函数；在第一象限内底数越大的指数函数图像越靠上方；记忆口诀：大底高升小底低降都过 $(0, 1)$。

**4. 运算方式**

- **比较大小**：同底数时利用单调性；$a > 1$ 时指数大则幂大，$0 < a < 1$ 时指数大则幂小。
- **解指数不等式**：$a^{f(x)} > a^{g(x)}$，若 $a > 1$ 则等价于 $f(x) > g(x)$；若 $0 < a < 1$ 则等价于 $f(x) < g(x)$（不等号方向改变）。
- **求最值**：利用单调性，指数函数在闭区间上的最值在端点处取得。

**5. 定理**

- **指数函数单调性定理**：指数函数 $y = a^x$（$a > 0$ 且 $a \neq 1$），当 $a > 1$ 时在 $(-\infty, +\infty)$ 上严格单调递增；当 $0 < a < 1$ 时在 $(-\infty, +\infty)$ 上严格单调递减。
- 前提：$a > 0$ 且 $a \neq 1$。
- 结论：单调性由底数与 1 的大小关系唯一确定。
- 使用场合：指数不等式求解、指数函数比大小、指数函数最值分析。

**6. 公式**

$$y = a^x: \quad D = \mathbb{R},\ R = (0, +\infty)$$

$$\text{定点: } a^0 = 1 \Rightarrow \text{过 } (0, 1)$$

$$a > 1: \quad a^x \text{ 严格单调递增}$$
$$0 < a < 1: \quad a^x \text{ 严格单调递减}$$

**7. 二级结论**

- 若 $a > 1$，则 $a^x > 1 \Leftrightarrow x > 0$，$a^x = 1 \Leftrightarrow x = 0$，$0 < a^x < 1 \Leftrightarrow x < 0$。
- 若 $0 < a < 1$，则 $a^x > 1 \Leftrightarrow x < 0$，$a^x = 1 \Leftrightarrow x = 0$，$0 < a^x < 1 \Leftrightarrow x > 0$。
- 对于函数 $y = a^{f(x)}$（$a > 0$ 且 $a \neq 1$），其单调性与 $f(x)$ 的关系遵循 "同增异减"：$a > 1$ 时与 $f(x)$ 同单调；$0 < a < 1$ 时与 $f(x)$ 反单调。

#### 知识点 2.1.5：指数函数的应用

**1. 定义**

指数函数的应用包括比较大小（同底用单调性，同指数用幂函数性质，都不同找中间量如 1）、解指数方程和不等式，以及实际应用模型如人口增长和复利计算。

**2. 重要性级别**：B

**3. 特性**

解指数不等式时，$a > 1$ 则不等号方向不变，$0 < a < 1$ 则不等号方向改变；实际模型：人口增长 $N(t) = N_0 \cdot e^{rt}$、复利计算 $A = P(1+r)^t$、放射性衰变。

**4. 运算方式**

- **比大小策略**：
  - 同底：用单调性；
  - 同指数：化为同底或取对数比较；
  - 都不同：找中间量（常用 0 或 1），如 $a > 1$ 时 $a^x > 1$（$x > 0$），$0 < a^x < 1$（$x < 0$）。

- **解指数方程**：$a^{f(x)} = a^{g(x)} \Rightarrow f(x) = g(x)$（$a > 0$ 且 $a \neq 1$）。

- **解指数不等式**：
  - $a > 1$：$a^{f(x)} > a^{g(x)} \Leftrightarrow f(x) > g(x)$
  - $0 < a < 1$：$a^{f(x)} > a^{g(x)} \Leftrightarrow f(x) < g(x)$（注意不等号方向反转）

**5. 定理**

- **指数方程求解定理**：方程 $a^{f(x)} = a^{g(x)}$（$a > 0$ 且 $a \neq 1$）等价于 $f(x) = g(x)$。
- 前提：$a > 0$ 且 $a \neq 1$。
- 结论：指数相等当且仅当指数上的表达式相等。
- 使用场合：解指数方程、指数型复合方程。

**6. 公式**

$$a^{f(x)} = a^{g(x)} \Leftrightarrow f(x) = g(x) \quad (a > 0,\ a \neq 1)$$
$$a^{f(x)} > a^{g(x)} \Leftrightarrow \begin{cases} f(x) > g(x), & a > 1 \\ f(x) < g(x), & 0 < a < 1 \end{cases}$$

实际模型：
$$\text{复利公式: } A = P(1 + r)^t$$
$$\text{人口增长: } N(t) = N_0 e^{rt}$$
$$\text{放射性衰变: } N(t) = N_0 \cdot 2^{-\frac{t}{T_{1/2}}}$$

**7. 二级结论**

- 比较 $a^b$ 与 $b^a$（如 $e^\pi$ 与 $\pi^e$）时，可构造函数 $f(x) = \frac{\ln x}{x}$ 利用单调性比较。
- 指数函数 $y = a^x$（$a > 1$）描述 "爆炸式增长"，增长速度最终会超过任何幂函数。
- 实际问题中建立指数模型时，关键是确定初始值和增长率（或衰减率），然后通过已知数据求底数 $a$。

#### 知识点 2.2.1：对数的概念

**1. 定义**

如果 $a^x = N$（$a > 0$ 且 $a \neq 1$），则称 $x$ 为以 $a$ 为底 $N$ 的对数，记作 $x = \log_a N$。其中 $a$ 叫做底数，$N$ 叫做真数。常用对数 $\lg N = \log_{10} N$，自然对数 $\ln N = \log_e N$。

**2. 重要性级别**：A

**3. 特性**

真数 $N > 0$ 即负数与零没有对数；对数与指数的互化：$a^x = N \Leftrightarrow x = \log_a N$；基本性质：$\log_a a = 1$，$\log_a 1 = 0$，$a^{\log_a N} = N$；$e \approx 2.71828$。

**4. 运算方式**

- **指数与对数的互化**：$a^x = N \Leftrightarrow x = \log_a N$。
- **求对数值**：将对数式转化为指数式求解，如求 $\log_2 8$，设 $\log_2 8 = x$，则 $2^x = 8 = 2^3$，所以 $x = 3$。
- **对数恒等式**：$a^{\log_a N} = N$（$N > 0$），$\log_a a^x = x$（$x \in \mathbb{R}$）。

**5. 定理**

- **对数恒等式**：$a^{\log_a N} = N$（$a > 0$ 且 $a \neq 1$，$N > 0$）。
- 前提：$a > 0$，$a \neq 1$，$N > 0$。
- 结论：指数与对数互为逆运算，复合后还原为真数。
- 使用场合：化简指数对数复合表达式、解指数方程时取对数。

- **对数的基本性质**：
  1. $\log_a 1 = 0$（因为 $a^0 = 1$）
  2. $\log_a a = 1$（因为 $a^1 = a$）
- 前提：$a > 0$ 且 $a \neq 1$。
- 结论：对 1 的对数为 0，对底数的对数为 1。
- 使用场合：快速计算特殊值、化简对数式。

**6. 公式**

$$a^x = N \Leftrightarrow x = \log_a N \quad (a > 0,\ a \neq 1,\ N > 0)$$
$$\log_a 1 = 0$$
$$\log_a a = 1$$
$$a^{\log_a N} = N \quad (N > 0)$$
$$\log_a a^x = x \quad (x \in \mathbb{R})$$
$$\lg N = \log_{10} N, \quad \ln N = \log_e N, \quad e \approx 2.71828$$

**7. 二级结论**

- 由对数定义知真数必须大于 0，因此 $\log_a x$ 中隐含 $x > 0$，这是求对数函数定义域的根本依据。
- 对数式 $\log_a N$ 的符号判断：若 $a > 1$，则 $N > 1$ 时对数为正，$0 < N < 1$ 时对数为负；若 $0 < a < 1$，则相反。
- 对数运算的互化思想是解决复杂指数对数问题的核心策略。

---

#### 知识点 2.2.2：对数的运算性质

**1. 定义**

积的对数：$\log_a(M \cdot N) = \log_a M + \log_a N$；商的对数：$\log_a\left(\frac{M}{N}\right) = \log_a M - \log_a N$；幂的对数：$\log_a M^n = n \cdot \log_a M$（前提：$M > 0$，$N > 0$，$a > 0$ 且 $a \neq 1$）。

**2. 重要性级别**：A

**3. 特性**

运算口诀：积化和，商化差，幂化乘；重要警示：$\log_a(M+N) \neq \log_a M + \log_a N$；$\log_a(M-N) \neq \log_a M - \log_a N$；$\log_a(M \cdot N) \neq \log_a M \cdot \log_a N$。

**4. 运算方式**

- **积化和**：将乘积的对数拆为对数之和，便于化简和求值。
- **商化差**：将商的对数拆为对数之差。
- **幂化乘**：将幂的对数转化为指数乘以对数，是化简含幂对数式的核心技巧。
- **综合运算**：多个性质联合使用，如 $\log_a\frac{M^p \cdot N^q}{R^r} = p\log_a M + q\log_a N - r\log_a R$。

**5. 定理**

- **对数运算性质定理**：设 $a > 0$ 且 $a \neq 1$，$M > 0$，$N > 0$，则：
  1. $\log_a(M \cdot N) = \log_a M + \log_a N$
  2. $\log_a\frac{M}{N} = \log_a M - \log_a N$
  3. $\log_a M^n = n \log_a M$（$n \in \mathbb{R}$）
- 前提：$M > 0$，$N > 0$，$a > 0$ 且 $a \neq 1$。
- 结论：对数的运算将乘除降为加减、将幂降为乘法。
- 使用场合：对数式的化简与求值、对数方程的求解。

**6. 公式**

$$\log_a(MN) = \log_a M + \log_a N \quad (M > 0,\ N > 0)$$
$$\log_a\frac{M}{N} = \log_a M - \log_a N \quad (M > 0,\ N > 0)$$
$$\log_a M^n = n \log_a M \quad (M > 0,\ n \in \mathbb{R})$$

**7. 二级结论**

- $\log_a M + \log_a N = \log_a(MN)$，$\log_a M - \log_a N = \log_a\frac{M}{N}$，上述性质的逆用同样重要。
- $\log_a \sqrt[n]{M} = \frac{1}{n}\log_a M$（根式化为分数指数再用幂的性质）。
- 常见错误警示：$\log_a(M+N)$、$\log_a(M-N)$、$\log_a M \cdot \log_a N$ 均不能直接拆分或合并。

---

#### 知识点 2.2.3：换底公式

**1. 定义**

换底公式：$\log_a b = \frac{\log_c b}{\log_c a}$（$a > 0$ 且 $a \neq 1$，$c > 0$ 且 $c \neq 1$，$b > 0$）。最常用的换底方式是换成常用对数或自然对数：$\log_a b = \frac{\lg b}{\lg a} = \frac{\ln b}{\ln a}$。

**2. 重要性级别**：A

**3. 特性**

推论：$\log_a b = \frac{1}{\log_b a}$；$\log_a b \cdot \log_b c = \log_a c$；$\log_{a^n} b = \frac{1}{n}\log_a b$；$\log_{a^m} b^n = \frac{n}{m}\log_a b$；换底公式用于不同底数对数的统一计算。

**4. 运算方式**

- **不同底化同底**：利用换底公式将所有对数化为同一底数（通常选 10 或 $e$），便于比较和运算。
- **倒数关系**：$\log_a b \cdot \log_b a = 1$。
- **链式法则**：$\log_a b \cdot \log_b c \cdot \log_c d = \log_a d$。
- **底数含幂的处理**：$\log_{a^m} b^n = \frac{n}{m}\log_a b$。

**5. 定理**

- **换底公式**：$\log_a b = \frac{\log_c b}{\log_c a}$（$a, c > 0$ 且 $\neq 1$，$b > 0$）。
- 前提：各对数均有意义。
- 结论：对数的值与底数的选择无关，仅与真数和底数的比值有关。
- 使用场合：不同底数对数的运算、计算器求任意对数值、证明对数恒等式。

**6. 公式**

$$\log_a b = \frac{\log_c b}{\log_c a} = \frac{\ln b}{\ln a} = \frac{\lg b}{\lg a}$$
$$\log_a b = \frac{1}{\log_b a}$$
$$\log_a b \cdot \log_b c = \log_a c$$
$$\log_{a^m} b^n = \frac{n}{m} \log_a b$$
$$\log_{a^n} b = \frac{1}{n} \log_a b$$
$$\log_{a^m} b = \frac{1}{m} \log_a b$$

**7. 二级结论**

- 由换底公式可推导出对数函数恒过定点：$y = \log_a(x-1) + 2$ 过定点 $(2, 2)$（因为 $\log_a 1 = 0$）。
- 对于正数 $M$，$N$，有 $\frac{\log_a M}{\log_a N} = \log_N M$（无论 $a$ 取何值），这体现了换底公式的本质。
- 比较不同底对数的大小，常借助中间量（如 0、1）或统一换底后比较。

#### 知识点 2.2.4：对数函数的概念

**1. 定义**

一般地，函数 $y = \log_a x$（$a > 0$ 且 $a \neq 1$）叫做对数函数，其中 $x$ 为自变量，定义域为 $(0, +\infty)$。对数函数 $y = \log_a x$ 是指数函数 $y = a^x$ 的反函数。

**2. 重要性级别**：A

**3. 特性**

定义域为 $x > 0$ 是因为真数必须大于 0；对数函数与指数函数互为反函数，定义域和值域互换；对数函数的定义域 $(0, +\infty)$ 对应指数函数的值域。

**4. 运算方式**

- **对数函数求值**：给定 $x_0 > 0$，计算 $y_0 = \log_a x_0$，即求 $a^{y_0} = x_0$ 中的 $y_0$。
- **复合对数函数**：$y = \log_a g(x)$ 的定义域由 $g(x) > 0$ 决定。

**5. 定理**

- **对数函数与指数函数的互逆性**：$y = \log_a x$（$a > 0$ 且 $a \neq 1$）是 $y = a^x$ 的反函数，即：
  $$a^{\log_a x} = x \quad (x > 0)$$
  $$\log_a(a^x) = x \quad (x \in \mathbb{R})$$
- 前提：$a > 0$ 且 $a \neq 1$。
- 结论：两者互为逆运算，定义域与值域互换。
- 使用场合：指数方程与对数方程的互化、反函数问题的求解。

**6. 公式**

$$y = \log_a x \quad (a > 0,\ a \neq 1,\ x > 0)$$
$$D = (0, +\infty),\quad R = \mathbb{R}$$
$$\text{反函数关系: } y = a^x \Leftrightarrow x = \log_a y$$

**7. 二级结论**

- 若 $y = \log_a x$，则 $x = a^y$，因此点 $(x, y)$ 在对数函数图像上当且仅当点 $(y, x)$ 在指数函数图像上，这解释了为何两者图像关于 $y = x$ 对称。
- 对数函数 $y = \log_a x$ 与 $y = \log_{\frac{1}{a}} x$ 的图像关于 $x$ 轴对称，因为 $\log_{\frac{1}{a}} x = -\log_a x$。

#### 知识点 2.2.5：对数函数的图像

**1. 定义**

对数函数 $y = \log_a x$（$a > 0$ 且 $a \neq 1$）的图像：$a > 1$ 时从左下向右上方单调上升；$0 < a < 1$ 时从左上向右下方单调下降。图像都过定点 $(1, 0)$，$y$ 轴是渐近线。

**2. 重要性级别**：A

**3. 特性**

$a > 1$ 时 $x > 1$ 则 $y > 0$，$0 < x < 1$ 则 $y < 0$；$0 < a < 1$ 时 $x > 1$ 则 $y < 0$，$0 < x < 1$ 则 $y > 0$；图像分布在第一、四象限；记忆口诀：大底低升小底高降都过 $(1, 0)$。

**4. 运算方式**

- **描点画图**：取 $x = \frac{1}{a}, 1, a$ 等关键点。当 $x = a$ 时 $y = 1$；当 $x = 1$ 时 $y = 0$；当 $x = \frac{1}{a}$ 时 $y = -1$。
- **图像变换**：$y = \log_a(x+b)$ 左右平移，$y = \log_a x + c$ 上下平移。

**5. 定理**

- **对数函数图像的定点定理**：无论底数 $a$ 取何值（$a > 0$ 且 $a \neq 1$），对数函数 $y = \log_a x$ 的图像必过定点 $(1, 0)$，即 $\log_a 1 = 0$。
- 前提：$y = \log_a x$ 为标准对数函数形式。
- 结论：图像过 $(1, 0)$。
- 使用场合：求对数型函数过定点问题（如 $y = \log_a(x-2) + 3$ 过定点 $(3, 3)$）。

- **对数函数图像的渐近线定理**：对数函数 $y = \log_a x$ 的图像以 $y$ 轴（即直线 $x = 0$）为垂直渐近线。
- 前提：$a > 0$ 且 $a \neq 1$。
- 结论：$\lim_{x \to 0^+} \log_a x = -\infty$（当 $a > 1$）；$\lim_{x \to 0^+} \log_a x = +\infty$（当 $0 < a < 1$）。
- 使用场合：分析函数图像趋势、理解对数函数在 $x \to 0^+$ 时的趋向。

**6. 公式**

$$y = \log_a x \text{ 过定点 } (1, 0)$$
$$\text{直线 } x = 1 \text{ 与图像交于 } (1, 0),\ \text{直线 } y = 1 \text{ 与图像交于 } (a, 1)$$

$$a > 1: \quad \lim_{x \to 0^+} \log_a x = -\infty,\ \lim_{x \to +\infty} \log_a x = +\infty$$
$$0 < a < 1: \quad \lim_{x \to 0^+} \log_a x = +\infty,\ \lim_{x \to +\infty} \log_a x = -\infty$$

**7. 二级结论**

- 在第一象限内（$x > 1$），底数越大的对数函数图像越靠下方，即若 $a > b > 1$，则当 $x > 1$ 时 $\log_a x < \log_b x$。
- 直线 $y = 1$ 与对数函数图像的交点为 $(a, 1)$，因此可以通过交点横坐标快速判断底数大小。
- 记忆口诀：大底低升小底高降，都过 $(1, 0)$（注意与指数函数口诀的对应关系）。

---

#### 知识点 2.2.6：对数函数的性质

**1. 定义**

对数函数 $y = \log_a x$ 的性质：定义域为 $(0, +\infty)$；值域为 $\mathbb{R}$；过定点 $(1, 0)$；$a > 1$ 时单调递增，$0 < a < 1$ 时单调递减；$y$ 轴是垂直渐近线。

**2. 重要性级别**：A

**3. 特性**

对数函数既不是奇函数也不是偶函数；在第一象限内 $x > 1$ 时底数越大的对数函数图像越靠下方；值域为 $\mathbb{R}$ 说明对数函数可以取到任意实数值。

**4. 运算方式**

- **比较大小**：同底数时利用单调性；$a > 1$ 时真数大则对数值大，$0 < a < 1$ 时真数大则对数值小。
- **解对数不等式**：$\log_a f(x) > \log_a g(x)$，若 $a > 1$ 则等价于 $f(x) > g(x) > 0$；若 $0 < a < 1$ 则等价于 $0 < f(x) < g(x)$。
- **解对数方程**：$\log_a f(x) = \log_a g(x) \Leftrightarrow f(x) = g(x) > 0$（注意真数必须大于 0）。

**5. 定理**

- **对数函数单调性定理**：对数函数 $y = \log_a x$（$a > 0$ 且 $a \neq 1$），当 $a > 1$ 时在 $(0, +\infty)$ 上严格单调递增；当 $0 < a < 1$ 时在 $(0, +\infty)$ 上严格单调递减。
- 前提：$a > 0$ 且 $a \neq 1$。
- 结论：单调性由底数与 1 的大小关系唯一确定。
- 使用场合：对数不等式求解、对数函数比大小、对数函数定义域分析。

**6. 公式**

$$y = \log_a x: \quad D = (0, +\infty),\ R = \mathbb{R}$$

$$\text{定点: } \log_a 1 = 0 \Rightarrow \text{过 } (1, 0)$$

$$a > 1: \quad \log_a x \text{ 严格单调递增}$$
$$0 < a < 1: \quad \log_a x \text{ 严格单调递减}$$

**7. 二级结论**

- 若 $a > 1$，则 $\log_a x > 0 \Leftrightarrow x > 1$，$\log_a x = 0 \Leftrightarrow x = 1$，$\log_a x < 0 \Leftrightarrow 0 < x < 1$。
- 若 $0 < a < 1$，则 $\log_a x > 0 \Leftrightarrow 0 < x < 1$，$\log_a x = 0 \Leftrightarrow x = 1$，$\log_a x < 0 \Leftrightarrow x > 1$。
- 对于函数 $y = \log_a f(x)$（$a > 0$ 且 $a \neq 1$），其单调性与 $f(x)$ 的关系：$a > 1$ 时与 $f(x)$ 同单调；$0 < a < 1$ 时与 $f(x)$ 反单调（在 $f(x) > 0$ 的前提下）。

---

#### 知识点 2.2.7：指数与对数函数的关系

**1. 定义**

指数函数 $y = a^x$ 与对数函数 $y = \log_a x$（$a > 0$ 且 $a \neq 1$）互为反函数，它们的图像关于直线 $y = x$ 对称，定义域和值域互换。

**2. 重要性级别**：A

**3. 特性**

反函数的存在条件：原函数必须是单射；指数函数的值域 $(0, +\infty)$ 成为对数函数的定义域；对数函数的值域 $\mathbb{R}$ 成为指数函数的定义域；点 $(a, 1)$ 和 $(1, a)$ 分别在两条曲线上。

**4. 运算方式**

- **求反函数**：
  1. 从 $y = a^x$ 出发，交换 $x$ 与 $y$ 得 $x = a^y$；
  2. 解出 $y = \log_a x$，即为反函数。
- **利用反函数关系求值**：若点 $(p, q)$ 在 $y = a^x$ 上，则点 $(q, p)$ 在 $y = \log_a x$ 上。
- **图像对称作图**：已知指数函数图像，关于 $y = x$ 对称即可得到对数函数图像。

**5. 定理**

- **反函数图像对称定理**：若函数 $y = f(x)$ 存在反函数 $y = f^{-1}(x)$，则它们的图像关于直线 $y = x$ 对称。
- 前提：$f(x)$ 为单射（一一对应）。
- 结论：$G_f$ 与 $G_{f^{-1}}$ 关于 $y = x$ 对称。
- 使用场合：由已知函数图像作反函数图像、反函数问题的几何求解。

- **反函数的定义域值域互换定理**：反函数的定义域等于原函数的值域，反函数的值域等于原函数的定义域。
- 前提：反函数存在。
- 结论：$D_{f^{-1}} = R_f$，$R_{f^{-1}} = D_f$。
- 使用场合：求反函数的定义域和值域、复合反函数问题的分析。

**6. 公式**

$$y = a^x \Leftrightarrow x = \log_a y$$
$$f(x) = a^x,\quad f^{-1}(x) = \log_a x$$
$$D_f = \mathbb{R},\ R_f = (0, +\infty) \Leftrightarrow D_{f^{-1}} = (0, +\infty),\ R_{f^{-1}} = \mathbb{R}$$

关于 $y = x$ 的对称变换：
$$(x, y) \in G_f \Leftrightarrow (y, x) \in G_{f^{-1}}$$

**7. 二级结论**

- 指数函数 $y = a^x$ 过点 $(0, 1)$，则对数函数 $y = \log_a x$ 过点 $(1, 0)$（关于 $y = x$ 对称）。
- 指数函数 $y = a^x$ 过点 $(1, a)$，则对数函数 $y = \log_a x$ 过点 $(a, 1)$。
- 若 $f(f^{-1}(x)) = x$（$x \in D_{f^{-1}}$）且 $f^{-1}(f(x)) = x$（$x \in D_f$），这是反函数的核心恒等式。
- 反函数的存在性要求原函数单调（在区间内），因此指数函数和对数函数的反函数关系是完美的对称模型。

#### 知识点 2.3.1：幂函数的概念

**1. 定义**

一般地，函数 $y = x^\alpha$ 叫做幂函数，其中 $x$ 为自变量，$\alpha$ 为常数。常见的幂函数有 $y = x$、$y = x^2$、$y = x^3$、$y = x^{\frac{1}{2}}$、$y = x^{-1}$ 等。

**2. 重要性级别**：A

**3. 特性**

幂函数与指数函数的区别：幂函数底数为变量指数为常数，指数函数底数为常数指数为变量；幂函数的定义域视指数 $\alpha$ 的取值而定；所有幂函数都过定点 $(1, 1)$。

**4. 运算方式**

- **幂函数求值**：给定 $x_0$ 和指数 $\alpha$，计算 $y_0 = x_0^\alpha$。
- **定义域确定**：
  - $\alpha > 0$：$x \in \mathbb{R}$（若 $\alpha$ 为分数，需进一步分析分母的奇偶性）；
  - $\alpha < 0$：$x \neq 0$；
  - $\alpha$ 的分母为偶数时（如 $x^{\frac{1}{2}}$），要求 $x \geq 0$。

**5. 定理**

- **幂函数的定点定理**：对于任意幂函数 $y = x^\alpha$（$\alpha \in \mathbb{R}$），当 $x = 1$ 时 $y = 1$，即图像必过定点 $(1, 1)$。
- 前提：$x = 1$ 在定义域内（对任意 $\alpha$ 均成立）。
- 结论：图像过 $(1, 1)$。
- 使用场合：识别幂函数图像、求幂型函数过定点。

**6. 公式**

$$y = x^\alpha \quad (\alpha \in \mathbb{R})$$
$$1^\alpha = 1 \Rightarrow \text{过定点 } (1, 1)$$

定义域分类：
$$\alpha \in \mathbb{N}^* \Rightarrow D = \mathbb{R}$$
$$\alpha = -n \ (n \in \mathbb{N}^*) \Rightarrow D = (-\infty, 0) \cup (0, +\infty)$$
$$\alpha = \frac{1}{2n} \ (n \in \mathbb{N}^*) \Rightarrow D = [0, +\infty)$$
$$\alpha = \frac{1}{2n+1} \ (n \in \mathbb{N}) \Rightarrow D = \mathbb{R}$$

**7. 二级结论**

- 幂函数 $y = x^\alpha$ 与 $y = x^\beta$ 的图像在 $x = 1$ 处相交，因此在 $(0, 1)$ 和 $(1, +\infty)$ 上大小关系往往相反。
- 当 $\alpha > 0$ 时，$0^\alpha = 0$，图像过原点；当 $\alpha < 0$ 时，图像不过原点，以坐标轴为渐近线。
- 幂函数的定义域分析是分母奇偶性判断的典型应用场景。

---

#### 知识点 2.3.2：幂函数的图像与性质

**1. 定义**

幂函数 $y = x^\alpha$ 的图像都过定点 $(1, 1)$。$\alpha > 0$ 时图像过原点且在 $(0, +\infty)$ 上单调递增；$\alpha < 0$ 时图像不过原点且在 $(0, +\infty)$ 上单调递减。

**2. 重要性级别**：A

**3. 特性**

$y = x$、$y = x^3$、$y = x^{-1}$ 是奇函数；$y = x^2$ 是偶函数；$y = x^{\frac{1}{2}}$ 是非奇非偶；$\alpha > 0$ 时在第一象限单调递增，$\alpha < 0$ 时单调递减；$\alpha$ 为偶数时图像关于 $y$ 轴对称。

**4. 运算方式**

- **图像分类讨论**：
  - $\alpha = 1$：过原点的直线，斜率为 1；
  - $\alpha = 2$：抛物线，关于 $y$ 轴对称；
  - $\alpha = 3$：立方抛物线，关于原点对称；
  - $\alpha = \frac{1}{2}$：上半抛物线（$x \geq 0$）；
  - $\alpha = -1$：双曲线，以坐标轴为渐近线。

- **单调性分析**：在 $(0, +\infty)$ 上，$\alpha > 0$ 时单调递增，$\alpha < 0$ 时单调递减。

- **奇偶性判定**：$\alpha$ 为整数时，奇数为奇函数，偶数为偶函数；$\alpha$ 为分数时需具体分析定义域对称性。

**5. 定理**

- **幂函数在第一象限的单调性定理**：幂函数 $y = x^\alpha$ 在区间 $(0, +\infty)$ 上，当 $\alpha > 0$ 时严格单调递增，当 $\alpha < 0$ 时严格单调递减。
- 前提：$x > 0$。
- 结论：单调性由指数 $\alpha$ 的符号决定。
- 使用场合：幂函数比大小、幂不等式的求解。

**6. 公式**

$$y = x^\alpha: \quad \text{过 } (1, 1),\ (0, 0) \text{（当 } \alpha > 0 \text{ 时）}$$

$$\alpha > 0: \quad x^\alpha \text{ 在 } (0, +\infty) \text{ 上 } \nearrow$$
$$\alpha < 0: \quad x^\alpha \text{ 在 } (0, +\infty) \text{ 上 } \searrow$$

常见幂函数：
$$y = x: \quad \text{奇函数，在 } \mathbb{R} \text{ 上 } \nearrow$$
$$y = x^2: \quad \text{偶函数，在 } (-\infty, 0] \searrow,\ [0, +\infty) \nearrow$$
$$y = x^3: \quad \text{奇函数，在 } \mathbb{R} \text{ 上 } \nearrow$$
$$y = x^{\frac{1}{2}}: \quad \text{非奇非偶，在 } [0, +\infty) \text{ 上 } \nearrow$$
$$y = x^{-1}: \quad \text{奇函数，在 } (-\infty, 0) \searrow,\ (0, +\infty) \searrow$$

**7. 二级结论**

- 在区间 $(1, +\infty)$ 上，指数越大的幂函数图像越靠上方；在区间 $(0, 1)$ 上，指数越大的幂函数图像越靠下方。
- 幂函数 $y = x^\alpha$ 中，若 $\alpha$ 为既约分数 $\frac{p}{q}$（$q$ 为偶数），则定义域为 $[0, +\infty)$，此时函数非奇非偶。
- 幂函数的奇偶性判定：先看定义域是否关于原点对称，不对称则直接判非奇非偶；再检验 $f(-x)$ 与 $\pm f(x)$ 的关系。

---

#### 知识点 2.3.3：幂函数的应用

**1. 定义**

幂函数的应用包括比较幂的大小、建立实际问题的函数模型等。幂函数与指数函数、对数函数共同构成了高中阶段的基本初等函数体系。

**2. 重要性级别**：B

**3. 特性**

三类函数增长对比：对数增长 $\ll$ 幂函数增长 $\ll$ 指数增长；幂函数 $y = x^k$ 的增长速度介于对数函数和指数函数之间；当 $x \to +\infty$ 时 $\log_a x \ll x^k \ll a^x$。

**4. 运算方式**

- **比较幂的大小**：
  - 同底数：用幂函数的单调性（$\alpha > 0$ 时底大幂大，$\alpha < 0$ 时相反）；
  - 同指数：用底数大小比较（$x > 1$ 时底大值大，$0 < x < 1$ 时底大值小）；
  - 都不同：找中间量（如 0、1）或取对数比较。

- **建立模型**：正方体边长与体积 $V = a^3$、圆面积与半径 $S = \pi r^2$ 都是幂函数模型。

**5. 定理**

- **三类函数增长速度比较定理**：当 $x \to +\infty$ 时，对于 $a > 1$，$k > 0$，有：
  $$\log_a x \ll x^k \ll a^x$$
  即对数函数增长最慢，幂函数居中，指数函数增长最快。
- 前提：$a > 1$，$k > 0$，$x \to +\infty$。
- 结论：指数增长最终会超过任何幂函数增长。
- 使用场合：选择合适的函数模型描述实际增长现象、极限分析。

**6. 公式**

$$\lim_{x \to +\infty} \frac{\log_a x}{x^k} = 0 \quad (a > 1,\ k > 0)$$
$$\lim_{x \to +\infty} \frac{x^k}{a^x} = 0 \quad (a > 1,\ k > 0)$$

**7. 二级结论**

- 幂函数 $y = x^k$（$k > 0$）描述的是多项式增长，在实际问题中常用于面积、体积与边长/半径的关系建模。
- 当 $x$ 很大时，即使很小的指数差异也会导致幂函数值巨大差异，例如 $10^3 = 1000$ 而 $10^4 = 10000$。
- 三类基本初等函数的选择策略：增长越来越慢选对数模型，匀速增长选线性/幂函数模型，爆炸式增长选指数模型。

#### 知识点 3.1.1：函数的零点

**1. 定义**

对于函数 $y = f(x)$，使 $f(x) = 0$ 的实数 $x$ 叫做函数的零点。函数的零点不是一个点（坐标）而是一个实数，是函数图像与 $x$ 轴交点的横坐标。

**2. 重要性级别**：A

**3. 特性**

代数意义：$f(x_0) = 0$；几何意义：点 $(x_0, 0)$ 在函数图像上；方程 $f(x) = 0$ 有实数根 $\Leftrightarrow$ 函数图像与 $x$ 轴有交点 $\Leftrightarrow$ 函数有零点；一次函数零点为 $x = -\frac{b}{a}$。

**4. 运算方式**

- **求零点**：解方程 $f(x) = 0$，其实数解即为零点。
- **判断零点个数**：
  - 代数法：解方程，看实数解个数；
  - 图像法：看函数图像与 $x$ 轴交点个数；
  - 性质法：利用单调性和零点存在性定理综合判断。

**5. 定理**

- **零点与方程根的等价定理**：函数 $y = f(x)$ 有零点 $x_0$ 的充要条件是方程 $f(x) = 0$ 有实数根 $x_0$。
- 前提：$f(x)$ 为实函数。
- 结论：函数零点 $\Leftrightarrow$ 方程实根。
- 使用场合：将函数问题转化为方程问题求解，或将方程根的问题转化为函数零点问题分析。

**6. 公式**

$$x_0 \text{ 是 } f(x) \text{ 的零点 } \Leftrightarrow f(x_0) = 0$$
$$\Leftrightarrow x_0 \text{ 是方程 } f(x) = 0 \text{ 的实数根}$$
$$\Leftrightarrow (x_0, 0) \text{ 是 } y = f(x) \text{ 图像与 } x \text{ 轴的交点}$$

一次函数 $f(x) = ax + b$（$a \neq 0$）的零点：
$$x = -\frac{b}{a}$$

**7. 二级结论**

- 零点是一个实数，不是一个点，因此说 "$x = 2$ 是零点"而非 "$(2, 0)$ 是零点"。
- 二次函数 $f(x) = ax^2 + bx + c$（$a \neq 0$）的零点个数由判别式 $\Delta = b^2 - 4ac$ 决定：$\Delta > 0$ 时两个零点，$\Delta = 0$ 时一个零点，$\Delta < 0$ 时无零点。
- 若 $f(x) = g(x) - h(x)$，则 $f(x)$ 的零点等价于方程 $g(x) = h(x)$ 的根，几何意义是 $y = g(x)$ 与 $y = h(x)$ 图像交点的横坐标。

---

#### 知识点 3.1.2：零点与方程根的关系

**1. 定义**

方程 $f(x) = 0$ 的实数根就是函数 $y = f(x)$ 的零点。求函数零点就是求方程 $f(x) = 0$ 的实数解。这是方程问题与函数问题相互转化的桥梁。

**2. 重要性级别**：A

**3. 特性**

一次函数有唯一零点；二次函数零点个数由判别式决定；指数函数无零点；对数函数零点为 $x = 1$；幂函数零点视 $\alpha$ 而定；体现了函数与方程的思想。

**4. 运算方式**

- **代数求解**：对方程 $f(x) = 0$ 进行因式分解、配方、求根公式等代数运算，求出精确根。
- **图像估计**：作 $y = f(x)$ 图像，观察与 $x$ 轴交点位置，估计零点所在区间。
- **转化为交点问题**：$f(x) = 0 \Leftrightarrow f(x) = g(x) - h(x) = 0 \Leftrightarrow g(x) = h(x)$，即转化为 $y = g(x)$ 与 $y = h(x)$ 的交点问题。

**5. 定理**

- **函数与方程的三重等价关系**：
  1. 方程 $f(x) = 0$ 有实数根；
  2. 函数 $y = f(x)$ 的图像与 $x$ 轴有交点；
  3. 函数 $y = f(x)$ 有零点。
- 前提：$f(x)$ 为实值函数。
- 结论：三者等价，可相互转化。
- 使用场合：灵活选择代数法或图像法研究方程根的问题。

**6. 公式**

$$f(x) = 0 \text{ 有实根 } \Leftrightarrow y = f(x) \text{ 与 } x \text{ 轴有交点 } \Leftrightarrow f(x) \text{ 有零点}$$

各类函数的零点情况：
$$f(x) = ax + b \ (a \neq 0): \quad \text{唯一零点 } x = -\frac{b}{a}$$
$$f(x) = ax^2 + bx + c \ (a \neq 0): \quad \text{零点个数由 } \Delta = b^2 - 4ac \text{ 决定}$$
$$f(x) = a^x \ (a > 0, a \neq 1): \quad \text{无零点（因 } a^x > 0 \text{ 恒成立）}$$
$$f(x) = \log_a x \ (a > 0, a \neq 1): \quad \text{唯一零点 } x = 1$$
$$f(x) = x^\alpha \ (\alpha > 0): \quad \text{零点 } x = 0$$
$$f(x) = x^\alpha \ (\alpha < 0): \quad \text{无零点}$$

**7. 二级结论**

- 对于超越方程（如 $e^x = x + 2$），通常无法求得精确解，需要借助函数图像分析零点个数和估计零点位置。
- 函数 $f(x) = g(x) - h(x)$ 的零点个数等于 $y = g(x)$ 与 $y = h(x)$ 图像的交点个数，这是数形结合思想的典型应用。
- 分段函数的零点需要分段讨论，分别求解各段上的零点，再检验是否落在对应区间内。

#### 知识点 3.1.3：零点存在性定理

**1. 定义**

如果函数 $y = f(x)$ 在闭区间 $[a, b]$ 上的图像是连续不断的一条曲线，且 $f(a) \cdot f(b) < 0$，则函数 $y = f(x)$ 在区间 $(a, b)$ 内有零点。

**2. 重要性级别**：A

**3. 特性**

前提：函数在 $[a, b]$ 上连续且端点函数值异号；结论：在开区间 $(a, b)$ 内至少存在一个零点；充分非必要条件；几何意义：连续曲线从 $x$ 轴一侧穿到另一侧必与 $x$ 轴相交。

**4. 运算方式**

- **零点区间估计**：计算函数在若干点处的函数值，寻找函数值变号的区间 $[a, b]$。
- **根的隔离**：通过逐步缩小区间范围，将零点限制在越来越小的区间内。
- **结合单调性**：若函数在 $[a, b]$ 上连续且单调，且 $f(a) \cdot f(b) < 0$，则零点唯一。

**5. 定理**

- **零点存在性定理（Bolzano 定理的特例）**：若 $f(x)$ 在 $[a, b]$ 上连续，且 $f(a) \cdot f(b) < 0$，则 $\exists c \in (a, b)$ 使得 $f(c) = 0$。
- 前提：$f(x)$ 在 $[a, b]$ 上连续，$f(a)$ 与 $f(b)$ 异号。
- 结论：$(a, b)$ 内至少有一个零点。
- 使用场合：判断方程在某区间内是否有根、估计零点所在位置、二分法求近似解的理论基础。

**6. 公式**

$$f(x) \text{ 在 } [a, b] \text{ 上连续},\ f(a) \cdot f(b) < 0$$
$$\Rightarrow \exists c \in (a, b),\ f(c) = 0$$

等价表述：
$$f(a) > 0,\ f(b) < 0 \text{ 或 } f(a) < 0,\ f(b) > 0 \Rightarrow \exists c \in (a, b),\ f(c) = 0$$

**7. 二级结论**

- $f(a) \cdot f(b) < 0$ 是充分非必要条件：即使 $f(a) \cdot f(b) > 0$，函数仍可能有零点（如 $f(x) = x^2 - 1$ 在 $[-2, 2]$ 上有两个零点但 $f(-2) \cdot f(2) = 3 \cdot 3 > 0$）。
- 连续性条件不可省略：若函数不连续，即使端点异号也未必有零点（如分段函数在间断点处跳过 $x$ 轴）。
- 对于多项式函数、指数函数、对数函数、三角函数等初等函数，在其定义域内都是连续的，因此可直接应用零点存在性定理。

#### 知识点 3.1.4：唯一零点判定

**1. 定义**

若函数 $y = f(x)$ 在 $[a, b]$ 上连续且单调，并且 $f(a) \cdot f(b) < 0$，则 $f(x)$ 在 $(a, b)$ 内有且仅有一个零点。

**2. 重要性级别**：A

**3. 特性**

零点存在性定理保证至少有一个零点；单调性保证至多有一个零点；两者结合得出恰有一个零点；可用于判断方程在区间内唯一实根的存在性。

**4. 运算方式**

- **判定步骤**：
  1. 验证 $f(x)$ 在 $[a, b]$ 上的连续性（初等函数在定义域内自动满足）；
  2. 求导或利用单调性定义判断 $f(x)$ 在 $[a, b]$ 上的单调性；
  3. 计算 $f(a)$ 与 $f(b)$，验证是否异号；
  4. 若以上条件均满足，则 $f(x)$ 在 $(a, b)$ 内有且仅有一个零点。

**5. 定理**

- **唯一零点判定定理**：若 $f(x)$ 在 $[a, b]$ 上连续，在 $(a, b)$ 上严格单调，且 $f(a) \cdot f(b) < 0$，则 $f(x)$ 在 $(a, b)$ 内有且仅有一个零点。
- 前提：连续性 + 严格单调性 + 端点异号。
- 结论：存在唯一的 $c \in (a, b)$ 使 $f(c) = 0$。
- 使用场合：证明方程在某区间内有唯一解、确定零点个数、二分法收敛性的保证。

**6. 公式**

$$\begin{cases} f(x) \text{ 在 } [a, b] \text{ 上连续} \\ f(x) \text{ 在 } (a, b) \text{ 上严格单调} \\ f(a) \cdot f(b) < 0 \end{cases} \Rightarrow \exists! c \in (a, b),\ f(c) = 0$$

**7. 二级结论**

- 若 $f(x)$ 在 $[a, b]$ 上连续，在 $(a, b)$ 上可导且 $f'(x) > 0$（或 $f'(x) < 0$）恒成立，则 $f(x)$ 严格单调，结合端点异号可推出唯一零点。
- 对于二次函数，$\Delta > 0$ 时有两个不同零点，$\Delta = 0$ 时有唯一零点（重根），$\Delta < 0$ 时无零点。单调区间内的二次函数至多有一个零点。
- 唯一零点判定定理是证明方程根的 "存在性 + 唯一性" 的标准工具。

---

#### 知识点 3.1.5：二分法

**1. 定义**

二分法是基于零点存在性定理，通过不断将包含零点的区间对半分割，逐步缩小零点所在范围，从而得到零点近似值的方法。

**2. 重要性级别**：A

**3. 特性**

步骤：确定区间验证 $f(a) \cdot f(b) < 0$ $\to$ 计算中点 $c = \frac{a+b}{2}$ $\to$ 判断 $f(c)$ 符号确定新区间 $\to$ 重复直至精度满足；每次迭代区间长度减半；误差估计：$|x_n - c| \leq \frac{b-a}{2^{n+1}}$。

**4. 运算方式**

- **二分法步骤**：
  1. **确定初始区间** $[a, b]$：验证 $f(a) \cdot f(b) < 0$；
  2. **计算中点**：$c = \frac{a+b}{2}$；
  3. **判断符号**：
     - 若 $f(c) = 0$，则 $c$ 即为精确零点；
     - 若 $f(a) \cdot f(c) < 0$，则零点在 $[a, c]$ 中，令 $b = c$；
     - 若 $f(c) \cdot f(b) < 0$，则零点在 $[c, b]$ 中，令 $a = c$；
  4. **检验精度**：若 $|b - a| < \varepsilon$（预设精度），则停止，取 $c \approx \frac{a+b}{2}$ 为零点近似值；否则返回步骤 2。

**5. 定理**

- **二分法收敛定理**：若 $f(x)$ 在 $[a, b]$ 上连续且 $f(a) \cdot f(b) < 0$，则二分法产生的区间序列 $[a_n, b_n]$ 必收敛到 $f(x)$ 的一个零点 $c$，且误差满足 $|c - c_n| \leq \frac{b-a}{2^{n+1}}$，其中 $c_n = \frac{a_n + b_n}{2}$。
- 前提：$f(x)$ 连续且端点异号。
- 结论：$n$ 次迭代后误差不超过初始区间长度的一半的 $n+1$ 次方。
- 使用场合：求方程近似根、算法设计、数值计算。

**6. 公式**

$$c_n = \frac{a_n + b_n}{2}$$

$$|c - c_n| \leq \frac{b - a}{2^{n+1}}$$

$$n \text{ 次迭代后区间长度} = \frac{b - a}{2^n}$$

达到精度 $\varepsilon$ 所需最少迭代次数：
$$n \geq \log_2\frac{b - a}{\varepsilon} - 1$$

**7. 二级结论**

- 二分法的优点是原理简单、编程容易、收敛稳定；缺点是收敛速度较慢（线性收敛），对于重根或切线根（$f'(c) = 0$ 时）效率降低。
- 每次二分后区间长度减半，因此从区间 $[a, b]$ 开始，经过 $n$ 次二分，区间长度变为 $\frac{b-a}{2^n}$。
- 若要求精确到小数点后 $k$ 位（即误差 $< 0.5 \times 10^{-k}$），可由 $\frac{b-a}{2^{n+1}} < 0.5 \times 10^{-k}$ 估算所需迭代次数。
- 二分法要求函数在区间端点异号，对于偶次重根（函数值同号但图像切于 $x$ 轴）无法直接应用。

#### 知识点 3.2.1：不同增长的函数模型

**1. 定义**

三类基本增长模型：线性增长 $y = kx + b$ 匀速增长速度恒定；指数增长 $y = a^x$（$a > 1$）爆炸式增长越来越快；对数增长 $y = \log_a x$（$a > 1$）增长越来越慢趋于平缓。

**2. 重要性级别**：A

**3. 特性**

线性增长图像为直线；指数增长图像下平上陡；对数增长图像上陡下平；实际问题中选择模型要根据增长特征判断。

**4. 运算方式**

- **识别增长特征**：
  - 匀速增长（单位时间内增量相同）$\to$ 线性模型；
  - 增长率恒定（单位时间内增长百分比相同）$\to$ 指数模型；
  - 增长速度递减（越往后增长越慢）$\to$ 对数模型；
  - 幂次关系（面积与边长、体积与半径）$\to$ 幂函数模型。

- **模型对比分析**：在同一坐标系中绘制各模型的图像，观察增长差异；计算单位增量或增长率的变化趋势。

**5. 定理**

- **三类增长模型的特征定理**：
  - 线性模型 $y = kx + b$（$k > 0$）：增量 $\Delta y = k \Delta x$ 恒定，图像为直线；
  - 指数模型 $y = a^x$（$a > 1$）：增长率 $\frac{\Delta y}{y}$ 近似恒定，图像下平上陡，呈 "J" 形；
  - 对数模型 $y = \log_a x$（$a > 1$）：增量 $\Delta y$ 越来越小，图像上陡下平，趋于水平。
- 前提：各模型参数在标准范围内。
- 结论：三类模型的增长特征截然不同。
- 使用场合：根据实际数据的增长特征选择合适的函数模型。

**6. 公式**

线性增长：
$$y = kx + b \quad (k > 0)$$
$$\Delta y = k \Delta x \text{（常数）}$$

指数增长：
$$y = N_0 a^t \quad (a > 1) \text{ 或 } y = N_0 e^{rt} \quad (r > 0)$$
$$\frac{y(t+1) - y(t)}{y(t)} \approx \text{常数}$$

对数增长：
$$y = k \log_a t + b \quad (a > 1,\ k > 0)$$
$$\Delta y \to 0 \text{（当 } t \to +\infty \text{ 时）}$$

**7. 二级结论**

- 线性增长是 "等量增长"，指数增长是 "等比增长"，对数增长是 "减速增长"。
- 复利计算、细菌繁殖、病毒传播等问题本质上是离散时间的指数增长；连续情形下用 $N(t) = N_0 e^{rt}$。
- 对数增长常用于描述学习曲线、技能提升等现象——初期进步快，后期趋于平台。

#### 知识点 3.2.2：增长速度排序

**1. 定义**

增长速度排序定理：当 $x \to +\infty$ 时，对数增长远小于幂函数增长远小于指数增长，即 $\log_a x \ll x^k \ll a^x$（$a > 1$，$k > 0$）。

**2. 重要性级别**：A

**3. 特性**

极限表示：$\lim_{x \to +\infty} \frac{\log_a x}{x^k} = 0$ 且 $\lim_{x \to +\infty} \frac{x^k}{a^x} = 0$；指数函数的增长速度会超过任何多项式函数；对数函数增长再快也追不上幂函数。

**4. 运算方式**

- **极限比较法**：通过计算 $\lim_{x \to +\infty} \frac{f(x)}{g(x)}$ 判断增长速度：
  - 若极限为 0，则 $f(x)$ 增长慢于 $g(x)$；
  - 若极限为 $+\infty$，则 $f(x)$ 增长快于 $g(x)$；
  - 若极限为非零常数，则两者同阶增长。

- **取对数比较法**：比较 $\ln f(x)$ 与 $\ln g(x)$ 的增长速度。

**5. 定理**

- **增长速度排序定理**：设 $a > 1$，$k > 0$，则当 $x \to +\infty$ 时：
  $$\log_a x \ll x^k \ll a^x \ll x! \ll x^x$$
- 前提：$x \to +\infty$，各参数在标准范围内。
- 结论：对数增长 $<$ 幂函数增长 $<$ 指数增长 $<$ 阶乘增长 $<$ 幂指增长。
- 使用场合：选择长期预测模型、分析算法复杂度、比较函数远期行为。

- **具体极限形式**：
  $$\lim_{x \to +\infty} \frac{\log_a x}{x^k} = 0 \quad (a > 1,\ k > 0)$$
  $$\lim_{x \to +\infty} \frac{x^k}{a^x} = 0 \quad (a > 1,\ k > 0)$$
- 前提：同上。
- 结论：任何正幂次的幂函数增长都快于对数函数，慢于指数函数。
- 使用场合：严格证明增长速度的差异、极限计算。

**6. 公式**

$$\lim_{x \to +\infty} \frac{\log_a x}{x^k} = 0 \quad (a > 1,\ k > 0)$$
$$\lim_{x \to +\infty} \frac{x^k}{a^x} = 0 \quad (a > 1,\ k > 0)$$
$$\lim_{x \to +\infty} \frac{\log_a x}{a^x} = 0 \quad (a > 1)$$

等价表述：
$$\log_a x = o(x^k) \quad (x \to +\infty)$$
$$x^k = o(a^x) \quad (x \to +\infty)$$

**7. 二级结论**

- 增长速度排序在 $x$ 足够大时才成立，在 $x$ 较小时可能出现 "反转" 现象。例如 $x = 2$ 时，$\log_2 2 = 1 < 2^2 = 4 < 2^2 = 4$；但当 $x = 100$ 时，$\log_2 100 \approx 6.64 \ll 100^2 = 10000 \ll 2^{100} \approx 1.27 \times 10^{30}$。
- 指数函数的增长优势是压倒性的：即使 $x^k$ 中 $k$ 再大，$a^x$ 最终也会超越它。
- 这个排序关系是选择长期预测模型的核心依据——短期可能近似线性或幂函数，长期必须考虑指数效应。

#### 知识点 3.2.3：函数模型的建立

**1. 定义**

数学建模的一般流程：实际问题 $\to$ 分析量与关系 $\to$ 建立函数模型 $\to$ 求解数学问题 $\to$ 检验与修正 $\to$ 实际解答。常用模型有线性、二次、指数、对数、幂函数模型。

**2. 重要性级别**：A

**3. 特性**

建模四步骤：审题理解明确变量 $\to$ 选择模型根据增长特征 $\to$ 确定参数列方程组 $\to$ 求解检验还原实际答案；实际问题中定义域有实际意义限制。

**4. 运算方式**

- **审题与假设**：明确问题中的自变量和因变量，确定关键参数，作出合理简化假设。
- **选择模型**：根据数据特征或理论背景选择合适模型：
  - 匀速变化 $\to$ 线性 $y = kx + b$
  - 等比变化 $\to$ 指数 $y = ab^x$ 或 $y = ae^{kx}$
  - 减速变化 $\to$ 对数 $y = a \ln x + b$
  - 面积/体积关系 $\to$ 幂函数 $y = ax^n$
  - 先增后减有峰值 $\to$ 二次函数 $y = ax^2 + bx + c$（$a < 0$）

- **确定参数**：将已知数据代入模型，建立关于参数的方程组，解出参数。
- **检验与修正**：将求得的结果代入检验，若不符合实际则修正模型或重新定义域。

**5. 定理**

- **模型存在性原则**：在实际问题的合理简化下，任何具有确定性因果关系的现象都可以用适当的函数模型描述。
- 前提：变量间存在确定性关系，忽略随机因素。
- 结论：存在函数 $y = f(x)$ 描述变量关系。
- 使用场合：建模的哲学基础、判断问题是否可建模。

**6. 公式**

常用模型形式：
$$\text{线性: } y = kx + b$$
$$\text{二次: } y = ax^2 + bx + c \quad (a \neq 0)$$
$$\text{指数: } y = ab^x \text{ 或 } y = ae^{kx}$$
$$\text{对数: } y = a \ln x + b \text{ 或 } y = a \log_c x + b$$
$$\text{幂函数: } y = ax^k \quad (k > 0)$$

**7. 二级结论**

- 实际问题的定义域常受物理限制：时间 $t \geq 0$，人数 $n \in \mathbb{N}^*$，长度、面积、体积为正等。
- 当用指数模型 $y = ae^{kx}$ 拟合数据时，取对数得 $\ln y = \ln a + kx$，可转化为线性回归问题。
- 建模的关键不在于模型多复杂，而在于能否抓住核心变量关系并作合理简化。

#### 知识点 3.2.4：已知函数模型求参数

**1. 定义**

已知函数模型求参数的核心方法是待定系数法：将已知数据代入函数模型得到关于参数的方程组，解方程组求出参数并检验。

**2. 重要性级别**：A

**3. 特性**

线性模型用两组数据代入解二元一次方程组；指数模型取对数转化为线性问题；幂函数模型取对数后求解；最后要代入检验确保模型正确。

**4. 运算方式**

- **线性模型 $y = kx + b$**：代入两组数据 $(x_1, y_1)$、$(x_2, y_2)$，得：
  $$\begin{cases} y_1 = kx_1 + b \\ y_2 = kx_2 + b \end{cases}$$
  解出 $k$ 和 $b$。

- **指数模型 $y = ab^x$**：取对数得 $\ln y = \ln a + x \ln b$，令 $Y = \ln y$，$A = \ln a$，$B = \ln b$，则 $Y = A + Bx$，转化为线性模型求解。

- **幂函数模型 $y = ax^k$**：取对数得 $\ln y = \ln a + k \ln x$，令 $Y = \ln y$，$X = \ln x$，$A = \ln a$，则 $Y = A + kX$，转化为线性模型。

- **对数模型 $y = a \ln x + b$**：代入两组含对数的数据直接求解。

**5. 定理**

- **待定系数法的唯一性定理**：若已知 $n$ 个独立条件，且模型恰有 $n$ 个待定参数，则参数通常可唯一确定（条件方程组可解且解唯一）。
- 前提：条件个数等于参数个数，且条件方程组线性无关。
- 结论：参数有唯一解。
- 使用场合：确定模型参数个数、判断是否需要补充数据。

**6. 公式**

线性模型：
$$\begin{cases} y_1 = kx_1 + b \\ y_2 = kx_2 + b \end{cases} \Rightarrow k = \frac{y_2 - y_1}{x_2 - x_1},\ b = y_1 - kx_1$$

指数模型（取对数后）：
$$y = ab^x \Rightarrow \ln y = \ln a + x \ln b$$
$$\text{令 } Y = \ln y,\ A = \ln a,\ B = \ln b \Rightarrow Y = A + Bx$$

幂函数模型（取对数后）：
$$y = ax^k \Rightarrow \ln y = \ln a + k \ln x$$
$$\text{令 } Y = \ln y,\ X = \ln x,\ A = \ln a \Rightarrow Y = A + kX$$

**7. 二级结论**

- 若数据点超过参数个数，则方程组可能无解（数据有误差），此时需要用最小二乘法等拟合技术求最佳近似参数。
- 指数模型求参数时，注意 $a$ 的还原：$a = e^A$（若用自然对数）或 $a = 10^A$（若用常用对数）。
- 检验模型正确性的标准：求出的参数代入后，模型在所有已知数据点上的误差都足够小。

#### 知识点 3.2.5：数据拟合

**1. 定义**

数据拟合是当实际数据不完全符合某个函数模型时，寻找一个函数使其图像最接近给定数据分布趋势的方法，常用残差平方和 SSE 评价拟合效果。

**2. 重要性级别**：B

**3. 特性**

拟合函数选择策略：大致呈直线分布选线性模型；先增后减有最值选二次模型；增长越来越快选指数模型；增长越来越慢趋于饱和选对数模型；SSE 越小拟合效果越好。

**4. 运算方式**

- **视觉判断**：将数据点绘制在坐标系中，观察大致分布趋势：
  - 直线趋势 $\to$ 线性 $y = kx + b$
  - 抛物线趋势 $\to$ 二次 $y = ax^2 + bx + c$
  - J 形趋势 $\to$ 指数 $y = ab^x$
  - 对数曲线趋势 $\to$ 对数 $y = a \ln x + b$
  - 幂次曲线趋势 $\to$ 幂函数 $y = ax^k$

- **残差分析**：对于候选模型，计算各点的残差 $e_i = y_i - \hat{y}_i$，残差平方和：
  $$SSE = \sum_{i=1}^{n} (y_i - \hat{y}_i)^2$$
  SSE 越小的模型拟合效果越好。

- **最小二乘法**：线性模型 $y = kx + b$ 的最小二乘解为：
  $$k = \frac{n\sum x_i y_i - \sum x_i \sum y_i}{n\sum x_i^2 - (\sum x_i)^2}$$
  $$b = \frac{\sum y_i - k \sum x_i}{n} = \bar{y} - k\bar{x}$$

**5. 定理**

- **最小二乘最优性定理**：在所有线性函数 $y = kx + b$ 中，由最小二乘法求得的参数使残差平方和 SSE 达到最小。
- 前提：数据点 $(x_i, y_i)$（$i = 1, 2, \dots, n$）给定。
- 结论：最小二乘解是线性模型的最优拟合（在 SSE 意义下）。
- 使用场合：线性回归、实验数据分析、趋势预测。

**6. 公式**

残差平方和：
$$SSE = \sum_{i=1}^{n} (y_i - \hat{y}_i)^2$$

线性最小二乘：
$$k = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^{n}(x_i - \bar{x})^2}$$
$$b = \bar{y} - k\bar{x}$$

其中：
$$\bar{x} = \frac{1}{n}\sum_{i=1}^{n}x_i,\quad \bar{y} = \frac{1}{n}\sum_{i=1}^{n}y_i$$

**7. 二级结论**

- 数据拟合不等于精确通过所有点，而是追求整体最优趋势，因此拟合曲线不一定经过任何一个数据点。
- 在实际建模中，模型选择应兼顾简洁性和拟合度——奥卡姆剃刀原则：在拟合效果相近时，选择更简单的模型。
- 残差图分析：若残差随机分布在零线附近，说明模型合适；若残差呈现明显模式（如曲线趋势），则说明模型选择不当，需要更换模型。
