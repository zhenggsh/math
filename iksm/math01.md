### 1.1 集合

集合是数学中最基本的概念之一，是构建现代数学大厦的基石。本节从集合的含义出发，逐步学习元素特性、表示方法、集合关系与运算。

---

#### 知识点 1.1.1：集合的含义

**重要性级别**：A（核心）

**定义**：
一般地，把研究对象统称为**元素**，把一些元素组成的总体叫做**集合**（简称集）。集合是数学中最基本的概念之一，属于不定义的原概念，即无法用更基本的概念来定义，只能通过描述来理解。

例如，全体自然数组成一个集合，方程 $x^2-1=0$ 的所有实数根组成一个集合，某班级所有学生组成一个集合。

**特性**：
- 集合具有**确定性**、**互异性**、**无序性**三大基本特性
- 元素通常用小写字母 $a,b,c,\dots$ 表示
- 集合通常用大写字母 $A,B,C,\dots$ 表示
- 元素与集合的关系用符号 $a \in A$（属于）或 $a \notin A$（不属于）表示

**运算方式**：
集合本身不涉及数值运算，但可以进行关系的判断：
- 判断某个对象是否为集合的元素
- 统计集合中元素的个数（基数），记作 $\text{card}(A)$ 或 $|A|$

**定理**：
- **空集存在定理**：不含任何元素的集合叫做空集，记作 $\varnothing$，空集是客观存在的集合
  - 前提条件：无
  - 使用场合：任何涉及集合的场合都要考虑空集的特殊性

**公式**：
- 元素个数的表示：若集合 $A$ 含有有限个元素，其个数记为 $|A|$ 或 $n(A)$

**二级结论**：
- 空集是任何集合的子集，这是由子集的定义直接推出的：对于任意集合 $A$，$\varnothing \subseteq A$ 恒成立，因为 $\varnothing$ 中没有任何元素需要验证是否属于 $A$

---

#### 知识点 1.1.2：元素的基本特性

**重要性级别**：A（核心）

**定义**：
集合中的元素具有三大基本特性，这是判断一个总体能否构成集合的根本标准：

1. **确定性**：给定一个集合，任何一个对象是否属于这个集合是明确的，不能模棱两可。即对于元素 $a$ 和集合 $A$，$a \in A$ 与 $a \notin A$ 两者必居其一且仅居其一。

2. **互异性**：集合中的任意两个元素都是不同的，不能有重复元素。若已知 $\{a,b,c\}$ 是一个集合，则隐含 $a \neq b$ 且 $b \neq c$ 且 $a \neq c$。

3. **无序性**：集合中的元素没有顺序之分，元素的排列顺序不影响集合本身。例如 $\{1,2,3\} = \{3,2,1\} = \{2,1,3\}$。

**特性**：
- 确定性是集合的本质属性，不满足确定性就不是集合
- 互异性在解题中常用于建立方程，如由 $\{a, a^2\} = \{1, 4\}$ 可得 $a=1$ 或 $a=-2$ 或 $a=2$ 等，需要逐一验证
- 无序性意味着集合相等的判断只与元素种类有关，与顺序无关

**运算方式**：
- 利用互异性求解参数：若集合中含有参数表达式，可由元素不等建立方程或不等式
- 例如：已知 $\{a, 2a-1, a^2\}$ 为集合，则由互异性有 $a \neq 2a-1$，$a \neq a^2$，$2a-1 \neq a^2$

**定理**：
- **集合相等判定定理**：两个集合相等，当且仅当它们含有完全相同的元素
  - 前提条件：两个集合都已明确给出
  - 使用场合：判断集合是否相等，或利用集合相等求参数

**公式**：
- 互异性约束：若集合表示为 $\{f_1(x), f_2(x), \dots, f_n(x)\}$，则需满足 $f_i(x) \neq f_j(x)$（$i \neq j$）

**二级结论**：
- 由确定性可知，＂高个子学生＂不能构成集合，因为＂高个子＂标准不明确；而＂身高超过175cm的学生＂可以构成集合
- 由互异性可知，在解集合相关的参数问题时，求出参数后必须回代检验是否满足互异性，否则容易产生增根

#### 知识点 1.1.3：集合的表示方法

**重要性级别**：A（核心）

**定义**：
集合有多种表示方法，各有适用场景：

1. **列举法**：把集合中的所有元素一一列举出来，并用花括号＂$\{\}$＂括起来。
   - 例如：小于5的自然数集合表示为 $\{0, 1, 2, 3, 4\}$
   - 适用于元素有限且较少的情况

2. **描述法**：用集合中元素的共同特征来表示，格式为 $\{x \in D \mid P(x)\}$，其中 $D$ 是元素的范围，$P(x)$ 是元素满足的条件。
   - 例如：偶数集合表示为 $\{x \in \mathbb{Z} \mid x = 2k, k \in \mathbb{Z}\}$
   - 适用于元素较多或无限的情况

3. **Venn图法**：用平面上的封闭曲线（通常是圆或椭圆）的内部来表示集合。
   - 直观形象，便于理解集合关系和运算
   - 常用于表示集合的交、并、补运算

4. **区间表示法**：实数集合的常用简写方式。
   - 闭区间 $[a, b] = \{x \mid a \leq x \leq b\}$
   - 开区间 $(a, b) = \{x \mid a < x < b\}$
   - 半开区间 $[a, b)$ 或 $(a, b]$
   - 无穷区间 $(-\infty, +\infty)$、$[a, +\infty)$、$(-\infty, b)$ 等

**特性**：
- 列举法直观明了但不适合元素过多的集合
- 描述法精确简洁但不够直观
- Venn图法便于理解关系但不精确
- 区间表示法仅适用于实数集合

**运算方式**：
不同表示法之间可以相互转化：
- 描述法 $\{x \mid 1 \leq x \leq 3\}$ 可转化为区间表示法 $[1, 3]$
- 区间 $(2, 5)$ 可转化为描述法 $\{x \in \mathbb{R} \mid 2 < x < 5\}$

**定理**：
- **区间运算对应定理**：区间的交、并、补运算对应集合的交、并、补运算
  - 前提条件：所讨论的集合都是实数集的子集
  - 使用场合：将集合运算问题转化为区间运算，利用数轴直观求解

**公式**：
- 描述法的一般形式：$\{x \in D \mid P(x)\}$
- 区间与集合的对应：
$$[a, b] = \{x \mid a \leq x \leq b\}, \quad (a, b) = \{x \mid a < x < b\}$$

**二级结论**：
- 区间表示法中，小括号＂$()$＂表示端点不包含，中括号＂$［］$＂表示端点包含；$+\infty$ 和 $-\infty$ 一侧永远用小括号
- 数轴是处理区间运算的有力工具，集合的交集对应数轴上区间的重叠部分，并集对应覆盖部分

#### 知识点 1.1.4：集合间的基本关系

**重要性级别**：A（核心）

**定义**：
集合之间主要有以下基本关系：

1. **子集**：如果集合 $A$ 的每一个元素都是集合 $B$ 的元素，则称 $A$ 是 $B$ 的子集，记作 $A \subseteq B$（或 $B \supseteq A$）。
   - 符号表示：$A \subseteq B \Leftrightarrow \forall x \in A, x \in B$

2. **集合相等**：如果 $A \subseteq B$ 且 $B \subseteq A$，则称集合 $A$ 与集合 $B$ 相等，记作 $A = B$。

3. **真子集**：如果 $A \subseteq B$ 且 $A \neq B$，则称 $A$ 是 $B$ 的真子集，记作 $A \subsetneq B$（或 $A \subsetneqq B$）。

**特性**：
- 空集是任何集合的子集：$\varnothing \subseteq A$
- 空集是任何非空集合的真子集
- 任何集合都是自身的子集：$A \subseteq A$
- 子集关系具有**自反性**（$A \subseteq A$）、**传递性**（若 $A \subseteq B$ 且 $B \subseteq C$，则 $A \subseteq C$）
- 含有 $n$ 个元素的集合有 $2^n$ 个子集、$2^n - 1$ 个真子集、$2^n - 2$ 个非空真子集

**运算方式**：
- 判断子集关系：逐一验证 $A$ 中元素是否都属于 $B$
- 计算子集个数：利用公式 $2^n$

**定理**：
- **子集个数定理**：含有 $n$ 个元素的集合，其子集个数为 $2^n$
  - 前提条件：集合为有限集，元素个数为 $n$
  - 结论：子集个数 $2^n$，真子集个数 $2^n - 1$，非空真子集个数 $2^n - 2$
  - 使用场合：快速计算子集个数，判断集合族的大小
  - 推理依据：每个元素在子集中有＂出现＂或＂不出现＂两种选择，由乘法原理得 $2 \times 2 \times \cdots \times 2 = 2^n$

**公式**：
- 子集个数：若 $|A| = n$，则子集个数为 $2^n$
- 真子集个数：$2^n - 1$
- 非空真子集个数：$2^n - 2$

**二级结论**：
- 含有 $n$ 个元素的集合的所有子集的元素个数之和为 $n \cdot 2^{n-1}$
  - 推理过程：每个元素在 $2^{n-1}$ 个子集中出现（其余 $n-1$ 个元素每个都有两种选择），所以总贡献为 $n \cdot 2^{n-1}$
- 若 $A \subseteq B$ 且 $B \subseteq C$，则 $A \subseteq C$，这是子集传递性的直接应用

#### 知识点 1.1.5：集合的基本运算

**重要性级别**：A（核心）

**定义**：
给定全集 $U$（包含所有研究对象的集合），集合有三种基本运算：

1. **并集**：由所有属于 $A$ 或属于 $B$ 的元素组成的集合，记作 $A \cup B$。
   $$A \cup B = \{x \mid x \in A \text{ 或 } x \in B\}$$

2. **交集**：由所有既属于 $A$ 又属于 $B$ 的元素组成的集合，记作 $A \cap B$。
   $$A \cap B = \{x \mid x \in A \text{ 且 } x \in B\}$$

3. **补集**：由全集 $U$ 中不属于 $A$ 的所有元素组成的集合，记作 $\complement_U A$（或 $\complement A$、$\overline{A}$）。
   $$\complement_U A = \{x \mid x \in U \text{ 且 } x \notin A\}$$

**特性**：
- 并集满足**交换律**和**结合律**：$A \cup B = B \cup A$，$(A \cup B) \cup C = A \cup (B \cup C)$
- 交集满足**交换律**和**结合律**：$A \cap B = B \cap A$，$(A \cap B) \cap C = A \cap (B \cap C)$
- 并集与交集满足**分配律**：
  - $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$
  - $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
- 补集满足**双重补集律**：$\complement_U(\complement_U A) = A$
- 德摩根律：$\complement_U(A \cup B) = \complement_U A \cap \complement_U B$，$\complement_U(A \cap B) = \complement_U A \cup \complement_U B$

**运算方式**：
- 利用数轴进行区间集合的运算
- 利用Venn图进行直观理解和验证
- 利用运算律进行化简和证明

**定理**：
- **德摩根定理**：对于全集 $U$ 的任意子集 $A$、$B$，有：
  $$\complement_U(A \cup B) = \complement_U A \cap \complement_U B, \quad \complement_U(A \cap B) = \complement_U A \cup \complement_U B$$
  - 前提条件：$A$、$B$ 均为全集 $U$ 的子集
  - 使用场合：简化含有补集的复合运算，化简集合表达式
  - 记忆口诀：＂并的补等于补的交，交的补等于补的并＂

**公式**：
- 并集：$A \cup B = \{x \mid x \in A \text{ 或 } x \in B\}$
- 交集：$A \cap B = \{x \mid x \in A \text{ 且 } x \in B\}$
- 补集：$\complement_U A = \{x \mid x \in U \text{ 且 } x \notin A\}$
- 德摩根律：
$$\complement_U(A \cup B) = \complement_U A \cap \complement_U B$$
$$\complement_U(A \cap B) = \complement_U A \cup \complement_U B$$

**二级结论**：
- $A \cup B = A \Leftrightarrow B \subseteq A$；$A \cap B = A \Leftrightarrow A \subseteq B$
  - 推理过程：若 $B \subseteq A$，则 $B$ 的所有元素已在 $A$ 中，所以 $A \cup B = A$；反之亦然
- $A \cup \complement_U A = U$，$A \cap \complement_U A = \varnothing$
  - 推理过程：由补集定义，$\complement_U A$ 包含 $U$ 中所有不属于 $A$ 的元素，两者合起来就是 $U$，没有交集
- 若 $A \subseteq B$，则 $\complement_U B \subseteq \complement_U A$（补集关系反转）
  - 推理过程：$A$ 中的元素都在 $B$ 中，所以不在 $B$ 中的元素一定不在 $A$ 中，即 $\complement_U B$ 的元素都在 $\complement_U A$ 中

### 1.2 函数及其表示

函数是描述客观世界中变量之间依赖关系的数学模型，是高中数学的核心概念。本节从映射出发，逐步建立函数概念，学习定义域、值域的求解以及函数的多种表示方法。

---

#### 知识点 1.2.1：映射的概念

**重要性级别**：A（核心）

**定义**：
设 $A$、$B$ 是两个非空集合，如果按照某种确定的对应关系 $f$，使集合 $A$ 中的**任意一个**元素 $x$，在集合 $B$ 中都有**唯一确定**的元素 $y$ 与之对应，则称 $f: A \to B$ 为从集合 $A$ 到集合 $B$ 的一个**映射**。

其中，$x$ 称为原像，$y$ 称为 $x$ 在映射 $f$ 下的像，记作 $y = f(x)$。

**特性**：
- 映射的三要素：**原像集合 $A$**、**像的集合**（$B$ 或其子集）、**对应法则 $f$**
- $A$ 中每个元素都要有对应，即 $A$ 中无＂剩余＂元素
- 对应必须唯一，即 $A$ 中的一个元素不能对应 $B$ 中的多个元素
- $B$ 中的元素可以没有原像，也可以有多个原像
- 函数是一种特殊的映射（$A$、$B$ 为数集）

**运算方式**：
- 判断对应是否为映射：检查 $A$ 中每个元素是否都有唯一确定的对应
- 求像：给定 $x \in A$，计算 $f(x)$
- 求原像：给定 $y \in B$，求满足 $f(x) = y$ 的所有 $x \in A$

**定理**：
- **映射唯一性定理**：若 $f: A \to B$ 是映射，则对任意 $x \in A$，其像 $f(x) \in B$ 是唯一确定的
  - 前提条件：$f$ 是合法的映射
  - 使用场合：保证映射的像可以唯一确定，是函数定义的基础

**公式**：
- 映射表示：$f: A \to B$，$x \mapsto f(x)$ 或 $y = f(x)$

**二级结论**：
- 若 $f: A \to B$ 是映射，则 $|A|$ 可以大于、等于或小于像集的元素个数，但 $A$ 中每个元素必须有唯一像
- 当 $B$ 中每个元素都有原像时，称 $f$ 为**满射**；当 $A$ 中不同元素对应 $B$ 中不同元素时，称 $f$ 为**单射**；既是单射又是满射的映射称为**一一映射**或**双射**

---

#### 知识点 1.2.2：函数的概念

**重要性级别**：A（核心）

**定义**：
设 $A$、$B$ 是非空的**数集**，如果按照某种确定的对应关系 $f$，使集合 $A$ 中的任意一个数 $x$，在集合 $B$ 中都有唯一确定的数 $f(x)$ 与之对应，则称 $f: A \to B$ 为从集合 $A$ 到集合 $B$ 的一个**函数**，记作：
$$y = f(x), \quad x \in A$$

其中，$x$ 叫做自变量，$x$ 的取值范围 $A$ 叫做函数的**定义域**；与 $x$ 值对应的 $y$ 值叫做函数值，函数值的集合 $\{f(x) \mid x \in A\}$ 叫做函数的**值域**。

**特性**：
- 函数是一种特殊的映射：要求 $A$、$B$ 必须是数集
- 自变量 $x$ 的取值范围称为定义域
- 函数值的集合称为值域，值域是 $B$ 的子集（不一定是 $B$ 本身）
- 对应关系 $f$ 是函数的核心，可以是解析式、图像、表格等多种形式

**运算方式**：
- 求函数值：给定 $x \in A$，计算 $y = f(x)$
- 已知 $y$ 求 $x$：解方程 $f(x) = y$

**定理**：
- **函数确定性定理**：给定函数 $f$ 和定义域内的自变量 $x$，函数值 $f(x)$ 是唯一确定的
  - 前提条件：$x$ 在定义域内
  - 使用场合：保证函数值可以唯一计算，是函数应用的基础

**公式**：
- 函数的一般形式：$y = f(x)$，$x \in D$（$D$ 为定义域）
- 函数值的计算：$f(a)$ 表示当 $x = a$ 时的函数值

**二级结论**：
- 判断一个对应关系是否为函数，关键是看定义域内的每个 $x$ 是否对应**唯一**的 $y$ 值。例如 $x^2 + y^2 = 1$ 不是函数，因为一个 $x$ 可能对应两个 $y$ 值
- 函数 $y = f(x)$ 中，$f$ 是对应关系，$f(x)$ 是函数值，二者概念不同但常混用

---

#### 知识点 1.2.3：函数的三要素

**重要性级别**：A（核心）

**定义**：
函数的三要素是**定义域**、**值域**和**对应关系**（对应法则）。

1. **定义域**：自变量 $x$ 的取值范围，即集合 $A$。定义域是函数的首要要素，不同的定义域意味着不同的函数。

2. **值域**：函数值 $f(x)$ 的集合，即 $\{f(x) \mid x \in A\}$。值域由定义域和对应关系共同决定。

3. **对应关系**：$x$ 与 $y$ 之间的映射规则，即如何由 $x$ 计算得到 $y$。

**特性**：
- 值域由定义域和对应关系**共同决定**，知道了定义域和对应关系，值域就确定了
- 判断两个函数是否相等：**只需看定义域和对应关系是否相同**（值域自然相同）
- 例如 $f(x) = x$（$x \in \mathbb{R}$）与 $g(x) = x$（$x \in \mathbb{N}$）不是同一函数，因为定义域不同

**运算方式**：
- 确定函数的定义域：根据解析式中各部分的限制条件求交集
- 确定函数的值域：根据定义域和对应关系计算函数值的范围
- 比较两个函数是否相等：比较定义域和对应关系

**定理**：
- **函数相等判定定理**：两个函数 $f$ 和 $g$ 相等，当且仅当它们的定义域相同且对应关系相同（即对定义域内任意 $x$，都有 $f(x) = g(x)$）
  - 前提条件：两个函数都已给出明确的定义域和对应关系
  - 使用场合：判断两个函数是否为同一函数，或证明函数相等

**公式**：
- 函数相等：$f = g \Leftrightarrow D_f = D_g \text{ 且 } \forall x \in D_f, f(x) = g(x)$

**二级结论**：
- 若两个函数的对应关系相同但定义域不同，则它们不是同一函数。例如 $f(x) = x^2$（$x \in \mathbb{R}$）与 $g(x) = x^2$（$x \in [0, +\infty)$）值域不同，前者为 $[0, +\infty)$，后者也为 $[0, +\infty)$，但定义域不同仍不是同一函数
- 改变函数的定义域，函数图像会相应改变（截断或延伸），但对应关系不变

---

#### 知识点 1.2.4：函数定义域的求解

**重要性级别**：A（核心）

**定义**：
求函数定义域的原则是找出使函数解析式有意义的自变量取值范围。常见约束条件包括：

1. **分式的分母不为零**：若 $f(x) = \frac{g(x)}{h(x)}$，则要求 $h(x) \neq 0$
2. **偶次根号下非负**：若 $f(x) = \sqrt[n]{g(x)}$（$n$ 为偶数），则要求 $g(x) \geq 0$
3. **对数的真数大于零**：若 $f(x) = \log_a g(x)$，则要求 $g(x) > 0$
4. **零次幂的底数不为零**：若 $f(x) = [g(x)]^0$，则要求 $g(x) \neq 0$
5. **实际问题中的实际意义**：如人数必须为正整数等

**特性**：
- 定义域必须用**集合**或**区间**表示，不能用不等式表示
- 多个限制条件取**交集**
- 复合函数定义域要注意内层函数的值域与外层函数定义域的对应关系
- 抽象函数 $f(g(x))$ 的定义域求解需注意：已知 $f(x)$ 定义域为 $D$，则 $f(g(x))$ 中 $g(x) \in D$

**运算方式**：
- 列不等式组：根据各类限制条件列出不等式
- 求交集：解不等式组，取各不等式解集的交集
- 区间表示：将结果用区间或集合表示

**定理**：
- **复合函数定义域定理**：若函数 $f(x)$ 的定义域为 $D$，则复合函数 $f(g(x))$ 的定义域为 $\{x \mid g(x) \in D\}$
  - 前提条件：$f$ 和 $g$ 都是合法函数
  - 使用场合：求解复合函数的定义域
  - 注意：是 $g(x)$ 的值落在 $f$ 的定义域中，而不是 $x$ 直接属于 $f$ 的定义域

**公式**：
- 分式：$y = \frac{1}{f(x)}$ 的定义域为 $\{x \mid f(x) \neq 0\}$
- 偶次根式：$y = \sqrt{f(x)}$ 的定义域为 $\{x \mid f(x) \geq 0\}$
- 对数：$y = \log_a f(x)$ 的定义域为 $\{x \mid f(x) > 0\}$
- 正切：$y = \tan x$ 的定义域为 $\{x \mid x \neq k\pi + \frac{\pi}{2}, k \in \mathbb{Z}\}$

**二级结论**：
- 已知 $f(x)$ 定义域为 $[a, b]$，则 $f(g(x))$ 的定义域由 $a \leq g(x) \leq b$ 解出
- 已知 $f(g(x))$ 的定义域为 $[a, b]$（即 $x \in [a, b]$），则 $f(x)$ 的定义域是 $g(x)$ 在 $x \in [a, b]$ 上的值域

---

#### 知识点 1.2.5：函数值域的求解

**重要性级别**：A（核心）

**定义**：
求函数值域的常用方法有：

1. **观察法**：通过观察函数类型直接得出值域。如一次函数 $y = kx + b$（$k \neq 0$）值域为 $\mathbb{R}$。

2. **配方法**：主要用于二次函数。将 $y = ax^2 + bx + c$ 配方为 $y = a(x - h)^2 + k$，根据开口方向确定最值。

3. **判别式法**：对于形如 $y = \frac{a_1 x^2 + b_1 x + c_1}{a_2 x^2 + b_2 x + c_2}$ 的分式函数，将式子变形为关于 $x$ 的方程，利用判别式 $\Delta \geq 0$ 求 $y$ 的范围。

4. **换元法**：对于含根式等复杂形式的函数，设新变量简化表达式，注意换元后新变量的取值范围。

5. **分离常数法**：对于分式函数 $y = \frac{ax + b}{cx + d}$（$c \neq 0$），分离常数后利用反比例函数性质求值域。

6. **单调性法**：利用函数在区间上的单调性求最值，从而确定值域。

7. **图像法**：通过画出函数图像，观察图像在 $y$ 轴上的投影范围。

**特性**：
- 一次函数值域为 $\mathbb{R}$（定义域为 $\mathbb{R}$ 时）
- 二次函数值域用配方法最方便
- 分式函数可尝试分离常数法
- 含根式可考虑换元法

**运算方式**：
- 配方：$y = a(x - h)^2 + k$，当 $a > 0$ 时最小值为 $k$，值域为 $[k, +\infty)$
- 分离常数：$y = \frac{ax + b}{cx + d} = \frac{a}{c} + \frac{b - \frac{ad}{c}}{cx + d}$
- 换元：令 $t = \sqrt{x}$ 或 $t = g(x)$，转化为已知函数求值域

**定理**：
- **二次函数值域定理**：二次函数 $f(x) = ax^2 + bx + c$（$a \neq 0$），当定义域为 $\mathbb{R}$ 时：
  - 若 $a > 0$，值域为 $\left[\frac{4ac - b^2}{4a}, +\infty\right)$
  - 若 $a < 0$，值域为 $\left(-\infty, \frac{4ac - b^2}{4a}\right]$
  - 前提条件：定义域为全体实数
  - 使用场合：快速确定二次函数在 $\mathbb{R}$ 上的值域

**公式**：
- 二次函数配方：
$$y = ax^2 + bx + c = a\left(x + \frac{b}{2a}\right)^2 + \frac{4ac - b^2}{4a}$$
- 顶点纵坐标：$y_{\min/\max} = \frac{4ac - b^2}{4a}$
- 分离常数：
$$y = \frac{ax + b}{cx + d} = \frac{a}{c} + \frac{bc - ad}{c(cx + d)} \quad (c \neq 0)$$

**二级结论**：
- 反比例函数 $y = \frac{k}{x}$（$k \neq 0$）的值域为 $(-\infty, 0) \cup (0, +\infty)$，即 $y \neq 0$
  - 推理过程：由 $y = \frac{k}{x}$ 得 $x = \frac{k}{y}$，$x$ 存在当且仅当 $y \neq 0$
- 函数 $y = \frac{ax + b}{cx + d}$（$c \neq 0$，$ad \neq bc$）的值域为 $y \neq \frac{a}{c}$
  - 推理过程：分离常数后 $y = \frac{a}{c} + \frac{bc - ad}{c(cx + d)}$，由于分式部分不能为零，故 $y \neq \frac{a}{c}$

#### 知识点 1.2.6：函数的表示法

**重要性级别**：A（核心）

**定义**：
函数的表示方法主要有三种：

1. **解析法（公式法）**：用数学表达式表示函数关系。
   - 例如：$f(x) = 2x + 1$，$x \in \mathbb{R}$
   - 优点：精确，便于进行理论分析和运算
   - 缺点：不够直观，有些实际问题难以用解析式表示

2. **列表法**：列出 $x$ 与 $y$ 的对应值表来表示函数。
   - 例如：平方表、三角函数表等
   - 优点：直接明了，不需要计算即可查得函数值
   - 缺点：不完整，只能表示有限个对应值

3. **图像法**：用平面直角坐标系中的图形（通常是曲线）来表示函数。
   - 例如：二次函数的抛物线图像
   - 优点：直观形象，便于观察函数的变化趋势和性质
   - 缺点：不精确，只能得到近似值

**特性**：
- 三种方法各有优缺点，常结合使用
- 解析法是理论研究的主要工具
- 图像法是理解函数性质的直观手段
- 列表法在实际应用中仍有价值（如统计表格）

**运算方式**：
- 由图像判断函数性质：观察增减性、对称性、最值等
- 由解析式画图像：列表、描点、连线
- 由列表数据求近似解析式：数据拟合

**定理**：
- **函数图像判定定理**：在平面直角坐标系中，一个图像表示函数的充要条件是：任意垂直于 $x$ 轴的直线与图像至多有一个交点（垂直线检验）
  - 前提条件：图像在坐标系中已给出
  - 使用场合：判断给定图像是否表示函数

**公式**：
- 函数图像上点的坐标满足函数解析式：点 $(x_0, y_0)$ 在函数 $y = f(x)$ 图像上 $\Leftrightarrow y_0 = f(x_0)$

**二级结论**：
- 由解析式画图的基本步骤：**列表**（取若干 $x$ 值计算 $y$）→ **描点**（在坐标系中标出各点）→ **连线**（用光滑曲线连接各点）
- 若点 $(a, b)$ 在函数 $y = f(x)$ 图像上，则 $b = f(a)$；反之，若 $b = f(a)$，则点 $(a, b)$ 在该函数图像上

#### 知识点 1.2.7：分段函数

**重要性级别**：A（核心）

**定义**：
在函数定义域内，对于自变量 $x$ 的不同取值范围有着不同的对应关系的函数叫做**分段函数**。

例如：
$$f(x) = \begin{cases} x + 1, & x \leq 0 \\ x^2, & x > 0 \end{cases}$$

**重要强调**：分段函数是一个函数，而不是多个函数。它的对应关系是统一的，只是在不同区间上用不同的表达式表示。

**特性**：
- 分段函数的图像由几段曲线（或直线）组成，可能在分界点处连接或不连接
- 求分段函数值时，要先确定 $x$ 所在的区间，再选择对应的表达式计算
- 分段函数的定义域是各段定义域的**并集**
- 分界点处的函数值要特别注意：可能只由某一段定义，也可能两段给出相同值（连续）或不同值（间断）

**运算方式**：
- 求函数值：先判断 $x$ 属于哪个区间，再代入对应表达式
- 求定义域：各段定义域的并集
- 画图像：在各段区间内分别画出对应函数的图像，注意分界点处的实心点或空心点

**定理**：
- **分段函数连续性判定定理**：若分段函数 $f(x)$ 在各段区间内连续，且在分界点 $x = a$ 处有 $\lim_{x \to a^-} f(x) = \lim_{x \to a^+} f(x) = f(a)$，则 $f(x)$ 在 $x = a$ 处连续
  - 前提条件：各段表达式在对应区间内连续
  - 使用场合：判断分段函数在分界点处是否连续

**公式**：
- 分段函数的一般形式：
$$f(x) = \begin{cases} f_1(x), & x \in D_1 \\ f_2(x), & x \in D_2 \\ \vdots \\ f_n(x), & x \in D_n \end{cases}$$
其中 $D_1 \cup D_2 \cup \cdots \cup D_n = D$（总定义域），且各段定义域互不相交

**二级结论**：
- 绝对值函数 $y = |x|$ 是最常见的分段函数：$y = \begin{cases} x, & x \geq 0 \\ -x, & x < 0 \end{cases}$
- 取整函数（高斯函数）$y = [x]$ 表示不超过 $x$ 的最大整数，也是分段函数
- 在解分段函数相关问题时，**分类讨论**是关键思想，要按照分界点将问题分成若干区间分别处理

### 1.3 函数的基本性质

函数的单调性、最值和奇偶性是函数的三个基本性质，它们反映了函数图像的变化趋势、取值范围和形态特征。掌握这些性质对于分析和解决函数相关问题至关重要。

---

#### 知识点 1.3.1：函数的单调性

**重要性级别**：A（核心）

**定义**：
一般地，设函数 $f(x)$ 的定义域为 $I$，如果对于定义域 $I$ 内某个区间 $D$ 上的**任意**两个自变量的值 $x_1$、$x_2$，当 $x_1 < x_2$ 时：

- 都有 $f(x_1) < f(x_2)$，则称 $f(x)$ 在区间 $D$ 上是**增函数**（单调递增）
- 都有 $f(x_1) > f(x_2)$，则称 $f(x)$ 在区间 $D$ 上是**减函数**（单调递减）

如果函数在区间 $D$ 上是增函数或减函数，则称函数在区间 $D$ 上具有**单调性**，区间 $D$ 称为函数的**单调区间**。

**特性**：
- 增函数的图像从左向右**上升**
- 减函数的图像从左向右**下降**
- 单调性是函数的**局部性质**，是针对某个区间而言的，不能脱离区间谈单调性
- 有些函数在整个定义域上单调（如一次函数），有些函数在不同区间有不同单调性（如二次函数）
- 单调区间必须写成区间形式，多个单调区间之间用＂和＂或逗号连接，不能用并集符号

**运算方式**：
- 定义法判断单调性：任取 $x_1 < x_2$，计算 $f(x_1) - f(x_2)$，判断符号
- 图像法：观察图像走向

**定理**：
- **单调性定义定理**：设 $f(x)$ 在区间 $D$ 上有定义，则：
  - $f(x)$ 在 $D$ 上递增 $\Leftrightarrow \forall x_1, x_2 \in D, x_1 < x_2 \Rightarrow f(x_1) < f(x_2)$
  - $f(x)$ 在 $D$ 上递减 $\Leftrightarrow \forall x_1, x_2 \in D, x_1 < x_2 \Rightarrow f(x_1) > f(x_2)$
  - 前提条件：$x_1, x_2$ 是区间 $D$ 上的**任意**两个值
  - 使用场合：证明函数的单调性，是定义法的理论依据

**公式**：
- 增函数：$x_1 < x_2 \Rightarrow f(x_1) < f(x_2)$
- 减函数：$x_1 < x_2 \Rightarrow f(x_1) > f(x_2)$
- 等价变形：$\frac{f(x_1) - f(x_2)}{x_1 - x_2} > 0 \Leftrightarrow f(x)$ 递增；$\frac{f(x_1) - f(x_2)}{x_1 - x_2} < 0 \Leftrightarrow f(x)$ 递减

**二级结论**：
- 若 $f(x)$ 在 $D$ 上递增，则 $-f(x)$ 在 $D$ 上递减
  - 推理过程：设 $x_1 < x_2$，则 $f(x_1) < f(x_2)$，于是 $-f(x_1) > -f(x_2)$，即 $-f(x)$ 递减
- 若 $f(x)$ 递增且恒正（或恒负），则 $\frac{1}{f(x)}$ 递减
  - 推理过程：$x_1 < x_2 \Rightarrow f(x_1) < f(x_2)$，若 $f(x) > 0$，则 $\frac{1}{f(x_1)} > \frac{1}{f(x_2)}$

---

#### 知识点 1.3.2：单调性的判断

**重要性级别**：A（核心）

**定义**：
判断函数单调性的常用方法有：

1. **定义法（作差法）**：任取 $x_1 < x_2$，作差 $f(x_1) - f(x_2)$，通过因式分解、配方等方法判断符号。
   - 若 $f(x_1) - f(x_2) < 0$，则函数递增
   - 若 $f(x_1) - f(x_2) > 0$，则函数递减

2. **图像法**：通过观察函数图像的走向来判断单调性。
   - 图像上升则递增，图像下降则递减

3. **性质法**：利用已知函数的单调性通过运算得到新函数的单调性。
   - 增 $+$ 增 $=$ 增
   - 减 $+$ 减 $=$ 减
   - $-$ 增 $=$ 减（即增函数前加负号变为减函数）
   - $-$ 减 $=$ 增
   - 增 $-$ 减 $=$ 增
   - 减 $-$ 增 $=$ 减

4. **复合函数法**（后续学习）：同增异减原则

**特性**：
- 定义法是判断单调性的根本方法，适用于所有函数
- 性质法可以快速判断由基本函数通过加减运算得到的新函数的单调性
- 实际问题中常结合图像观察
- 注意：增 $\times$ 增 不一定是增，不能随意推广乘法运算

**运算方式**：
- 定义法步骤：任取 $x_1 < x_2$ $\to$ 计算 $f(x_1) - f(x_2)$ $\to$ 变形判断符号 $\to$ 得出结论
- 性质法：将复杂函数拆分为基本函数的组合，利用已知性质推导

**定理**：
- **单调性运算定理**：若 $f(x)$、$g(x)$ 在同一区间上单调，则：
  - $f(x) + g(x)$ 与 $f(x)$、$g(x)$ 同向单调（同增同减）
  - $-f(x)$ 与 $f(x)$ 反向单调
  - 前提条件：$f$、$g$ 在同一区间上有定义且分别单调
  - 使用场合：快速判断和函数的单调性，简化分析过程

**公式**：
- 基本初等函数的单调性：
  - 一次函数 $y = kx + b$：$k > 0$ 时递增，$k < 0$ 时递减
  - 二次函数 $y = ax^2 + bx + c$：$a > 0$ 时在 $(-\infty, -\frac{b}{2a}]$ 递减，在 $[-\frac{b}{2a}, +\infty)$ 递增
  - 反比例函数 $y = \frac{k}{x}$：$k > 0$ 时在 $(-\infty, 0)$ 和 $(0, +\infty)$ 上分别递减

**二级结论**：
- 若 $f(x)$ 在区间 $D$ 上递增，$g(x)$ 在区间 $D$ 上递减，则 $f(x) - g(x)$ 在 $D$ 上递增
  - 推理过程：$-g(x)$ 递增，增 $+$ 增 $=$ 增
- 函数 $y = x + \frac{k}{x}$（$k > 0$）在 $(0, \sqrt{k}]$ 上递减，在 $[\sqrt{k}, +\infty)$ 上递增（对勾函数性质）

---

#### 知识点 1.3.3：函数的最值

**重要性级别**：A（核心）

**定义**：
一般地，设函数 $y = f(x)$ 的定义域为 $I$，如果存在实数 $M$ 满足：

1. 对于任意的 $x \in I$，都有 $f(x) \leq M$
2. 存在 $x_0 \in I$，使得 $f(x_0) = M$

则称 $M$ 是函数 $y = f(x)$ 的**最大值**。

类似地，如果存在实数 $m$ 满足对于任意的 $x \in I$ 都有 $f(x) \geq m$，且存在 $x_0 \in I$ 使 $f(x_0) = m$，则称 $m$ 是函数的**最小值**。

**特性**：
- 最值是函数值域的**边界值**
- 最大值和最小值不一定同时存在
- 闭区间上的连续函数必有最大值和最小值（最值存在定理）
- 最值是一个**函数值**（数），而不是一个点
- 求最值的常用方法：配方法、单调性法、图像法、判别式法等

**运算方式**：
- 配方法求二次函数最值：$f(x) = a(x - h)^2 + k$，顶点处取最值
- 单调性法：在闭区间上，单调函数的最值在端点处取得
- 图像法：观察图像的最高点和最低点

**定理**：
- **闭区间最值存在定理**：若函数 $f(x)$ 在闭区间 $[a, b]$ 上连续，则 $f(x)$ 在 $[a, b]$ 上必有最大值和最小值
  - 前提条件：闭区间 $[a, b]$，函数在该区间上连续
  - 使用场合：保证最值的存在性，是求最值的理论基础
  - 注意：开区间上的连续函数不一定有最值

**公式**：
- 二次函数 $f(x) = ax^2 + bx + c$（$a \neq 0$）在 $\mathbb{R}$ 上的最值：
$$M = \frac{4ac - b^2}{4a} \text{（} a < 0 \text{ 时为最大值，} a > 0 \text{ 时为最小值）}$$
- 二次函数在闭区间 $[m, n]$ 上的最值：比较顶点和端点的函数值

**二级结论**：
- 若 $f(x)$ 在 $[a, b]$ 上递增，则最小值为 $f(a)$，最大值为 $f(b)$
  - 推理过程：由递增定义，$a \leq x \leq b \Rightarrow f(a) \leq f(x) \leq f(b)$
- 若 $f(x)$ 在 $[a, b]$ 上递减，则最大值为 $f(a)$，最小值为 $f(b)$
- 二次函数在闭区间上的最值：若对称轴在区间内，则顶点为一个最值，离对称轴较远的端点为另一个最值；若对称轴在区间外，则最值在两个端点处取得

#### 知识点 1.3.4：函数的奇偶性

**重要性级别**：A（核心）

**定义**：
设函数 $f(x)$ 的定义域为 $D$，如果对于定义域 $D$ 内**任意一个** $x$：

- 都有 $-x \in D$（定义域关于原点对称），且 $f(-x) = f(x)$，则称 $f(x)$ 为**偶函数**
- 都有 $-x \in D$（定义域关于原点对称），且 $f(-x) = -f(x)$，则称 $f(x)$ 为**奇函数**

**特性**：
- 偶函数的图像关于**$y$ 轴**对称
- 奇函数的图像关于**原点**对称
- **奇偶性的前提条件是定义域关于原点对称**，若定义域不关于原点对称，则函数既不是奇函数也不是偶函数（称为非奇非偶函数）
- 有些函数既是奇函数又是偶函数：$f(x) = 0$（定义域关于原点对称时）
- 常数函数 $f(x) = c$（$c \neq 0$）是偶函数

**运算方式**：
- 判断奇偶性：先检查定义域是否关于原点对称，再计算 $f(-x)$ 并与 $f(x)$ 比较
- 利用奇偶性简化计算：已知一侧的函数值或表达式，可直接得到另一侧的信息

**定理**：
- **奇偶性图像对称定理**：
  - 函数 $f(x)$ 是偶函数 $\Leftrightarrow$ 图像关于 $y$ 轴对称
  - 函数 $f(x)$ 是奇函数 $\Leftrightarrow$ 图像关于原点对称
  - 前提条件：函数定义域关于原点对称
  - 使用场合：利用图像对称性快速判断奇偶性，或利用奇偶性画函数图像

**公式**：
- 偶函数：$f(-x) = f(x)$，$\forall x \in D$
- 奇函数：$f(-x) = -f(x)$，$\forall x \in D$
- 等价变形：奇函数可写成 $f(-x) + f(x) = 0$；偶函数可写成 $f(-x) - f(x) = 0$

**二级结论**：
- 若奇函数 $f(x)$ 在 $x = 0$ 处有定义，则必有 $f(0) = 0$
  - 推理过程：由奇函数定义，$f(-0) = -f(0)$，即 $f(0) = -f(0)$，所以 $2f(0) = 0$，得 $f(0) = 0$
  - 逆命题不成立：$f(0) = 0$ 不能推出 $f(x)$ 是奇函数（如 $f(x) = x^2$ 满足 $f(0) = 0$ 但是偶函数）
- 若 $f(x)$ 是偶函数且在 $(0, +\infty)$ 上递增，则在 $(-\infty, 0)$ 上递减
  - 推理过程：由对称性，$x_1 < x_2 < 0$ 时，$-x_1 > -x_2 > 0$，$f(-x_1) > f(-x_2)$，即 $f(x_1) > f(x_2)$，所以递减

---

#### 知识点 1.3.5：奇偶性的判断

**重要性级别**：A（核心）

**定义**：
判断函数奇偶性的标准步骤：

**第一步：判断定义域是否关于原点对称**
- 若定义域不关于原点对称，则函数**非奇非偶**，判断结束
- 若定义域关于原点对称，进入第二步

**第二步：计算 $f(-x)$ 并与 $f(x)$ 比较**
- 若 $f(-x) = f(x)$，则函数为**偶函数**
- 若 $f(-x) = -f(x)$，则函数为**奇函数**
- 若以上两者都不满足，则函数**非奇非偶**

**特性**：
- 定义域不关于原点对称 $\Rightarrow$ 非奇非偶（这是排除法的关键）
- $f(-x) = f(x)$ $\Rightarrow$ 偶函数
- $f(-x) = -f(x)$ $\Rightarrow$ 奇函数
- 两者都不满足 $\Rightarrow$ 非奇非偶
- 两者都满足（且 $f(x)$ 不恒为零）的情况不可能出现，但 $f(x) = 0$（定义域对称时）既是奇函数又是偶函数

**运算方式**：
- 判断定义域对称性：若 $x \in D$ 时必有 $-x \in D$，则 $D$ 关于原点对称
- 计算 $f(-x)$：将解析式中所有 $x$ 替换为 $-x$，化简后与 $f(x)$ 比较

**定理**：
- **奇偶性判断定理**：设函数 $f(x)$ 的定义域为 $D$，则：
  - 若 $D$ 关于原点对称且 $f(-x) = f(x)$，则 $f(x)$ 为偶函数
  - 若 $D$ 关于原点对称且 $f(-x) = -f(x)$，则 $f(x)$ 为奇函数
  - 前提条件：定义域 $D$ 必须关于原点对称
  - 使用场合：判断具体函数的奇偶性

**公式**：
- 定义域关于原点对称的判定：$D = -D$（即 $x \in D \Leftrightarrow -x \in D$）

**二级结论**：
- 判断定义域是否关于原点对称的快速方法：看定义域区间是否成对出现（如 $(-a, a)$、$[-a, a]$、$(-\infty, +\infty)$ 等）
- 若定义域包含 $x = a$ 但不包含 $x = -a$（$a \neq 0$），则定义域不对称，直接判定非奇非偶
- 多项式函数中，只含偶次项的为偶函数，只含奇次项的为奇函数

---

#### 知识点 1.3.6：函数性质的运算规律

**重要性级别**：B（重要）

**定义**：
函数的单调性和奇偶性在函数运算下有以下规律：

**单调性运算规律**：
- 增 $+$ 增 $=$ 增（两个增函数之和为增函数）
- 减 $+$ 减 $=$ 减（两个减函数之和为减函数）
- $-$ 增 $=$ 减（增函数乘以 $-1$ 为减函数）
- $-$ 减 $=$ 增（减函数乘以 $-1$ 为增函数）
- 增 $-$ 减 $=$ 增
- 减 $-$ 增 $=$ 减

**奇偶性运算规律**：
- 奇 $\pm$ 奇 $=$ 奇
- 偶 $\pm$ 偶 $=$ 偶
- 奇 $\times$ 偶 $=$ 奇
- 偶 $\times$ 偶 $=$ 偶
- 奇 $\times$ 奇 $=$ 偶
- 奇 $\div$ 偶 $=$ 奇（分母不为零）

**特性**：
- 奇函数在 $x = 0$ 处有定义时，必有 $f(0) = 0$
- 奇函数在关于原点对称的区间上单调性**相同**
- 偶函数在关于原点对称的区间上单调性**相反**
- 上述运算规律的前提：各函数在公共区间上有定义且分别有相应的奇偶性

**运算方式**：
- 判断和函数的单调性/奇偶性：利用上述运算规律
- 分解复杂函数：将复杂函数拆分为基本函数的组合，分别分析性质

**定理**：
- **奇偶性运算定理**：设 $f(x)$、$g(x)$ 在关于原点对称的公共定义域上有定义，则：
  - 若 $f$、$g$ 同为奇函数，则 $f \pm g$ 为奇函数，$f \cdot g$ 为偶函数
  - 若 $f$、$g$ 同为偶函数，则 $f \pm g$ 为偶函数，$f \cdot g$ 为偶函数
  - 若 $f$ 为奇函数、$g$ 为偶函数，则 $f \cdot g$ 为奇函数
  - 前提条件：公共定义域关于原点对称
  - 使用场合：判断复合函数的奇偶性，简化分析

**公式**：
- 单调性：设 $f$、$g$ 在区间 $D$ 上分别单调，则：
  - $f + g$ 与 $f$、$g$ 同向（同增同减）
  - $-f$ 与 $f$ 反向

**二级结论**：
- 奇函数 $\pm$ 偶函数的结果一般**非奇非偶**
  - 推理过程：设 $h = f_{奇} + g_{偶}$，则 $h(-x) = f_{奇}(-x) + g_{偶}(-x) = -f_{奇}(x) + g_{偶}(x)$，既不等于 $h(x)$ 也不等于 $-h(x)$（除非其中一个为零函数）
- 若 $f(x)$ 是偶函数，则 $f(x) = f(|x|)$，这一性质常用于简化偶函数的问题
  - 推理过程：$x \geq 0$ 时 $|x| = x$，显然成立；$x < 0$ 时 $|x| = -x$，由偶函数性质 $f(x) = f(-x) = f(|x|)$

#### 知识点 2.1.1：指数幂的运算

**重要性级别**：A（核心）

**定义**：
**n次方根**：如果 $x^n = a$（$n \in \mathbb{N}^*$ 且 $n > 1$），则称 $x$ 为 $a$ 的 $n$ 次方根。

- 当 $n$ 为奇数时，正数的 $n$ 次方根为正数，负数的 $n$ 次方根为负数，记作 $\sqrt[n]{a}$
- 当 $n$ 为偶数时，正数的 $n$ 次方根有两个（互为相反数），记作 $\pm\sqrt[n]{a}$；负数没有实数 $n$ 次方根
- 零的 $n$ 次方根为零

**分数指数幂**：
- 正分数指数幂：$a^{\frac{m}{n}} = \sqrt[n]{a^m}$（$a > 0$，$m, n \in \mathbb{N}^*$，$n > 1$）
- 负分数指数幂：$a^{-\frac{m}{n}} = \frac{1}{a^{\frac{m}{n}}} = \frac{1}{\sqrt[n]{a^m}}$（$a > 0$）
- 零的正分数指数幂为零，零的负分数指数幂无意义

**特性**：
- $n$ 为奇数时，$\sqrt[n]{a^n} = a$
- $n$ 为偶数时，$\sqrt[n]{a^n} = |a|$
- 指数幂的运算将整数指数幂的性质推广到了有理数指数幂

**运算方式**：
指数幂的运算性质（$a > 0$，$b > 0$，$r, s \in \mathbb{Q}$）：
1. $a^r \cdot a^s = a^{r + s}$（同底数幂相乘，底数不变，指数相加）
2. $(a^r)^s = a^{r \cdot s}$（幂的乘方，底数不变，指数相乘）
3. $(ab)^r = a^r \cdot b^r$（积的乘方，等于各因式乘方的积）

**定理**：
- **指数幂运算定理**：设 $a > 0$，$b > 0$，$r, s \in \mathbb{Q}$，则上述三条运算性质成立
  - 前提条件：底数为正数，指数为有理数
  - 使用场合：化简指数表达式，进行指数运算
  - 注意：底数为负时，分数指数幂可能无意义或产生复数

**公式**：
- $a^r \cdot a^s = a^{r + s}$，$a > 0$，$r, s \in \mathbb{Q}$
- $(a^r)^s = a^{rs}$，$a > 0$，$r, s \in \mathbb{Q}$
- $(ab)^r = a^r b^r$，$a > 0$，$b > 0$，$r \in \mathbb{Q}$
- $a^{\frac{m}{n}} = \sqrt[n]{a^m}$，$a > 0$
- $a^{-\frac{m}{n}} = \frac{1}{\sqrt[n]{a^m}}$，$a > 0$

**二级结论**：
- $\sqrt[n]{a^n} = \begin{cases} a, & n \text{ 为奇数} \\ |a|, & n \text{ 为偶数} \end{cases}$
  - 推理过程：$n$ 为偶数时，$\sqrt[n]{a^n}$ 表示 $a^n$ 的正 $n$ 次方根，而 $|a|$ 的 $n$ 次方等于 $a^n$，所以 $\sqrt[n]{a^n} = |a|$
- $a^0 = 1$（$a \neq 0$），这是指数运算性质的推论：$a^0 = a^{r - r} = \frac{a^r}{a^r} = 1$

---

#### 知识点 2.1.2：指数函数的概念

**重要性级别**：A（核心）

**定义**：
一般地，函数 $y = a^x$（$a > 0$ 且 $a \neq 1$，$x \in \mathbb{R}$）叫做**指数函数**，其中 $x$ 为自变量，定义域为 $\mathbb{R}$。

**限制条件的理解**：
- $a > 0$：保证对任意实数 $x$，$a^x$ 都有意义（若 $a < 0$，则如 $a^{\frac{1}{2}}$ 无意义）
- $a \neq 1$：避免得到常函数 $y = 1^x = 1$，失去研究价值
- $a = 0$ 时 $0^x$ 在 $x \leq 0$ 时无意义，故排除

**特性**：
- 指数函数与幂函数的区别：
  - 指数函数：底数为**常数**，指数为**变量**，即 $y = a^x$
  - 幂函数：底数为**变量**，指数为**常数**，即 $y = x^a$
- 指数函数的定义域为 $\mathbb{R}$
- 指数函数的值域为 $(0, +\infty)$

**运算方式**：
- 求函数值：给定 $x$，计算 $a^x$
- 比较大小：利用指数函数的单调性
- 解指数方程：将两边化为同底数幂，利用指数函数的单调性

**定理**：
- **指数函数定义定理**：形如 $y = a^x$（$a > 0$ 且 $a \neq 1$）的函数称为指数函数，其定义域为 $\mathbb{R}$
  - 前提条件：$a > 0$ 且 $a \neq 1$
  - 使用场合：判断给定函数是否为指数函数
  - 注意：$y = 2 \cdot 3^x$ 不是纯粹的指数函数（有系数），$y = 3^{2x}$ 可化为 $y = (3^2)^x = 9^x$ 是指数函数

**公式**：
- 指数函数的标准形式：$y = a^x$（$a > 0$ 且 $a \neq 1$）

**二级结论**：
- 指数函数 $y = a^x$ 与 $y = a^{-x} = (\frac{1}{a})^x$ 的图像关于 $y$ 轴对称
  - 推理过程：$y = a^{-x}$ 即 $y = (a^{-1})^x$，而点 $(x, a^x)$ 关于 $y$ 轴的对称点为 $(-x, a^x) = (-x, a^{-(-x)})$，在 $y = a^{-x}$ 上
- 若函数 $y = c \cdot a^x$（$c \neq 0$，$a > 0$ 且 $a \neq 1$），称为指数型函数，但不是标准指数函数

---

#### 知识点 2.1.3：指数函数的图像

**重要性级别**：A（核心）

**定义**：
指数函数 $y = a^x$（$a > 0$ 且 $a \neq 1$）的图像特征：

- 当 $a > 1$ 时：图像从左下向右上单调上升，呈增长趋势
- 当 $0 < a < 1$ 时：图像从左上向右下单调下降，呈衰减趋势

**图像共同特征**：
- 所有指数函数图像都在 $x$ 轴**上方**，即 $y > 0$
- 所有指数函数图像都过定点 $(0, 1)$，因为 $a^0 = 1$
- $x$ 轴（即直线 $y = 0$）是图像的水平渐近线
- 图像不与 $x$ 轴相交

**特性**：
- $a > 1$ 时：$x > 0$ 则 $y > 1$；$x < 0$ 则 $0 < y < 1$
- $0 < a < 1$ 时：$x > 0$ 则 $0 < y < 1$；$x < 0$ 则 $y > 1$
- 底数 $a$ 越大（$a > 1$ 时），图像上升越陡峭；底数 $a$ 越小（$0 < a < 1$ 时），图像下降越平缓
- 在第一象限内，底数越大的指数函数图像越靠上方

**运算方式**：
- 利用图像判断指数函数的单调性和取值范围
- 利用图像比较不同指数函数的大小关系

**定理**：
- **指数函数图像定点定理**：指数函数 $y = a^x$（$a > 0$ 且 $a \neq 1$）的图像必过定点 $(0, 1)$
  - 前提条件：$a > 0$ 且 $a \neq 1$
  - 使用场合：快速确定指数函数图像的位置，求参数
  - 推理依据：$a^0 = 1$ 对任意 $a > 0$ 且 $a \neq 1$ 恒成立

**公式**：
- $y = a^x > 0$，$\forall x \in \mathbb{R}$
- $a^0 = 1$

**二级结论**：
- 函数 $y = a^x$ 与 $y = (\frac{1}{a})^x$ 的图像关于 $y$ 轴对称
  - 推理过程：$y = (\frac{1}{a})^x = a^{-x}$，点 $(x, a^x)$ 变为 $(-x, a^x) = (-x, a^{-(-x)})$ 恰好在后者上
- 过点 $(0, 1)$ 作 $y$ 轴垂线右侧，底数越大的指数函数图像越靠上；左侧则相反

---

#### 知识点 2.1.4：指数函数的性质

**重要性级别**：A（核心）

**定义**：
指数函数 $y = a^x$（$a > 0$ 且 $a \neq 1$）的性质总结如下：

| 性质 | $a > 1$ | $0 < a < 1$ |
|------|---------|-------------|
| 定义域 | $\mathbb{R}$ | $\mathbb{R}$ |
| 值域 | $(0, +\infty)$ | $(0, +\infty)$ |
| 过定点 | $(0, 1)$ | $(0, 1)$ |
| 单调性 | 在 $\mathbb{R}$ 上单调递增 | 在 $\mathbb{R}$ 上单调递减 |
| 渐近线 | $y = 0$（$x$ 轴） | $y = 0$（$x$ 轴） |
| $x > 0$ 时 | $y > 1$ | $0 < y < 1$ |
| $x < 0$ 时 | $0 < y < 1$ | $y > 1$ |

**特性**：
- 值域 $y > 0$ 恒成立，即指数函数的值永远是正数
- 指数函数既不是奇函数也不是偶函数（除非退化的特殊情况）
- 在第一象限内，底数越大的指数函数图像越靠上方
- 指数函数是**一一映射**（单调），因此存在反函数（即对数函数）

**运算方式**：
- 利用单调性比较指数幂的大小
- 利用单调性解指数不等式
- 利用值域确定函数取值范围

**定理**：
- **指数函数单调性定理**：指数函数 $y = a^x$ 在 $\mathbb{R}$ 上：
  - 当 $a > 1$ 时单调递增
  - 当 $0 < a < 1$ 时单调递减
  - 前提条件：$a > 0$ 且 $a \neq 1$
  - 使用场合：比较指数幂大小、解指数不等式

**公式**：
- $a > 1$ 时：$a^{x_1} > a^{x_2} \Leftrightarrow x_1 > x_2$
- $0 < a < 1$ 时：$a^{x_1} > a^{x_2} \Leftrightarrow x_1 < x_2$

**二级结论**：
- 指数函数 $y = a^x$ 的值恒为正，因此方程 $a^x = 0$ 无解，$a^x = b$ 当 $b \leq 0$ 时无解
  - 推理过程：$a^x > 0$ 对所有实数 $x$ 成立，故不可能等于非正数
- 若 $a > 1$，则 $\lim_{x \to +\infty} a^x = +\infty$，$\lim_{x \to -\infty} a^x = 0$
  - 这说明 $a > 1$ 时函数增长无界，衰减趋于零

---

#### 知识点 2.1.5：指数函数的应用

**重要性级别**：B（重要）

**定义**：
指数函数的应用主要包括以下几个方面：

1. **比较大小**：
   - 同底数：利用单调性，$a > 1$ 时指数大的幂大，$0 < a < 1$ 时指数大的幂小
   - 同指数：利用幂函数性质，比较底数
   - 底数和指数都不同：找中间量（常用 $1$ 或 $0$）进行比较

2. **解指数方程**：化为同底数幂，利用 $a^{f(x)} = a^{g(x)} \Leftrightarrow f(x) = g(x)$（$a > 0$ 且 $a \neq 1$）

3. **解指数不等式**：
   - $a > 1$ 时：$a^{f(x)} > a^{g(x)} \Leftrightarrow f(x) > g(x)$
   - $0 < a < 1$ 时：$a^{f(x)} > a^{g(x)} \Leftrightarrow f(x) < g(x)$（注意不等号方向改变）

4. **实际应用模型**：
   - 人口增长模型：$N(t) = N_0 \cdot a^t$
   - 放射性衰变：$N(t) = N_0 \cdot (\frac{1}{2})^{\frac{t}{T}}$
   - 复利计算：$A = P(1 + r)^t$

**特性**：
- 解指数不等式时，$a > 1$ 则不等号方向**不变**，$0 < a < 1$ 则不等号方向**改变**
- 指数模型适用于描述＂增长越来越快＂或＂衰减越来越慢＂的现象

**运算方式**：
- 比较大小：化为同底或找中间量
- 解方程：化为同底，去掉底数
- 解不等式：根据底数大小决定不等号方向是否改变

**定理**：
- **指数方程求解定理**：若 $a^{f(x)} = a^{g(x)}$（$a > 0$ 且 $a \neq 1$），则 $f(x) = g(x)$
  - 前提条件：$a > 0$ 且 $a \neq 1$
  - 使用场合：解指数方程
  - 注意：必须化为同底数才能去掉底数

**公式**：
- 同底数比较：$a > 1$ 时，$a^m > a^n \Leftrightarrow m > n$
- 同底数比较：$0 < a < 1$ 时，$a^m > a^n \Leftrightarrow m < n$

**二级结论**：
- 比较 $a^m$ 与 $a^n$ 时，先看底数：$a > 1$ 则随指数增大而增大，$0 < a < 1$ 则随指数增大而减小
- 对于 $a^x > b$（$a > 1$，$b > 0$），解为 $x > \log_a b$；若 $0 < a < 1$，则解为 $x < \log_a b$
  - 推理过程：两边取对数，$a > 1$ 时对数函数递增，不等号方向不变；$0 < a < 1$ 时对数函数递减，不等号方向改变

### 2.2 对数函数

对数是指数的逆运算，对数函数与指数函数互为反函数。对数在简化计算、测量标度（如分贝、地震震级）和科学研究中有着广泛应用。

---

#### 知识点 2.2.1：对数的概念

**重要性级别**：A（核心）

**定义**：
如果 $a^x = N$（$a > 0$ 且 $a \neq 1$），则称 $x$ 为以 $a$ 为底 $N$ 的**对数**，记作：
$$x = \log_a N$$

其中，$a$ 叫做**底数**，$N$ 叫做**真数**。

**常用对数**：
- 常用对数（以10为底）：$\lg N = \log_{10} N$
- 自然对数（以 $e$ 为底，$e \approx 2.71828$）：$\ln N = \log_e N$

**特性**：
- 真数 $N > 0$，即**负数与零没有对数**
- 对数与指数的互化：$a^x = N \Leftrightarrow x = \log_a N$（$a > 0$ 且 $a \neq 1$，$N > 0$）
- 对数的基本性质：
  - $\log_a a = 1$（因为 $a^1 = a$）
  - $\log_a 1 = 0$（因为 $a^0 = 1$）
  - $a^{\log_a N} = N$（对数恒等式）

**运算方式**：
- 指数式与对数式的互化
- 利用对数恒等式简化表达式

**定理**：
- **对数存在定理**：对于任意 $a > 0$ 且 $a \neq 1$，以及任意 $N > 0$，存在唯一的实数 $x$ 使得 $a^x = N$，即 $\log_a N$ 是唯一确定的
  - 前提条件：$a > 0$ 且 $a \neq 1$，$N > 0$
  - 使用场合：保证对数运算的合法性

**公式**：
- 对数定义：$a^x = N \Leftrightarrow x = \log_a N$（$a > 0$ 且 $a \neq 1$，$N > 0$）
- 基本性质：
$$\log_a a = 1, \quad \log_a 1 = 0, \quad a^{\log_a N} = N$$

**二级结论**：
- $\log_a a^n = n$（$a > 0$ 且 $a \neq 1$）
  - 推理过程：设 $x = \log_a a^n$，则 $a^x = a^n$，由指数函数的单射性得 $x = n$
- $\log_a \frac{1}{N} = -\log_a N$
  - 推理过程：$\log_a \frac{1}{N} = \log_a N^{-1} = -\log_a N$（利用后续学习的幂的对数性质）

---

#### 知识点 2.2.2：对数的运算性质

**重要性级别**：A（核心）

**定义**：
对数的运算性质是将对数运算转化为真数运算的桥梁（$a > 0$ 且 $a \neq 1$，$M > 0$，$N > 0$）：

1. **积的对数**：$\log_a(M \cdot N) = \log_a M + \log_a N$
2. **商的对数**：$\log_a\frac{M}{N} = \log_a M - \log_a N$
3. **幂的对数**：$\log_a M^n = n \cdot \log_a M$（$n \in \mathbb{R}$）

**运算口诀**：
- 积化**和**，商化**差**，幂化**乘**

**特性**：
- 注意：$\log_a(M + N) \neq \log_a M + \log_a N$（对数的和不是和的对数）
- 注意：$\log_a(M - N) \neq \log_a M - \log_a N$（对数的差不是差的对数）
- 对数运算将乘除运算降为加减运算，将幂运算降为乘法运算，这是历史上对数用于简化复杂计算的原理

**运算方式**：
- 正用：将积、商、幂的对数展开为对数的和、差、积
- 逆用：将对数的和、差、积合并为真数的积、商、幂
- 化简对数表达式：利用运算性质将复杂表达式化简

**定理**：
- **对数运算性质定理**：设 $a > 0$ 且 $a \neq 1$，$M > 0$，$N > 0$，则：
  - $\log_a(MN) = \log_a M + \log_a N$
  - $\log_a\frac{M}{N} = \log_a M - \log_a N$
  - $\log_a M^n = n\log_a M$（$n \in \mathbb{R}$）
  - 前提条件：真数必须为正数，底数 $a > 0$ 且 $a \neq 1$
  - 使用场合：化简对数表达式，解对数方程

**公式**：
- $\log_a(MN) = \log_a M + \log_a N$
- $\log_a\frac{M}{N} = \log_a M - \log_a N$
- $\log_a M^n = n\log_a M$

**二级结论**：
- $\log_a \sqrt[n]{M} = \frac{1}{n}\log_a M$（$n \in \mathbb{N}^*$）
  - 推理过程：$\sqrt[n]{M} = M^{\frac{1}{n}}$，利用幂的对数性质得 $\log_a M^{\frac{1}{n}} = \frac{1}{n}\log_a M$
- $\log_a M = \frac{1}{\log_M a}$（倒数关系，由换底公式推出）

---

#### 知识点 2.2.3：换底公式

**重要性级别**：A（核心）

**定义**：
**换底公式**：
$$\log_a b = \frac{\log_c b}{\log_c a}$$

其中 $a > 0$ 且 $a \neq 1$，$c > 0$ 且 $c \neq 1$，$b > 0$。

最常用的换底方式是换成常用对数或自然对数：
$$\log_a b = \frac{\lg b}{\lg a} = \frac{\ln b}{\ln a}$$

**特性**：
- 换底公式的作用是将不同底数的对数转换为同底数对数，便于计算和比较
- 可以换成任意方便的底数（常用10或 $e$）

**运算方式**：
- 将不同底数的对数化为同底数
- 利用计算器计算对数值时，通常转换为常用对数或自然对数

**定理**：
- **换底公式**：$\log_a b = \frac{\log_c b}{\log_c a}$（$a, c > 0$ 且 $a, c \neq 1$，$b > 0$）
  - 前提条件：各底数和真数满足对数定义域要求
  - 使用场合：不同底数对数的转换、计算、化简
  - 证明：设 $x = \log_a b$，则 $a^x = b$，两边取以 $c$ 为底的对数得 $x \log_c a = \log_c b$，所以 $x = \frac{\log_c b}{\log_c a}$

**公式**：
- 换底公式：$\log_a b = \frac{\log_c b}{\log_c a}$
- 常用形式：$\log_a b = \frac{\lg b}{\lg a} = \frac{\ln b}{\ln a}$

**二级结论**：
- $\log_a b = \frac{1}{\log_b a}$（倒数关系）
  - 推理过程：在换底公式中取 $c = b$，得 $\log_a b = \frac{\log_b b}{\log_b a} = \frac{1}{\log_b a}$
- $\log_a b \cdot \log_b c = \log_a c$（链式法则）
  - 推理过程：$\log_a b \cdot \log_b c = \frac{\ln b}{\ln a} \cdot \frac{\ln c}{\ln b} = \frac{\ln c}{\ln a} = \log_a c$
- $\log_{a^n} b = \frac{1}{n}\log_a b$（$n \neq 0$）
  - 推理过程：$\log_{a^n} b = \frac{\log_a b}{\log_a a^n} = \frac{\log_a b}{n}$

---

#### 知识点 2.2.4：对数函数的概念

**重要性级别**：A（核心）

**定义**：
一般地，函数 $y = \log_a x$（$a > 0$ 且 $a \neq 1$）叫做**对数函数**，其中 $x$ 为自变量，定义域为 $(0, +\infty)$。

对数函数 $y = \log_a x$ 是指数函数 $y = a^x$ 的**反函数**。

**特性**：
- 定义域为 $x > 0$，因为真数必须大于零
- 值域为 $\mathbb{R}$
- 对数函数与指数函数互为反函数，它们的定义域和值域**互换**
  - 指数函数：定义域 $\mathbb{R}$，值域 $(0, +\infty)$
  - 对数函数：定义域 $(0, +\infty)$，值域 $\mathbb{R}$

**运算方式**：
- 求函数值：给定 $x > 0$，计算 $\log_a x$
- 已知 $y$ 求 $x$：由 $y = \log_a x$ 得 $x = a^y$

**定理**：
- **对数函数定义定理**：形如 $y = \log_a x$（$a > 0$ 且 $a \neq 1$）的函数称为对数函数，定义域为 $(0, +\infty)$
  - 前提条件：$a > 0$ 且 $a \neq 1$
  - 使用场合：判断给定函数是否为对数函数

**公式**：
- 对数函数的标准形式：$y = \log_a x$（$a > 0$ 且 $a \neq 1$）
- 定义域：$D = (0, +\infty)$

**二级结论**：
- 函数 $y = \log_a x$ 与 $y = a^x$ 互为反函数，满足：$a^{\log_a x} = x$（$x > 0$）和 $\log_a(a^x) = x$（$x \in \mathbb{R}$）
  - 推理过程：由反函数定义，$f(f^{-1}(x)) = x$ 且 $f^{-1}(f(x)) = x$
- 对数型函数 $y = \log_a f(x)$ 的定义域由 $f(x) > 0$ 确定

---

#### 知识点 2.2.5：对数函数的图像

**重要性级别**：A（核心）

**定义**：
对数函数 $y = \log_a x$（$a > 0$ 且 $a \neq 1$）的图像特征：

- 当 $a > 1$ 时：图像从左下向右上方单调上升
- 当 $0 < a < 1$ 时：图像从左上向右下方单调下降

**图像共同特征**：
- 所有对数函数图像都在 $y$ 轴**右侧**（即 $x > 0$）
- 所有对数函数图像都过定点 $(1, 0)$，因为 $\log_a 1 = 0$
- $y$ 轴（即直线 $x = 0$）是图像的垂直渐近线
- 图像不与 $y$ 轴相交

**特性**：
- $a > 1$ 时：$x > 1$ 则 $y > 0$；$0 < x < 1$ 则 $y < 0$
- $0 < a < 1$ 时：$x > 1$ 则 $y < 0$；$0 < x < 1$ 则 $y > 0$
- 对数函数图像分布在第一、四象限
- 在第一象限内，$x > 1$ 时底数越大的对数函数图像越靠下方

**运算方式**：
- 利用图像判断对数函数的单调性和符号
- 利用图像比较不同对数函数的大小关系

**定理**：
- **对数函数图像定点定理**：对数函数 $y = \log_a x$（$a > 0$ 且 $a \neq 1$）的图像必过定点 $(1, 0)$
  - 前提条件：$a > 0$ 且 $a \neq 1$
  - 使用场合：快速确定对数函数图像的位置
  - 推理依据：$\log_a 1 = 0$ 对任意 $a > 0$ 且 $a \neq 1$ 恒成立

**公式**：
- $y = \log_a x$，$x > 0$
- $\log_a 1 = 0$

**二级结论**：
- 函数 $y = \log_a x$ 与 $y = \log_{\frac{1}{a}} x$ 的图像关于 $x$ 轴对称
  - 推理过程：$y = \log_{\frac{1}{a}} x = \frac{\ln x}{\ln \frac{1}{a}} = \frac{\ln x}{-\ln a} = -\log_a x$，所以两者关于 $x$ 轴对称
- 过点 $(1, 0)$ 作 $x$ 轴垂线右侧，底数越小的对数函数（$a > 1$ 时）图像越靠上

---

#### 知识点 2.2.6：对数函数的性质

**重要性级别**：A（核心）

**定义**：
对数函数 $y = \log_a x$（$a > 0$ 且 $a \neq 1$）的性质总结如下：

| 性质 | $a > 1$ | $0 < a < 1$ |
|------|---------|-------------|
| 定义域 | $(0, +\infty)$ | $(0, +\infty)$ |
| 值域 | $\mathbb{R}$ | $\mathbb{R}$ |
| 过定点 | $(1, 0)$ | $(1, 0)$ |
| 单调性 | 在 $(0, +\infty)$ 上单调递增 | 在 $(0, +\infty)$ 上单调递减 |
| 渐近线 | $x = 0$（$y$ 轴） | $x = 0$（$y$ 轴） |
| $x > 1$ 时 | $y > 0$ | $y < 0$ |
| $0 < x < 1$ 时 | $y < 0$ | $y > 0$ |

**特性**：
- 对数函数既不是奇函数也不是偶函数
- 在第一象限内，$x > 1$ 时底数越大的对数函数图像越靠下方
- 对数函数是**一一映射**（单调），因此存在反函数（即指数函数）
- 对数函数增长缓慢，适合描述增长越来越慢的现象

**运算方式**：
- 利用单调性比较对数值的大小
- 利用单调性解对数不等式

**定理**：
- **对数函数单调性定理**：对数函数 $y = \log_a x$ 在 $(0, +\infty)$ 上：
  - 当 $a > 1$ 时单调递增
  - 当 $0 < a < 1$ 时单调递减
  - 前提条件：$a > 0$ 且 $a \neq 1$
  - 使用场合：比较对数值大小、解对数不等式

**公式**：
- $a > 1$ 时：$\log_a x_1 > \log_a x_2 \Leftrightarrow x_1 > x_2 > 0$
- $0 < a < 1$ 时：$\log_a x_1 > \log_a x_2 \Leftrightarrow 0 < x_1 < x_2$

**二级结论**：
- 方程 $\log_a x = 0$ 的唯一解为 $x = 1$
  - 推理过程：$\log_a x = 0 \Leftrightarrow x = a^0 = 1$
- 当 $a > 1$ 时，$\lim_{x \to +\infty} \log_a x = +\infty$，$\lim_{x \to 0^+} \log_a x = -\infty$
  - 这说明对数函数可以取到任意实数值，值域为全体实数

---

#### 知识点 2.2.7：指数与对数函数的关系

**重要性级别**：A（核心）

**定义**：
指数函数 $y = a^x$ 与对数函数 $y = \log_a x$（$a > 0$ 且 $a \neq 1$）互为**反函数**。

互为反函数的两个函数具有以下关系：
- 它们的图像关于直线 $y = x$ 对称
- 它们的定义域和值域互换
- $f(f^{-1}(x)) = x$ 且 $f^{-1}(f(x)) = x$

**特性**：
- 反函数的存在条件：原函数必须是**单射**（一一映射）
- 指数函数 $y = a^x$ 是单射（单调），因此存在反函数
- 指数函数的值域 $(0, +\infty)$ 成为对数函数的定义域
- 对数函数的值域 $\mathbb{R}$ 成为指数函数的定义域

**运算方式**：
- 利用反函数关系求指数或对数表达式的值
- 利用图像对称性理解两个函数的关系
- 由指数函数性质推断对数函数性质，反之亦然

**定理**：
- **反函数图像对称定理**：若函数 $y = f(x)$ 与其反函数 $y = f^{-1}(x)$ 都存在，则它们的图像关于直线 $y = x$ 对称
  - 前提条件：$f$ 是单射（一一映射）
  - 使用场合：利用一个函数的图像画出其反函数的图像

**公式**：
- 反函数关系：
$$a^{\log_a x} = x \quad (x > 0), \quad \log_a(a^x) = x \quad (x \in \mathbb{R})$$

**二级结论**：
- 点 $(p, q)$ 在 $y = a^x$ 上 $\Leftrightarrow$ 点 $(q, p)$ 在 $y = \log_a x$ 上
  - 推理过程：$(p, q)$ 在 $y = a^x$ 上意味着 $q = a^p$，即 $p = \log_a q$，所以 $(q, p)$ 在 $y = \log_a x$ 上
- 若 $f(x) = a^x$，则 $f^{-1}(x) = \log_a x$，且 $f(f^{-1}(x)) = x$，$f^{-1}(f(x)) = x$

---

### 2.3 幂函数

幂函数是基本初等函数中形式最简单的一类，其底数为自变量、指数为常数。幂函数与指数函数的区别在于变量所在的位置不同。

---

#### 知识点 2.3.1：幂函数的概念

**重要性级别**：A（核心）

**定义**：
一般地，函数 $y = x^a$ 叫做**幂函数**，其中 $x$ 为自变量，$a$ 为常数。

常见的幂函数有：
- $y = x$（一次函数，$a = 1$）
- $y = x^2$（二次函数，$a = 2$）
- $y = x^3$（三次函数，$a = 3$）
- $y = x^{\frac{1}{2}} = \sqrt{x}$（平方根函数，$a = \frac{1}{2}$）
- $y = x^{-1} = \frac{1}{x}$（反比例函数，$a = -1$）

**特性**：
- 幂函数与指数函数的区别：
  - 幂函数：底数为**变量**，指数为**常数**，即 $y = x^a$
  - 指数函数：底数为**常数**，指数为**变量**，即 $y = a^x$
- 幂函数的定义域视指数 $a$ 的取值而定
  - $a$ 为正整数时，定义域为 $\mathbb{R}$
  - $a$ 为负整数时，定义域为 $(-\infty, 0) \cup (0, +\infty)$
  - $a = \frac{1}{2}$ 时，定义域为 $[0, +\infty)$
  - $a = -\frac{1}{2}$ 时，定义域为 $(0, +\infty)$

**运算方式**：
- 根据指数 $a$ 确定定义域
- 分析不同指数下幂函数的性质

**定理**：
- **幂函数定义域定理**：幂函数 $y = x^a$ 的定义域由指数 $a$ 决定：
  - 若 $a \in \mathbb{N}^*$，定义域为 $\mathbb{R}$
  - 若 $a \in \mathbb{Z}^-$，定义域为 $(-\infty, 0) \cup (0, +\infty)$
  - 若 $a = \frac{p}{q}$（既约分数），$q$ 为偶数时定义域为 $[0, +\infty)$（$p > 0$）或 $(0, +\infty)$（$p < 0$）
  - 前提条件：指数为有理数
  - 使用场合：确定幂函数的定义域

**公式**：
- 幂函数标准形式：$y = x^a$（$a$ 为常数）

**二级结论**：
- 幂函数 $y = x^a$ 当 $a > 0$ 时必过原点 $(0, 0)$；当 $a < 0$ 时不过原点
  - 推理过程：$a > 0$ 时 $0^a = 0$；$a < 0$ 时 $0^a$ 无意义（分母为零）
- 所有幂函数都过定点 $(1, 1)$，因为 $1^a = 1$ 对任意 $a$ 成立

---

#### 知识点 2.3.2：幂函数的图像与性质

**重要性级别**：A（核心）

**定义**：
幂函数 $y = x^a$ 的图像与性质根据指数 $a$ 的不同而有所区别：

**共性**：
- 所有幂函数图像都过定点 $(1, 1)$
- $a > 0$ 时图像过原点 $(0, 0)$；$a < 0$ 时图像不过原点

**$a > 0$ 时的性质**：
- 图像过原点 $(0, 0)$
- 在 $(0, +\infty)$ 上单调递增
- 图像从原点出发向上延伸

**$a < 0$ 时的性质**：
- 图像不过原点
- 在 $(0, +\infty)$ 上单调递减
- $x$ 轴和 $y$ 轴是渐近线

**特殊幂函数的奇偶性**：
- 奇函数：$y = x$，$y = x^3$，$y = x^{-1}$（指数为奇数）
- 偶函数：$y = x^2$（指数为偶数）
- 非奇非偶：$y = x^{\frac{1}{2}}$（定义域 $[0, +\infty)$ 不对称）

**特性**：
- $a > 0$ 时在第一象限单调递增
- $a < 0$ 时在第一象限单调递减
- 在第一象限内，$x > 1$ 时指数越大的幂函数图像越靠上方；$0 < x < 1$ 时则相反

**运算方式**：
- 利用幂函数的单调性比较幂的大小
- 结合奇偶性简化分析和计算

**定理**：
- **幂函数单调性定理**：幂函数 $y = x^a$ 在 $(0, +\infty)$ 上：
  - 当 $a > 0$ 时单调递增
  - 当 $a < 0$ 时单调递减
  - 前提条件：$x > 0$
  - 使用场合：比较正数幂的大小

**公式**：
- $a > 0$ 时：$x_1 > x_2 > 0 \Rightarrow x_1^a > x_2^a$
- $a < 0$ 时：$x_1 > x_2 > 0 \Rightarrow x_1^a < x_2^a$

**二级结论**：
- 在第一象限，幂函数 $y = x^a$ 的图像：$a > 1$ 时上凸，$0 < a < 1$ 时下凸，$a < 0$ 时递减趋近于坐标轴
- 若 $a$ 为偶数，则 $y = x^a$ 是偶函数；若 $a$ 为奇数，则 $y = x^a$ 是奇函数（在定义域关于原点对称的前提下）

---

#### 知识点 2.3.3：幂函数的应用

**重要性级别**：B（重要）

**定义**：
幂函数的应用包括比较幂的大小、建立实际问题的函数模型等。幂函数与指数函数、对数函数共同构成了高中阶段的基本初等函数体系。

**三类函数的增长速度比较**：
当 $x \to +\infty$ 时：
$$\log_a x \ll x^k \ll a^x \quad (a > 1, k > 0)$$

即对数增长远小于幂函数增长，幂函数增长远小于指数增长。

**特性**：
- 幂函数 $y = x^a$ 的增长速度介于对数函数和指数函数之间
- 当 $x \to +\infty$ 时，无论 $k$ 多大，$x^k$ 的增长速度都不及 $a^x$（$a > 1$）
- 幂函数适用于描述幂律关系，如万有引力定律 $F \propto r^{-2}$、面积与边长关系 $S \propto l^2$ 等

**运算方式**：
- 比较幂的大小：化为同指数或同底数进行比较
- 建立模型：根据实际问题的增长特征选择合适的函数类型

**定理**：
- **增长速度比较定理**：当 $x \to +\infty$ 时，$\log_a x \ll x^k \ll a^x$（$a > 1, k > 0$）
  - 前提条件：$x \to +\infty$，$a > 1$，$k > 0$
  - 使用场合：选择函数模型，判断长期增长趋势
  - 意义：指数增长最终会超过任何幂函数增长，对数增长最慢

**公式**：
- 增长速度排序（$x \to +\infty$）：$\log_a x < x^k < a^x$（$a > 1, k > 0$）

**二级结论**：
- 当 $x$ 足够大时，$a^x$（$a > 1$）的值会超过 $x$ 的任何固定次幂
  - 推理过程：这是指数函数与幂函数增长率的本质区别，指数函数的指数位置使增长速度呈爆炸式
- 实际问题中选择模型：匀速增长选线性模型，增长越来越快选指数模型，增长越来越慢趋于饱和选对数模型，按比例缩放选幂函数模型

### 3.1 函数与方程

函数与方程有着密切的联系。函数的零点就是对应方程的实数根，这种联系为我们提供了用函数方法解决方程问题的新途径。

---

#### 知识点 3.1.1：函数的零点

**重要性级别**：A（核心）

**定义**：
对于函数 $y = f(x)$，使 $f(x) = 0$ 的实数 $x$ 叫做函数的**零点**。

**重要强调**：
- 函数的零点不是一个＂点＂（坐标），而是一个**实数**
- 函数的零点是函数图像与 $x$ 轴交点的**横坐标**
- 例如：$f(x) = x^2 - 1$ 的零点为 $x = 1$ 和 $x = -1$，图像与 $x$ 轴交于点 $(1, 0)$ 和 $(-1, 0)$

**特性**：
- 代数意义：$x_0$ 是零点 $\Leftrightarrow f(x_0) = 0$
- 几何意义：$(x_0, 0)$ 是函数图像与 $x$ 轴的交点
- 方程 $f(x) = 0$ 有实数根 $\Leftrightarrow$ 函数图像与 $x$ 轴有交点 $\Leftrightarrow$ 函数有零点
- 这三者是**等价**的，体现了数形结合的思想

**运算方式**：
- 求零点：解方程 $f(x) = 0$
- 判断零点个数：利用图像、判别式或零点存在性定理

**定理**：
- **零点定义定理**：$x_0$ 是函数 $f(x)$ 的零点 $\Leftrightarrow f(x_0) = 0$
  - 前提条件：$x_0$ 在定义域内
  - 使用场合：将函数零点问题转化为方程求解问题

**公式**：
- 零点满足：$f(x_0) = 0$
- 方程 $f(x) = 0$ 的实数根 $=$ 函数 $f(x)$ 的零点

**二级结论**：
- 函数 $f(x)$ 的零点个数等于方程 $f(x) = 0$ 的实数根的个数
  - 推理过程：由零点的定义直接得出，二者是同一问题的不同表述
- 若 $f(x) = g(x) - h(x)$，则 $f(x)$ 的零点就是方程 $g(x) = h(x)$ 的解，也就是函数 $y = g(x)$ 与 $y = h(x)$ 图像交点的横坐标
  - 推理过程：$f(x) = 0 \Leftrightarrow g(x) = h(x)$，交点处两函数值相等

---

#### 知识点 3.1.2：零点与方程根的关系

**重要性级别**：A（核心）

**定义**：
方程 $f(x) = 0$ 的实数根就是函数 $y = f(x)$ 的零点。求函数零点就是求方程 $f(x) = 0$ 的实数解。

这是方程问题与函数问题相互转化的桥梁，体现了函数与方程的思想。

**各类函数的零点情况**：
- 一次函数 $f(x) = ax + b$（$a \neq 0$）的零点为 $x = -\frac{b}{a}$
- 二次函数 $f(x) = ax^2 + bx + c$（$a \neq 0$）的零点用判别式判断后用求根公式求解
- 指数函数 $y = a^x$（$a > 0$ 且 $a \neq 1$）无零点（因为 $a^x > 0$）
- 对数函数 $y = \log_a x$（$a > 0$ 且 $a \neq 1$）的零点为 $x = 1$

**特性**：
- 方程的根、函数的零点、图像与 $x$ 轴的交点横坐标，三者等价
- 利用这种等价关系，可以用函数方法研究方程，也可以用方程方法研究函数

**运算方式**：
- 求函数零点：解方程 $f(x) = 0$
- 判断方程根的情况：利用函数图像或性质

**定理**：
- **零点等价定理**：以下三个命题等价：
  1. 方程 $f(x) = 0$ 有实数根 $x_0$
  2. 函数 $y = f(x)$ 有零点 $x_0$
  3. 函数图像与 $x$ 轴有交点 $(x_0, 0)$
  - 前提条件：$f(x)$ 是实函数，$x_0$ 在定义域内
  - 使用场合：方程与函数问题的相互转化

**公式**：
- 一次函数零点：$x = -\frac{b}{a}$（$a \neq 0$）
- 二次函数零点：$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$（$\Delta \geq 0$）

**二级结论**：
- 函数 $y = f(x) - g(x)$ 的零点个数等于方程 $f(x) = g(x)$ 的实数解个数，也等于函数 $y = f(x)$ 与 $y = g(x)$ 图像交点的个数
  - 推理过程：$f(x) - g(x) = 0 \Leftrightarrow f(x) = g(x)$，即两函数图像交点处函数值相等
- 若函数 $f(x)$ 可以因式分解为 $f(x) = (x - x_1)(x - x_2)\cdots(x - x_n)g(x)$，则 $x_1, x_2, \dots, x_n$ 都是函数的零点

---

#### 知识点 3.1.3：零点存在性定理

**重要性级别**：A（核心）

**定义**：
**零点存在性定理**：如果函数 $y = f(x)$ 在闭区间 $[a, b]$ 上的图像是**连续不断**的一条曲线，且 $f(a) \cdot f(b) < 0$（即 $f(a)$ 与 $f(b)$ **异号**），则函数 $y = f(x)$ 在区间 $(a, b)$ 内**至少有一个**零点。

即存在 $c \in (a, b)$，使得 $f(c) = 0$。

**特性**：
- **前提条件**：
  1. 函数在 $[a, b]$ 上**连续**
  2. 端点函数值**异号**：$f(a) \cdot f(b) < 0$
- **结论**：在开区间 $(a, b)$ 内**至少存在一个**零点
- $f(a) \cdot f(b) < 0$ 是**充分条件**而非必要条件
  - 函数有零点不一定要求端点异号（如 $f(x) = x^2$ 在 $[-1, 1]$ 上有零点 $x = 0$，但 $f(-1) \cdot f(1) = 1 > 0$）
- 定理只能判断零点**存在**，不能判断零点**个数**

**运算方式**：
- 判断函数在某区间内是否有零点：验证连续性和端点符号
- 寻找零点所在区间：计算若干点的函数值，找异号区间

**定理**：
- **零点存在性定理**（介值定理的推论）：设 $f(x)$ 在 $[a, b]$ 上连续，且 $f(a) \cdot f(b) < 0$，则存在 $c \in (a, b)$ 使 $f(c) = 0$
  - 前提条件：连续性 + 端点异号
  - 使用场合：判断零点存在、确定零点所在大致区间
  - 注意：只能保证至少一个零点，可能有多个

**公式**：
- 零点存在条件：$f(x)$ 在 $[a, b]$ 连续且 $f(a) \cdot f(b) < 0$ $\Rightarrow$ $\exists c \in (a, b), f(c) = 0$

**二级结论**：
- 若 $f(x)$ 在 $[a, b]$ 上连续，且 $f(a) \cdot f(b) > 0$，则不能确定是否有零点（可能有，也可能没有）
  - 推理过程：$f(a)$ 与 $f(b)$ 同号时，函数可能始终为正或始终为负（无零点），也可能先正后负再正（有零点）
- 若 $f(x)$ 在 $[a, b]$ 上连续，$f(a) \cdot f(b) < 0$，且 $f(x)$ 单调，则零点**唯一**
  - 推理过程：零点存在性定理保证至少有一个，单调性保证至多有一个

---

#### 知识点 3.1.4：唯一零点判定

**重要性级别**：A（核心）

**定义**：
若函数 $y = f(x)$ 在 $[a, b]$ 上**连续且单调**，并且 $f(a) \cdot f(b) < 0$，则 $f(x)$ 在 $(a, b)$ 内有且**仅有一个**零点。

**判定唯一零点的条件**：
1. **连续性**：保证图像不断开，可以用零点存在性定理
2. **单调性**：保证图像在区间内只上升或只下降，不会回头，因此最多与 $x$ 轴相交一次
3. **端点异号**：保证至少与 $x$ 轴相交一次

三者结合得出**恰有一个**零点的结论。

**特性**：
- 零点存在性定理保证**至少**有一个零点
- 单调性保证**至多**有一个零点
- 两者结合得出**恰有**一个零点的结论
- 这是判断零点唯一性的常用方法

**运算方式**：
- 判断连续性：初等函数在其定义域内都是连续的
- 判断单调性：利用导数或定义法
- 验证端点异号：计算 $f(a)$ 和 $f(b)$

**定理**：
- **唯一零点定理**：若 $f(x)$ 在 $[a, b]$ 上连续且严格单调，且 $f(a) \cdot f(b) < 0$，则 $f(x)$ 在 $(a, b)$ 内有且仅有一个零点
  - 前提条件：连续性 + 严格单调性 + 端点异号
  - 使用场合：证明零点唯一性，二分法求近似值的理论基础

**公式**：
- 唯一零点条件：$f(x)$ 在 $[a, b]$ 连续单调 + $f(a) \cdot f(b) < 0$ $\Rightarrow$ 恰有一个零点

**二级结论**：
- 若 $f(x)$ 在 $[a, b]$ 上连续，$f(a) \cdot f(b) < 0$，且在 $(a, b)$ 内 $f'(x) > 0$（或 $f'(x) < 0$），则 $f(x)$ 在 $(a, b)$ 内有唯一零点
  - 推理过程：导数恒正（或恒负）说明函数严格单调，结合零点存在性定理即得唯一零点
- 对于二次函数，$\Delta > 0$ 时有两个零点，$\Delta = 0$ 时有一个零点（重根），$\Delta < 0$ 时无实零点

---

#### 知识点 3.1.5：二分法

**重要性级别**：A（核心）

**定义**：
**二分法**是基于零点存在性定理，通过不断将包含零点的区间对半分割，逐步缩小零点所在范围，从而得到零点近似值的方法。

**二分法的步骤**：
1. **确定区间**：确定初始区间 $[a, b]$，验证 $f(a) \cdot f(b) < 0$
2. **计算中点**：计算区间的中点 $c = \frac{a + b}{2}$
3. **判断符号**：计算 $f(c)$
   - 若 $f(c) = 0$，则 $c$ 就是零点
   - 若 $f(a) \cdot f(c) < 0$，则零点在 $(a, c)$ 内，令 $b = c$
   - 若 $f(c) \cdot f(b) < 0$，则零点在 $(c, b)$ 内，令 $a = c$
4. **重复迭代**：重复步骤 2 和 3，直到区间长度 $|a - b|$ 小于预设精度 $\varepsilon$
5. **输出结果**：取区间中点作为零点的近似值

**特性**：
- 每次迭代区间长度**减半**
- 初始区间长度为 $L$，经过 $n$ 次迭代后区间长度为 $\frac{L}{2^n}$
- 二分法只能求**近似值**，不能求精确值（除非恰好中点就是零点）
- 二分法要求函数在区间内**连续**且**端点异号**

**运算方式**：
- 确定初始区间：通过估算或图像观察
- 迭代计算：反复取中点、判断符号、缩小区间
- 精度控制：当区间长度小于给定精度时停止

**定理**：
- **二分法收敛定理**：设 $f(x)$ 在 $[a, b]$ 上连续且 $f(a) \cdot f(b) < 0$，则二分法产生的区间序列收敛于 $f(x)$ 的一个零点
  - 前提条件：连续性 + 端点异号
  - 使用场合：求函数零点的近似值
  - 收敛速度：每次迭代精度提高约一位二进制位

**公式**：
- 中点公式：$c = \frac{a + b}{2}$
- $n$ 次迭代后的区间长度：$\frac{b - a}{2^n}$
- 达到精度 $\varepsilon$ 所需最少迭代次数：$n > \log_2\frac{b - a}{\varepsilon}$

**二级结论**：
- 二分法的收敛速度是**线性收敛**，每次迭代区间长度减半，即精度提高约 $0.3$ 位十进制数（因为 $\log_{10} 2 \approx 0.301$）
  - 推理过程：$\frac{L}{2^n} < \varepsilon \Rightarrow n > \log_2\frac{L}{\varepsilon} = \frac{\log_{10}\frac{L}{\varepsilon}}{\log_{10} 2} \approx 3.32 \log_{10}\frac{L}{\varepsilon}$，即约每 $3.32$ 次迭代精度提高一位十进制数
- 二分法适用于连续函数的零点近似计算，但无法求重根（此时 $f(a) \cdot f(b) < 0$ 仍可能成立但收敛变慢）

### 3.2 函数模型及其应用

函数模型是连接数学与现实世界的桥梁。本节将学习不同类型的增长模型、增长速度的比较、如何建立函数模型以及如何利用数据进行拟合。

---

#### 知识点 3.2.1：不同增长的函数模型

**重要性级别**：A（核心）

**定义**：
三类基本增长模型：

1. **线性增长模型**（一次函数）：$y = kx + b$（$k > 0$）
   - 特征：**匀速增长**，增长速度恒定
   - 图像：一条直线
   - 适用：均匀变化的现象，如匀速运动的路程

2. **指数增长模型**（指数函数）：$y = a^x$（$a > 1$）
   - 特征：**爆炸式增长**，增长速度越来越快
   - 图像：下平上陡的曲线
   - 适用：快速增长的现象，如人口增长、复利计算、细菌繁殖

3. **对数增长模型**（对数函数）：$y = \log_a x$（$a > 1$）
   - 特征：增长越来越慢，趋于平缓
   - 图像：上陡下平的曲线
   - 适用：趋于饱和的现象，如学习曲线、信息传播后期

**特性**：
- 线性增长图像为**直线**
- 指数增长图像**下平上陡**
- 对数增长图像**上陡下平**
- 当 $x \to +\infty$ 时，增长速度：对数 $\ll$ 线性 $\ll$ 指数（在一定范围内）

**运算方式**：
- 根据实际问题的增长特征选择合适的函数模型
- 利用模型进行预测和计算

**定理**：
- **增长特征判定定理**：
  - 若增长量（增量）恒定，选线性模型
  - 若增长率（增速与当前量的比值）恒定，选指数模型
  - 若增长越来越慢趋于饱和，选对数模型
  - 使用场合：实际问题中选择合适的数学模型

**公式**：
- 线性模型：$y = kx + b$（$k > 0$）
- 指数模型：$y = a \cdot b^x$（$b > 1$）或 $y = a \cdot e^{kx}$（$k > 0$）
- 对数模型：$y = a \cdot \ln x + b$（$x > 0$）

**二级结论**：
- 指数增长模型中，当自变量增加1时，函数值乘以 $b$（增长因子）
  - 推理过程：$y(x+1) = a \cdot b^{x+1} = a \cdot b^x \cdot b = y(x) \cdot b$，即增量与当前值成比例
- 对数模型适用于描述＂报酬递减＂现象，即投入增加带来的产出增加越来越少

---

#### 知识点 3.2.2：增长速度排序

**重要性级别**：A（核心）

**定义**：
**增长速度排序定理**：当 $x \to +\infty$ 时，
$$\log_a x \ll x^k \ll a^x \ll x! \ll x^x \quad (a > 1, k > 0)$$

即：对数增长 $\ll$ 幂函数增长 $\ll$ 指数增长

其中 "$\ll$＂表示＂远小于＂，意味着前者的增长速度与后者相比可以忽略不计。

**极限表示**：
- $\lim_{x \to +\infty} \frac{\log_a x}{x^k} = 0$（对数增长远慢于幂函数增长）
- $\lim_{x \to +\infty} \frac{x^k}{a^x} = 0$（幂函数增长远慢于指数增长）

**特性**：
- 指数函数的增长速度会超过任何多项式函数（幂函数）
- 无论 $k$ 多大，$x^k$ 的增长速度最终都不及 $a^x$（$a > 1$）
- 对数函数的增长速度比任何正幂次的幂函数都慢

**运算方式**：
- 利用增长速度排序判断长期趋势
- 比较大函数在 $x \to +\infty$ 时的相对大小

**定理**：
- **增长速度极限定理**：
  - $\lim_{x \to +\infty} \frac{\log_a x}{x^k} = 0$（$a > 1, k > 0$）
  - $\lim_{x \to +\infty} \frac{x^k}{a^x} = 0$（$a > 1, k > 0$）
  - 前提条件：$x \to +\infty$
  - 使用场合：判断函数长期增长趋势，选择适合的模型

**公式**：
- 增长速度排序（$x \to +\infty$）：$\log_a x < x^k < a^x$（$a > 1, k > 0$）

**二级结论**：
- 在实际应用中，若短期数据三种模型都能拟合，但长期预测差异巨大，需根据问题本质选择正确模型
  - 推理过程：三种模型在局部可能接近，但渐近行为截然不同，线性无限延伸、指数爆炸增长、对数趋于平缓
- 对于足够大的 $x$，指数函数 $a^x$（$a > 1$）的值大于 $x$ 的任何固定次幂
  - 这是指数增长超越多项式增长的核心结论

---

#### 知识点 3.2.3：函数模型的建立

**重要性级别**：A（核心）

**定义**：
数学建模的一般流程：

```
实际问题 → 分析量与关系 → 建立函数模型 → 求解数学问题 → 检验与修正 → 实际解答
```

**常用函数模型**：
| 模型类型 | 函数形式 | 适用场景 |
|---------|---------|---------|
| 线性模型 | $y = kx + b$ | 匀速增长/衰减 |
| 二次模型 | $y = ax^2 + bx + c$ | 先增后减或有最值的问题 |
| 指数模型 | $y = a \cdot b^x$ | 增长越来越快 |
| 对数模型 | $y = a \cdot \ln x + b$ | 增长越来越慢趋于饱和 |
| 反比例模型 | $y = \frac{k}{x}$ | 成反比关系 |
| 幂函数模型 | $y = a \cdot x^b$ | 幂律关系 |

**特性**：
- 实际问题中定义域有实际意义限制（如时间不能为负、人数必须为正整数等）
- 建立模型后需要检验，看是否符合实际数据或常识
- 模型可能需要多次修正才能较好地描述实际问题

**运算方式**：
- 分析变量关系：确定自变量和因变量
- 选择模型类型：根据增长特征或散点图形状
- 确定参数：利用已知数据求参数
- 检验模型：用其他数据验证模型的合理性

**定理**：
- **模型选择指导定理**：
  - 数据呈直线分布 → 线性模型
  - 数据先增后减（或先减后增）→ 二次模型
  - 数据增长越来越快 → 指数模型
  - 数据增长越来越慢趋于极限 → 对数模型
  - 使用场合：根据数据特征选择合适的函数模型

**公式**：
- 线性模型：$y = kx + b$
- 二次模型：$y = ax^2 + bx + c$
- 指数模型：$y = a \cdot b^x$（$a > 0$，$b > 0$ 且 $b \neq 1$）
- 对数模型：$y = a \cdot \log_b x + c$（$x > 0$）

**二级结论**：
- 实际问题中求出的函数模型，其定义域必须结合实际意义确定，不能简单地取自然定义域
  - 例如人口模型中 $x \geq 0$（时间不能为负），且 $y$ 应为非负整数（人数）
- 建立模型后必须检验：将已知数据代入模型，看计算结果与实际数据是否接近

#### 知识点 3.2.4：已知函数模型求参数

**重要性级别**：A（核心）

**定义**：
已知函数模型求参数的核心方法是**待定系数法**：将已知数据代入函数模型得到关于参数的方程组，解方程组求出参数。

**各类模型的参数求解**：

1. **线性模型** $y = kx + b$：
   - 需要两组数据 $(x_1, y_1)$、$(x_2, y_2)$
   - 代入得：$\begin{cases} y_1 = kx_1 + b \\ y_2 = kx_2 + b \end{cases}$
   - 解二元一次方程组求 $k$、$b$

2. **二次模型** $y = ax^2 + bx + c$：
   - 需要三组数据
   - 代入得三元一次方程组，求解 $a$、$b$、$c$

3. **指数模型** $y = a \cdot b^x$：
   - 两边取对数：$\ln y = \ln a + x \ln b$
   - 令 $Y = \ln y$，$A = \ln a$，$B = \ln b$，化为线性模型 $Y = A + Bx$
   - 用线性回归方法求解

**特性**：
- 线性模型用两组数据代入解二元一次方程组
- 指数模型取对数后转化为线性问题
- 最后要代入检验确保模型正确
- 参数个数等于需要的已知数据组数

**运算方式**：
- 待定系数法：代入已知数据 → 列方程（组）→ 解方程（组）→ 得到参数
- 检验：将求得的参数代回，验证是否满足所有已知数据

**定理**：
- **待定系数法唯一性定理**：若函数模型有 $n$ 个待定参数，且已知 $n$ 组不同的数据点，在一般情况下可以唯一确定这 $n$ 个参数
  - 前提条件：数据点不导致方程组奇异（退化）
  - 使用场合：用已知数据确定函数模型的参数

**公式**：
- 线性模型：$\begin{cases} y_1 = kx_1 + b \\ y_2 = kx_2 + b \end{cases} \Rightarrow k = \frac{y_2 - y_1}{x_2 - x_1}$，$b = y_1 - kx_1$
- 指数模型取对数：$y = a \cdot b^x \Rightarrow \ln y = \ln a + x \ln b$

**二级结论**：
- 若数据点超过所需组数，方程组可能无精确解，此时需要用最小二乘法等拟合方法求最优近似参数
- 对于指数模型 $y = a \cdot b^x$，若取两组数据 $(x_1, y_1)$、$(x_2, y_2)$，则 $b = (\frac{y_2}{y_1})^{\frac{1}{x_2 - x_1}}$，$a = \frac{y_1}{b^{x_1}}$
  - 推理过程：由 $y_1 = a \cdot b^{x_1}$ 和 $y_2 = a \cdot b^{x_2}$，两式相除得 $\frac{y_2}{y_1} = b^{x_2 - x_1}$，所以 $b = (\frac{y_2}{y_1})^{\frac{1}{x_2 - x_1}}$

#### 知识点 3.2.5：数据拟合

**重要性级别**：B（重要）

**定义**：
**数据拟合**是当实际数据不完全符合某个函数模型时，寻找一个函数使其图像最接近给定数据分布趋势的方法。

**拟合函数选择策略**：
- 数据大致呈**直线**分布 → 选**线性模型** $y = kx + b$
- 数据**先增后减**有最值 → 选**二次模型** $y = ax^2 + bx + c$
- 数据增长越来越**快** → 选**指数模型** $y = a \cdot b^x$
- 数据增长越来越**慢**趋于饱和 → 选**对数模型** $y = a \cdot \ln x + b$
- 数据成**反比**趋势 → 选**反比例模型** $y = \frac{k}{x}$

**特性**：
- 拟合不要求函数图像经过所有数据点，只要求整体趋势最接近
- 常用**最小二乘法**衡量拟合的好坏：使所有数据点到拟合曲线的垂直距离之平方和最小
- 拟合效果用**相关系数** $R^2$ 衡量，$R^2$ 越接近1，拟合效果越好

**运算方式**：
- 观察散点图形状，判断大致趋势
- 选择适当的模型类型
- 用待定系数法或最小二乘法确定参数
- 计算拟合优度，评估模型质量

**定理**：
- **最小二乘法原理**：对于线性模型 $y = kx + b$，使 $\sum_{i=1}^n (y_i - kx_i - b)^2$ 最小的参数 $k$、$b$ 为最优拟合参数
  - 前提条件：数据点 $(x_i, y_i)$ 已给出
  - 使用场合：数据点超过参数个数时的最优拟合
  - 求解：对 $k$ 和 $b$ 分别求偏导并令其为零，解正规方程组

**公式**：
- 线性拟合的最小二乘估计：
$$k = \frac{\sum_{i=1}^n (x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^n (x_i - \bar{x})^2}, \quad b = \bar{y} - k\bar{x}$$
其中 $\bar{x} = \frac{1}{n}\sum_{i=1}^n x_i$，$\bar{y} = \frac{1}{n}\sum_{i=1}^n y_i$

**二级结论**：
- 当数据点恰好满足某函数关系时，拟合结果就是精确解；当数据有误差时，拟合结果是最佳近似
- 拟合前画散点图是重要步骤，可以直观判断应选择哪种函数模型
  - 推理过程：人眼对趋势的判断非常有效，散点图的形状直接反映了数据的内在规律，是模型选择的首要依据
- 对于指数型数据，取对数后化为线性数据再拟合，这种方法称为＂对数线性化＂
  - 推理过程：$y = a \cdot b^x \Rightarrow \ln y = \ln a + x \ln b$，令 $Y = \ln y$，则 $Y$ 与 $x$ 呈线性关系
