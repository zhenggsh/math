### 1.1 空间几何体的结构（编号1.1）

#### 本节概览

本节介绍多面体（棱柱、棱锥、棱台）和旋转体（圆柱、圆锥、圆台、球）的结构特征，以及简单组合体的构成方式。通过观察实物抽象几何特征，认识柱、锥、台、球的结构特征是后续计算的基础。

**知识脉络**：棱柱 → 棱锥 → 圆柱/圆锥/圆台 → 球 → 简单组合体

**与前后知识联系**：本节是本章的基础，结构特征的识别直接影响1.2节三视图的画法和1.3节表面积与体积的计算。

---

#### 知识点 1.1.1：棱柱的结构特征

**【重要性】A级 —— 高考常考，基础核心**

**定义**：棱柱是有两个面互相平行，其余各面都是四边形，并且每相邻两个四边形的公共边都互相平行的多面体。两个互相平行的面叫做**底面**，其余各面叫做**侧面**，相邻侧面的公共边叫做**侧棱**。

**特性**：
- 按底面边数分为三棱柱、四棱柱、五棱柱等
- 侧棱互相平行且相等
- 直棱柱的侧棱垂直于底面
- 正棱柱的底面是正多边形且侧棱垂直于底面
- 两个底面是全等的多边形，侧面都是平行四边形

**定理与公式**：
- 棱柱的侧面积：$S_{\text{侧}} = c \cdot l$（直棱柱，$c$为底面周长，$l$为侧棱长）
- 棱柱的体积：$V = S_{\text{底}} \cdot h$

**二级结论**：
- 平行于底面的截面与底面全等
- 过不相邻两条侧棱的截面是平行四边形
- $n$棱柱有$(n+2)$个面、$2n$个顶点、$3n$条棱

---

#### 知识点 1.1.2：棱锥的结构特征

**【重要性】A级 —— 高考常考，基础核心**

**定义**：棱锥是有一个面是多边形，其余各面是有一个公共顶点的三角形的多面体。多边形的面叫做**底面**，其余各面叫做**侧面**，公共顶点叫做**顶点**。

**特性**：
- 按底面边数分为三棱锥（四面体）、四棱锥、五棱锥等
- 正棱锥的底面是正多边形，且顶点在底面的正上方（顶点到底面中心的连线垂直于底面）
- 棱锥的体积是等底等高柱体体积的三分之一
- 正棱锥各侧面是全等的等腰三角形

**定理与公式**：
- 正棱锥侧面积：$S_{\text{侧}} = \frac{1}{2} c \cdot l'$（$c$为底面周长，$l'$为斜高）
- 棱锥体积：$V = \frac{1}{3} S_{\text{底}} \cdot h$

**二级结论**：
- 平行于底面的截面与底面相似，面积比等于相似比的平方
- 正棱锥的高、斜高和斜高在底面的射影构成直角三角形
- $n$棱锥有$(n+1)$个面、$(n+1)$个顶点、$2n$条棱

---

#### 知识点 1.1.3：圆柱、圆锥、圆台的结构特征

**【重要性】A级 —— 高考常考，基础核心**

**定义**：
- **圆柱**：以矩形的一边所在直线为旋转轴，其余三边旋转形成的面所围成的旋转体
- **圆锥**：以直角三角形的一条直角边所在直线为旋转轴旋转形成的旋转体
- **圆台**：用平行于圆锥底面的平面截去顶部后得到的几何体

**特性**：
- 圆柱的轴截面是矩形
- 圆锥的轴截面是等腰三角形
- 圆台的轴截面是等腰梯形
- 三者均可看作是由矩形或直角三角形绕轴旋转而成
- 圆柱的所有母线（侧面上平行于轴的线段）长度相等

**定理与公式**：
- 圆柱侧面积：$S_{\text{侧}} = 2\pi r l$
- 圆锥侧面积：$S_{\text{侧}} = \pi r l$
- 圆台侧面积：$S_{\text{侧}} = \pi (r_1 + r_2) l$
- 圆柱体积：$V = \pi r^2 h$
- 圆锥体积：$V = \frac{1}{3} \pi r^2 h$
- 圆台体积：$V = \frac{1}{3} \pi h (r_1^2 + r_2^2 + r_1 r_2)$

**二级结论**：
- 圆台可补成圆锥来研究，圆台的上底半径$r_1$、下底半径$r_2$和高$h$满足原圆锥与截去小圆锥的相似关系
- 圆锥侧面展开图是扇形，扇形弧长等于底面周长$2\pi r$，半径等于母线长$l$
- 圆台侧面展开图是扇环

---

#### 知识点 1.1.4：球的结构特征

**【重要性】A级 —— 高考常考，基础核心**

**定义**：球是以半圆的直径所在直线为旋转轴，半圆面旋转一周形成的旋转体。半圆的圆心叫做**球心**，半圆的半径叫做球的**半径**。

**特性**：
- 球面上任意一点到球心的距离都相等（等于半径）
- 球的截面是圆
- 球的大圆是经过球心的截面圆，半径等于球的半径
- 球的小圆是不经过球心的截面圆
- 球心到截面圆心的连线垂直于截面

**定理与公式**：
- 球心到截面的距离$d$、截面圆半径$r$、球半径$R$满足：$r^2 + d^2 = R^2$
- 球的表面积：$S = 4\pi R^2$
- 球的体积：$V = \frac{4}{3}\pi R^3$

**二级结论**：
- 球内接正方体：正方体棱长$a$与球半径$R$关系为$\sqrt{3}a = 2R$（正方体的体对角线等于球的直径）
- 球外切正方体：正方体棱长$a$与球半径$R$关系为$a = 2R$（球的直径等于正方体的棱长）
- 球内接长方体：长方体的体对角线等于球的直径
- 纬度圈半径：$r = R \cos\theta$（$\theta$为纬度角）

---

#### 知识点 1.1.5：简单组合体的结构特征

**【重要性】B级 —— 重要理解内容**

**定义**：简单组合体是由柱、锥、台、球等基本几何体组合而成的几何体。组合方式主要有**拼接**（两个或多个基本几何体拼接在一起）和**切割**（从基本几何体中切去一部分）两种。

**特性**：
- 识别组合体要分析其由哪些基本几何体构成
- 组合体的表面积要注意重合部分的面积不计入
- 组合体的体积等于各部分体积之和

**运算方式**：
- **分解法**：将组合体分解为若干个基本几何体，分别计算后求和或求差
- **补形法**：将不规则组合体补成规则几何体，再用减法求体积

**二级结论**：
- 两几何体拼接时，重合面的面积在总表面积中需减去两倍（两个几何体各失去一个面）
- 求组合体的表面积时，先看外表由哪些基本几何体的表面组成，再看内部有哪些面被遮住

### 1.2 空间几何体的三视图和直观图（编号1.2）

#### 本节概览

本节学习空间几何体在平面上的两种表示方法：三视图（从正面、侧面、上面三个方向观察得到的图形）和斜二测画法得到的直观图。三视图反映几何体的长宽高三维信息；斜二测画法是绘制空间图形直观图的标准方法。

**知识脉络**：三视图（投影规律与还原） → 斜二测画法（直观图绘制）

**与前后知识联系**：三视图是1.1节结构特征的平面表示，也是1.3节表面积体积计算的重要辅助工具。斜二测画法则服务于直观图的绘制。

---

#### 知识点 1.2.1：三视图

**【重要性】A级 —— 高考常考，核心技能**

**定义**：三视图是从三个不同方向观察空间几何体得到的平面图形，包括：
- **主视图**（正视图）：从正面看得到的图形
- **左视图**（侧视图）：从左面看得到的图形
- **俯视图**：从上面看得到的图形

**特性**：
- 主视图反映长和高
- 左视图反映宽和高
- 俯视图反映长和宽
- 三视图的投影规律：
  - **长对正**：主视图与俯视图长度相等，上下对正
  - **高平齐**：主视图与左视图高度相等，左右平齐
  - **宽相等**：左视图与俯视图宽度相等

**运算方式 —— 由三视图还原几何体**：
1. 从俯视图确定几何体的底面形状
2. 从主视图确定几何体的高度和各部分的上下位置
3. 从左视图验证宽度和前后位置关系
4. 综合三个视图想象几何体的空间形状

**二级结论**：
- 三视图中可见轮廓线用实线表示，不可见轮廓线用虚线表示
- 两个视图都是矩形 → 几何体可能是柱体
- 俯视图是圆 + 主视图是三角形 → 几何体是圆锥
- 由三视图求体积时，关键是准确还原几何体的形状和尺寸

---

#### 知识点 1.2.2：斜二测画法

**【重要性】A级 —— 高考常考，基础技能**

**定义**：斜二测画法是一种绘制空间图形直观图的方法。在已知图形中取互相垂直的$x$轴和$y$轴，画直观图时使$x'$轴与$y'$轴成$45°$（或$135°$），$x$轴方向长度不变，$y$轴方向长度变为原来的一半。

**特性**：
- 平行于$x$轴的线段长度不变
- 平行于$y$轴的线段长度减半
- 平行关系保持不变
- $z$轴方向（竖直方向）长度不变

**定理与公式**：
- 斜二测画法下平面图形的面积与原图形面积的关系：
  $$S_{\text{直观图}} = \frac{\sqrt{2}}{4} S_{\text{原图形}}$$
  即
  $$S_{\text{原图形}} = 2\sqrt{2} \cdot S_{\text{直观图}}$$

**运算方式 —— 斜二测画法步骤**：
1. 在原图中建立直角坐标系$xOy$
2. 画直观图时，$x'$轴水平，$y'$轴与$x'$轴成$45°$（或$135°$）
3. 平行于$x$轴的线段长度不变，平行于$y$轴的线段长度减半
4. 连线完成直观图

**二级结论**：
- 斜二测画法中，角度一般会发生改变（直角不再表现为直角）
- 平行性保持不变是斜二测画法的核心性质
- 由斜二测直观图还原原图形时，$y$方向长度需要翻倍

### 1.3 空间几何体的表面积与体积（编号1.3）

#### 本节概览

本节推导柱、锥、台、球的表面积和体积公式。表面积公式基于展开图，体积公式基于祖暅原理。球的表面积和体积公式是重点，组合体的表面积和体积计算要注意重合面。

**知识脉络**：柱体 → 锥体 → 台体 → 球 → 组合体

**与前后知识联系**：本节公式以1.1节的结构特征为基础，常与1.2节的三视图结合考查。体积计算思想（分解、补形、等积变换）是后续学习空间向量法的基础。

---

#### 知识点 1.3.1：柱体的表面积与体积

**【重要性】A级 —— 高考常考，核心公式**

**定义**：棱柱和圆柱的侧面积是展开图的面积；表面积等于侧面积与两个底面积之和。柱体的体积公式为$V = Sh$，其中$S$为底面积，$h$为高。

**特性**：
- 直棱柱侧面积$S_{\text{侧}} = c \cdot h$（$c$为底面周长）
- 圆柱侧面积$S_{\text{侧}} = 2\pi r l$
- 柱体体积$V = Sh$（棱柱和圆柱通用）
- 体积公式可由祖暅原理推导

**定理与公式**：

直棱柱：
- 侧面积：$S_{\text{侧}} = c \cdot h$
- 表面积：$S_{\text{表}} = S_{\text{侧}} + 2S_{\text{底}}$
- 体积：$V = S_{\text{底}} \cdot h$

圆柱：
- 侧面积：$S_{\text{侧}} = 2\pi r h$
- 表面积：$S_{\text{表}} = 2\pi r h + 2\pi r^2 = 2\pi r(h+r)$
- 体积：$V = \pi r^2 h$

**二级结论**：
- 祖暅原理：夹在两个平行平面间的两个几何体，如果被平行于这两个平面的任意平面所截得的截面面积都相等，那么这两个几何体的体积相等
- 柱体的体积只与底面积和高有关，与柱体的形状（直或斜）无关

---

#### 知识点 1.3.2：锥体的表面积与体积

**【重要性】A级 —— 高考常考，核心公式**

**定义**：棱锥和圆锥的侧面积是展开图的面积；表面积等于侧面积与底面积之和。锥体的体积公式为$V = \frac{1}{3}Sh$，其中$S$为底面积，$h$为高。

**特性**：
- 正棱锥侧面积$S_{\text{侧}} = \frac{1}{2} c \cdot l$（$c$为底面周长，$l$为斜高）
- 圆锥侧面积$S_{\text{侧}} = \pi r l$
- 锥体体积是等底等高柱体体积的三分之一

**定理与公式**：

正棱锥：
- 侧面积：$S_{\text{侧}} = \frac{1}{2} c \cdot l$
- 表面积：$S_{\text{表}} = S_{\text{侧}} + S_{\text{底}}$
- 体积：$V = \frac{1}{3} S_{\text{底}} \cdot h$

圆锥：
- 侧面积：$S_{\text{侧}} = \pi r l$（$l$为母线长）
- 表面积：$S_{\text{表}} = \pi r l + \pi r^2 = \pi r(l+r)$
- 体积：$V = \frac{1}{3} \pi r^2 h$
- 母线、高、底面半径关系：$l^2 = r^2 + h^2$

**二级结论**：
- 圆锥侧面展开图是扇形，扇形圆心角$\theta = \frac{r}{l} \times 360° = \frac{2\pi r}{l}$（弧度制）
- 等底等高的锥体体积是柱体体积的$\frac{1}{3}$
- 三棱锥（四面体）是最简单的多面体，任何一个多面体都可分割成若干个四面体

---

#### 知识点 1.3.3：台体的表面积与体积

**【重要性】A级 —— 高考常考，核心公式**

**定义**：棱台和圆台的侧面积是展开图的面积；表面积等于侧面积、上底面积和下底面积之和。台体的体积公式为$V = \frac{1}{3}h(S_1 + S_2 + \sqrt{S_1 S_2})$。

**特性**：
- 正棱台侧面积公式$S_{\text{侧}} = \frac{1}{2}(c_1 + c_2)l$
- 圆台侧面积$S_{\text{侧}} = \pi(r_1 + r_2)l$
- 台体体积可由大锥体减去小锥体得到
- 体积公式与梯形面积公式形式相似

**定理与公式**：

正棱台：
- 侧面积：$S_{\text{侧}} = \frac{1}{2}(c_1 + c_2) \cdot l$
- 表面积：$S_{\text{表}} = S_{\text{侧}} + S_1 + S_2$
- 体积：$V = \frac{1}{3}h(S_1 + S_2 + \sqrt{S_1 S_2})$

圆台：
- 侧面积：$S_{\text{侧}} = \pi(r_1 + r_2)l$
- 表面积：$S_{\text{表}} = \pi(r_1 + r_2)l + \pi r_1^2 + \pi r_2^2$
- 体积：$V = \frac{1}{3}\pi h(r_1^2 + r_2^2 + r_1 r_2)$
- 母线、高、半径关系：$l^2 = h^2 + (r_2 - r_1)^2$

**二级结论**：
- 台体可以补成锥体来研究，利用相似三角形求解
- 当$S_1 = S_2$时，台体体积公式退化为柱体体积公式$V = Sh$
- 当$S_1 = 0$时，台体体积公式退化为锥体体积公式$V = \frac{1}{3}Sh$

---

#### 知识点 1.3.4：球的表面积与体积

**【重要性】A级 —— 高考重点，必会公式**

**定义**：半径为$R$的球的表面积公式为$S = 4\pi R^2$，体积公式为$V = \frac{4}{3}\pi R^3$。球的表面积等于大圆面积的4倍。

**特性**：
- 球的表面积$S = 4\pi R^2$
- 球的体积$V = \frac{4}{3}\pi R^3$
- 球的表面积等于大圆面积$\pi R^2$的4倍

**定理与公式**：
- 球的表面积：$$S = 4\pi R^2$$
- 球的体积：$$V = \frac{4}{3}\pi R^3$$

**二级结论**：
- 球内接正方体：正方体棱长$a$与球半径$R$满足$\sqrt{3}a = 2R$
- 球外切正方体：正方体棱长$a$与球半径$R$满足$a = 2R$
- 球内接圆柱（轴截面为正方形）：圆柱底面半径$r$、高$h$满足$h = 2r$，且球半径$R = \sqrt{2}r$
- 球与正方体的各种位置关系中，关键是找到球的直径与正方体棱长或对角线的关系

---

#### 知识点 1.3.5：组合体的表面积与体积

**【重要性】B级 —— 重要应用**

**定义**：组合体的表面积等于各组成部分表面积之和减去重合部分的面积；组合体的体积等于各组成部分体积之和。

**特性**：
- 表面积计算要注意减去内部重合面的面积
- 体积计算可直接相加
- 通过分解或补形的方法将组合体转化为基本几何体

**运算方式**：
1. **分解法**：将组合体分解为若干个基本几何体
   - 体积：$V = V_1 + V_2 + \cdots + V_n$
   - 表面积：$S = S_1 + S_2 + \cdots + S_n - 2S_{\text{重合}}$
2. **补形法**：将不规则几何体补成规则几何体
   - $V = V_{\text{大}} - V_{\text{小}}$
3. **等积变换法**：利用祖暅原理或等底等高变换

**二级结论**：
- 表面积计算的关键是"外表面分析"：哪些面露在外面，哪些面被遮住
- 两个几何体拼接时，每拼接一次，总表面积减少两倍重合面的面积
- 对于挖去一部分的几何体，体积用减法，表面积可能需要加法（增加了内部新露出的面）

### 2.1 空间点线面及其位置关系（编号2.1）

#### 本节概览

本节引入平面的概念和表示方法，研究空间中点、直线、平面之间的各种位置关系，包括三个基本公理和它们的推论。三个公理是立体几何推理的基础。

**知识脉络**：平面的概念 → 三个公理 → 线线关系 → 线面关系 → 面面关系

**与前后知识联系**：本节是整个立体几何的基石，三个公理和位置关系的分类是2.2节平行关系和2.3节垂直关系的前提基础。

---

#### 知识点 2.1.1：平面的概念与表示

**【重要性】A级 —— 基础概念**

**定义**：几何里的平面是无限延展的，没有厚薄之分。平面是一个原始概念，不予严格定义，可以通过生活的实例（如平静的水面、桌面等）来理解。

**表示方法**：
- 用希腊字母表示：平面$\alpha$、平面$\beta$、平面$\gamma$
- 用平行四边形的对角顶点字母表示：平面$AC$
- 用平面内不共线的三个点表示：平面$ABC$

**特性**：
- 平面是无限延展的，没有边界
- 水平放置的平面通常画成平行四边形
- 点$A$在平面$\alpha$内记作$A \in \alpha$
- 点$A$不在平面$\alpha$内记作$A \notin \alpha$
- 直线$l$在平面$\alpha$内记作$l \subset \alpha$

**二级结论**：
- 平面没有面积的概念（无限延展），画图时只是画出平面的一部分
- 平面的"平"体现在：过平面内任意两点作直线，直线上所有点都在该平面内

---

#### 知识点 2.1.2：三个基本公理

**【重要性】A级 —— 立体几何推理的基石**

**定义**：

**公理1**：如果一条直线上的两点在一个平面内，那么这条直线在此平面内。
$$\text{若 } A \in \alpha, B \in \alpha, \text{且 } A \in l, B \in l, \text{ 则 } l \subset \alpha$$

**公理2**（不共线的三点确定一个平面）：过不在一条直线上的三点，有且只有一个平面。
$$\text{若 } A, B, C \text{ 三点不共线，则存在唯一的平面 } \alpha \text{ 使 } A, B, C \in \alpha$$

**公理3**：如果两个不重合的平面有一个公共点，那么它们有且只有一条过该点的公共直线。
$$\text{若 } P \in \alpha \text{ 且 } P \in \beta, \text{ 则 } \alpha \cap \beta = l \text{ 且 } P \in l$$

**特性**：
- **公理1**：判断直线在平面内的依据
- **公理2**：确定平面的依据（三点不共线）
- **公理3**：确定两平面交线的依据
- 三个公理是立体几何推理的基础

**公理2的三个推论**：

| 推论 | 内容 | 图示说明 |
|:---:|:---|:---|
| 推论1 | 经过一条直线和这条直线外一点，有且只有一个平面 | 直线$l$和点$A \notin l$确定平面$\alpha$ |
| 推论2 | 经过两条相交直线，有且只有一个平面 | 直线$a \cap b = P$确定平面$\alpha$ |
| 推论3 | 经过两条平行直线，有且只有一个平面 | 直线$a \parallel b$确定平面$\alpha$ |

**使用场合**：
- 证明点共面、线共面时用公理2及推论
- 证明点在交线上时用公理3
- 证明线在面内时用公理1

**二级结论**：
- 证明多点共线的方法：证明这些点都在两个平面的交线上
- 证明多线共点的方法：证明两条直线相交于一点，再证该点在其余直线上
- 证明点线共面的方法：先由部分元素确定一个平面，再证其余元素也在该平面内

---

#### 知识点 2.1.3：空间中直线与直线的位置关系

**【重要性】A级 —— 核心概念**

**定义**：空间中两条直线的位置关系有三种：

| 位置关系 | 共面情况 | 公共点个数 | 符号表示 |
|:---:|:---:|:---:|:---|
| 相交直线 | 在同一平面内 | 有且只有一个 | $a \cap b = P$ |
| 平行直线 | 在同一平面内 | 没有 | $a \parallel b$ |
| 异面直线 | 不同在任何一个平面内 | 没有 | $a$与$b$异面 |

**特性**：
- 异面直线是立体几何特有的概念（平面几何中不存在）
- 异面直线所成角的范围是$(0°, 90°]$

**定理 —— 等角定理**：
空间中如果两个角的两边分别对应平行，那么这两个角相等或互补。

**前提条件**：两个角在空间中，它们的两边分别对应平行。

**使用场合**：证明空间角相等时。

**运算方式 —— 求异面直线所成的角**：
1. **平移法**：过空间任一点作两条异面直线的平行线，两条平行线所成的锐角（或直角）即为所求
2. **向量法**：设两直线的方向向量分别为$\vec{a}$和$\vec{b}$，则
   $$\cos\theta = \frac{|\vec{a} \cdot \vec{b}|}{|\vec{a}| \cdot |\vec{b}|}$$
   （取绝对值是因为异面直线所成角的范围是$(0°, 90°]$）

**二级结论**：
- 异面直线的判定：过平面外一点与平面内一点的直线，和平面内不经过该点的直线是异面直线
- 求异面直线所成角时，常用"一角三角形"法：平移后构造直角三角形，用三角函数求解

---

#### 知识点 2.1.4：空间中直线与平面的位置关系

**【重要性】A级 —— 核心概念**

**定义**：空间中直线与平面的位置关系有三种：

| 位置关系 | 公共点个数 | 符号表示 | 备注 |
|:---:|:---:|:---|:---|
| 直线在平面内 | 无数个 | $a \subset \alpha$ | 直线上所有点都在平面内 |
| 直线与平面相交 | 有且只有一个 | $a \cap \alpha = A$ | 直线与平面有唯一公共点$A$ |
| 直线与平面平行 | 没有 | $a \parallel \alpha$ | 直线与平面没有公共点 |

**特性**：
- 线面平行和线面相交统称为**直线在平面外**，记作$a \not\subset \alpha$
- 用符号表示：$a \subset \alpha$（线在面内）、$a \cap \alpha = A$（线面相交于点$A$）、$a \parallel \alpha$（线面平行）

**定理**：
- 过平面外一点有且只有一条直线与已知平面平行（在过该点且与已知平面平行的平面内）
- 过平面外一点有无数条直线与已知平面相交

**二级结论**：
- 判断线面位置关系的方法：看直线上有多少个点在平面内——无数个（线在面内）、一个（线面相交）、零个（线面平行）
- 如果一条直线和平面不平行，那么它们必相交

---

#### 知识点 2.1.5：空间中平面与平面的位置关系

**【重要性】A级 —— 核心概念**

**定义**：空间中两个平面的位置关系有两种：

| 位置关系 | 公共点情况 | 符号表示 | 备注 |
|:---:|:---:|:---|:---|
| 两个平面平行 | 没有公共点 | $\alpha \parallel \beta$ | 两平面无交线 |
| 两个平面相交 | 有一条公共直线 | $\alpha \cap \beta = l$ | 两平面交于直线$l$ |

**特性**：
- 两个不重合的平面要么平行要么相交
- 公理3是确定两平面交线的依据
- 用符号表示：$\alpha \parallel \beta$（面面平行）、$\alpha \cap \beta = l$（面面相交于直线$l$）

**定理**：
- 经过平面外一点有且只有一个平面与已知平面平行
- 两个平面相交，交线是唯一的

**二级结论**：
- 两个平行平面同时与第三个平面相交，所得的两条交线平行（面面平行的性质定理，见2.2.4）
- 夹在两个平行平面间的平行线段长度相等
- 两个平行平面间的距离处处相等

### 2.2 直线、平面平行的判定及其性质（编号2.2）

#### 本节概览

本节研究直线与平面平行、平面与平面平行的判定定理和性质定理。转化思想是核心：线线平行$\rightarrow$线面平行$\rightarrow$面面平行。高考必考内容。

**知识脉络**：线面平行的判定 → 线面平行的性质 → 面面平行的判定 → 面面平行的性质

**与前后知识联系**：以2.1节的线面、面面位置关系为基础，为2.3节垂直关系的学习提供方法借鉴。

---

#### 知识点 2.2.1：直线与平面平行的判定定理

**【重要性】A级 —— 高考必考，核心定理**

**定义**：直线与平面平行的判定定理：

> 平面外一条直线与此平面内的一条直线平行，则该直线与此平面平行。

**简记为**：线线平行$\rightarrow$线面平行

**符号表示**：
$$\left.\begin{array}{l} a \not\subset \alpha \\ b \subset \alpha \\ a \parallel b \end{array}\right\} \Rightarrow a \parallel \alpha$$

**前提条件**：
1. 直线$a$在平面$\alpha$外（$a \not\subset \alpha$）—— 必须有此条件
2. 直线$b$在平面$\alpha$内（$b \subset \alpha$）
3. 两条直线平行（$a \parallel b$）

**三个条件缺一不可**！若缺少条件1（$a \not\subset \alpha$），则$a$可能在平面内，不能得出平行的结论。

**使用场合**：证明直线与平面平行时最常用的方法。

**证明步骤**：
1. 在已知平面内找到一条直线
2. 证明该直线与已知直线平行
3. 说明已知直线在平面外
4. 由判定定理得出结论

**二级结论**：
- 线面平行判定定理的本质是"降维"：将线面平行问题转化为线线平行问题
- 找平行线时常用的方法：中位线、平行四边形对边、相似三角形等

---

#### 知识点 2.2.2：直线与平面平行的性质定理

**【重要性】A级 —— 高考必考，核心定理**

**定义**：直线与平面平行的性质定理：

> 一条直线与一个平面平行，则过这条直线的任一平面与此平面的交线与该直线平行。

**简记为**：线面平行$\rightarrow$线线平行

**符号表示**：
$$\left.\begin{array}{l} a \parallel \alpha \\ a \subset \beta \\ \alpha \cap \beta = b \end{array}\right\} \Rightarrow a \parallel b$$

**前提条件**：
1. 直线$a$与平面$\alpha$平行（$a \parallel \alpha$）
2. 直线$a$在平面$\beta$内（$a \subset \beta$）
3. 平面$\alpha$与$\beta$相交于直线$b$（$\alpha \cap \beta = b$）

**使用场合**：已知线面平行，需要得到线线平行时使用。

**操作方法**：过已知直线作辅助平面与已知平面相交，交线即为所求的平行线。

**二级结论**：
- 性质定理给出了由线面平行推出线线平行的方法
- 线面平行时，直线上任意一点到平面的距离都相等
- 线面平行的性质定理常与判定定理配合使用，实现"线线$\leftrightarrow$线面"的相互转化

---

#### 知识点 2.2.3：平面与平面平行的判定定理

**【重要性】A级 —— 高考必考，核心定理**

**定义**：平面与平面平行的判定定理：

> 一个平面内的两条相交直线与另一个平面平行，则这两个平面平行。

**简记为**：线面平行$\rightarrow$面面平行

**符号表示**：
$$\left.\begin{array}{l} a \subset \alpha, b \subset \alpha \\ a \cap b = P \\ a \parallel \beta, b \parallel \beta \end{array}\right\} \Rightarrow \alpha \parallel \beta$$

**前提条件**：
1. 两条直线$a$、$b$都在平面$\alpha$内（$a \subset \alpha, b \subset \alpha$）
2. 两条直线相交（$a \cap b = P$）—— **必须是相交直线，两条平行直线不行**
3. 两条直线都与平面$\beta$平行（$a \parallel \beta, b \parallel \beta$）

**使用场合**：证明两个平面平行时使用。

**推论**：垂直于同一直线的两个平面平行。

**二级结论**：
- 也可证明一个平面内两条相交直线分别与另一平面内两条直线平行
- 平行于同一平面的两个平面互相平行
- 证明面面平行的常用思路：先证线面平行，再用判定定理

---

#### 知识点 2.2.4：平面与平面平行的性质定理

**【重要性】A级 —— 高考必考，核心定理**

**定义**：平面与平面平行的性质定理：

> 如果两个平行平面同时与第三个平面相交，那么它们的交线平行。

**简记为**：面面平行$\rightarrow$线线平行

**符号表示**：
$$\left.\begin{array}{l} \alpha \parallel \beta \\ \gamma \cap \alpha = a \\ \gamma \cap \beta = b \end{array}\right\} \Rightarrow a \parallel b$$

**前提条件**：
1. 平面$\alpha$与$\beta$平行（$\alpha \parallel \beta$）
2. 平面$\gamma$与$\alpha$相交于直线$a$（$\gamma \cap \alpha = a$）
3. 平面$\gamma$与$\beta$相交于直线$b$（$\gamma \cap \beta = b$）

**使用场合**：已知面面平行，需要得到线线平行时使用。

**二级结论**：
- 两个平行平面内的任意一对平行线段相等
- 经过平面外一点有且只有一个平面与已知平面平行
- 夹在两个平行平面间的平行线段长度相等
- 两个平行平面间的距离处处相等

**平行关系转化链**：

```mermaid
graph LR
    A［线线平行］ -->|判定定理| B［线面平行］
    B -->|判定定理| C［面面平行］
    C -->|性质定理| A
    B -->|性质定理| A
    C -->|性质定理| B
```

### 2.3 直线、平面垂直的判定及其性质（编号2.3）

#### 本节概览

**知识脉络**：线面垂直的判定与性质 → 面面垂直的判定与性质 → 线面角 → 二面角

**与前后知识联系**：以2.1节的位置关系和2.2节的平行关系为基础，垂直关系是高考立体几何大题的核心考点。

---

#### 知识点 2.3.1：直线与平面垂直的判定定理

**【重要性】A级 —— 高考必考，核心定理**

**定义**：直线与平面垂直的判定定理：

> 一条直线与一个平面内的两条相交直线都垂直，则该直线与此平面垂直。

**简记为**：线线垂直$\rightarrow$线面垂直

**符号表示**：
$$\left.\begin{array}{l} a \subset \alpha, b \subset \alpha \\ a \cap b = P \\ l \perp a, l \perp b \end{array}\right\} \Rightarrow l \perp \alpha$$

**前提条件**：
1. 直线$a$、$b$都在平面$\alpha$内（$a \subset \alpha, b \subset \alpha$）
2. 两条直线相交（$a \cap b = P$）—— **必须是两条相交直线**
3. 直线$l$分别与$a$、$b$垂直（$l \perp a, l \perp b$）

**注意**：一条直线垂直于平面内的无数条平行直线**不能**推出线面垂直。

**使用场合**：证明直线与平面垂直时最常用的方法。

**二级结论**：
- 如果一条直线垂直于一个平面，那么它垂直于平面内的所有直线
- 线面垂直的定义可作为性质使用：$l \perp \alpha$，$a \subset \alpha$ $\Rightarrow$ $l \perp a$
- 证明线面垂直时，常通过证明线线垂直来实现，有时需要多次转化

---

#### 知识点 2.3.2：直线与平面垂直的性质定理

**【重要性】A级 —— 高考必考，核心定理**

**定义**：直线与平面垂直的性质定理：

> 垂直于同一个平面的两条直线平行。

**简记为**：线面垂直$\rightarrow$线线平行

**符号表示**：
$$\left.\begin{array}{l} a \perp \alpha \\ b \perp \alpha \end{array}\right\} \Rightarrow a \parallel b$$

**前提条件**：
1. 直线$a$垂直于平面$\alpha$（$a \perp \alpha$）
2. 直线$b$垂直于平面$\alpha$（$b \perp \alpha$）

**使用场合**：已知两条直线都垂直于同一平面，需要证明它们平行时使用。

**相关结论**：
- 点到平面的距离定义：过该点作平面的垂线段的长度
- 两条平行线中一条垂直于平面，另一条也垂直于该平面

**二级结论**：
- 此性质可用于证明两条直线平行
- 唯一性：过一点有且只有一条直线与已知平面垂直
- 平行线等距：两条平行线到同一平面的距离相等

---

#### 知识点 2.3.3：平面与平面垂直的判定定理

**【重要性】A级 —— 高考必考，核心定理**

**定义**：平面与平面垂直的判定定理：

> 一个平面过另一个平面的垂线，则这两个平面垂直。

**简记为**：线面垂直$\rightarrow$面面垂直

**符号表示**：
$$\left.\begin{array}{l} l \perp \beta \\ l \subset \alpha \end{array}\right\} \Rightarrow \alpha \perp \beta$$

**前提条件**：
1. 直线$l$垂直于平面$\beta$（$l \perp \beta$）
2. 直线$l$在平面$\alpha$内（$l \subset \alpha$）

**关键理解**：一个平面内有一条直线垂直于另一个平面，则两个平面垂直。

**使用场合**：证明两个平面垂直时使用。

**二级结论**：
- 证明面面垂直的基本思路：先证线面垂直，再证该直线在另一平面内
- 如果两个平面垂直，那么在一个平面内垂直于交线的直线垂直于另一个平面（性质定理，见2.3.4）

---

#### 知识点 2.3.4：平面与平面垂直的性质定理

**【重要性】A级 —— 高考必考，核心定理**

**定义**：平面与平面垂直的性质定理：

> 两个平面垂直，则一个平面内垂直于交线的直线与另一个平面垂直。

**简记为**：面面垂直$\rightarrow$线面垂直

**符号表示**：
$$\left.\begin{array}{l} \alpha \perp \beta \\ \alpha \cap \beta = l \\ a \subset \alpha \\ a \perp l \end{array}\right\} \Rightarrow a \perp \beta$$

**前提条件**：
1. 平面$\alpha$与$\beta$垂直（$\alpha \perp \beta$）
2. 两平面交于直线$l$（$\alpha \cap \beta = l$）
3. 直线$a$在平面$\alpha$内（$a \subset \alpha$）
4. 直线$a$垂直于交线$l$（$a \perp l$）

**关键词**：一个平面内、垂直于交线、垂直于另一平面。

**使用场合**：面面垂直条件下证明线面垂直的重要工具。

**操作方法**：常需作辅助线——在其中一个平面内过某点作交线的垂线。

**垂直关系转化链**：

```mermaid
graph LR
    D［线线垂直］ -->|判定定理| E［线面垂直］
    E -->|判定定理| F［面面垂直］
    F -->|性质定理| E
    E -->|定义| D
```

**二级结论**：
- 此性质是面面垂直条件下证明线面垂直的重要工具
- 常需作辅助线构造垂线
- 面面垂直 + 线垂直于交线 $\Rightarrow$ 线垂直于另一平面

---

#### 知识点 2.3.5：直线与平面所成的角

**【重要性】A级 —— 高考常考，重点计算**

**定义**：平面的一条斜线和它在平面上的射影所成的锐角，叫做这条直线和这个平面所成的角。

特殊情况：
- 当直线与平面垂直时，所成角为$90°$
- 当直线与平面平行或在平面内时，所成角为$0°$

**特性**：
- 线面角的范围是$[0°, 90°]$
- 斜线：与平面相交但不垂直的直线
- 射影：过斜线上一点作平面的垂线，垂足与斜足的连线即为射影

**运算方式 —— 求线面角的步骤**：
1. **找垂线**：过直线上某点（或空间某点）作平面的垂线，确定垂足
2. **确定射影**：连接垂足与斜足（直线与平面的交点），得到射影
3. **构造直角三角形**：斜线、垂线、射影构成直角三角形
4. **用三角函数求解**：设线面角为$\theta$，则$\sin\theta = \frac{h}{l}$（$h$为垂线长，$l$为斜线长）

**替代方法 —— 等体积法**：
1. 用等体积变换求出点到平面的距离$h$
2. 求出斜线长度$l$
3. $\sin\theta = \frac{h}{l}$

**公式**：
$$\sin\theta = \frac{\text{点到平面的距离}}{\text{斜线段长度}}$$

**二级结论**：
- 斜线与平面所成角是斜线与平面内所有直线所成角中最小的
- 等体积法在垂线不易直接作出时特别有用
- 设直线方向向量为$\vec{s}$，平面法向量为$\vec{n}$，则
  $$\sin\theta = \frac{|\vec{s} \cdot \vec{n}|}{|\vec{s}| \cdot |\vec{n}|}$$

---

#### 知识点 2.3.6：二面角

**【重要性】A级 —— 高考难点，重点计算**

**定义**：
- **二面角**：从一条直线出发的两个半平面所组成的图形叫做二面角。这条直线叫做二面角的**棱**。
- **二面角的平面角**：以二面角的棱上任一点为端点，在两个半平面内分别作垂直于棱的两条射线，这两条射线所成的角叫做二面角的平面角。

**特性**：
- 二面角的平面角的范围是$[0°, 180°]$
- 两个平面垂直等价于二面角为$90°$

**运算方式 —— 求二面角的方法**：

**方法一：定义法（在棱上取点作垂线）**
1. 在二面角的棱上任取一点
2. 分别在两个半平面内过该点作棱的垂线
3. 两条垂线所成的角即为二面角的平面角

**方法二：垂面法（作垂直于棱的平面）**
1. 过空间一点作一个垂直于棱的平面
2. 该平面与两个半平面的交线所成的角即为二面角的平面角

**方法三：面积射影法**
$$\cos\theta = \frac{S_{\text{射影}}}{S_{\text{原}}}$$

其中$\theta$为二面角的平面角，$S_{\text{原}}$是原图形的面积，$S_{\text{射影}}$是该图形在另一平面上的射影面积。

**方法四：三垂线定理法**
1. 在一个半平面内取一点，作另一平面的垂线
2. 由垂足向棱作垂线，连接后构造二面角的平面角

**前提条件**：面积射影法要求原图形所在平面与另一平面所成的角就是所求二面角。

**使用场合**：
- 定义法适用于棱上容易取点的情况
- 面积射影法适用于图形面积容易计算的情况
- 三垂线定理法是最常用的方法

**二级结论**：
- 二面角的平面角大小与棱上取点的位置无关
- 两个平面垂直 $\Leftrightarrow$ 它们所成的二面角是直二面角（$90°$）
- 面积射影法在求斜面与底面所成角时特别方便

### 3.1 直线的倾斜角与斜率（编号3.1）

#### 本节概览

本节引入直线的倾斜角和斜率的概念，研究斜率的计算公式和倾斜角与斜率之间的关系。倾斜角描述直线的"倾斜程度"，斜率则将这种几何描述代数化，为建立直线方程奠定基础。

**知识脉络**：倾斜角 → 斜率定义 → 斜率公式

**与前后知识联系**：本节是3.2节直线方程的基础，斜率是建立各种形式直线方程的核心要素；3.3节的交点和距离问题也依赖斜率的计算。

---

#### 知识点 3.1.1：直线的倾斜角

**【重要性】A级 —— 基础概念**

**定义**：当直线$l$与$x$轴相交时，取$x$轴作为基准，$x$轴正向与直线$l$向上方向之间所成的角$\alpha$叫做直线$l$的倾斜角。当直线$l$与$x$轴平行或重合时，规定倾斜角为$0°$。

**特性**：
- 倾斜角的取值范围是$[0°, 180°)$
- 每条直线都有唯一的倾斜角
- 倾斜角反映了直线的倾斜程度
- 倾斜角为$90°$时直线垂直于$x$轴

**运算方式**：
- 由直线的方向确定倾斜角：$x$轴正方向逆时针旋转到与直线重合所需的最小正角
- 倾斜角的取值范围限制了斜率的变化规律

**二级结论**：
- 水平直线的倾斜角为$0°$
- 竖直直线的倾斜角为$90°$
- 倾斜角越大（$\alpha < 90°$时），直线越陡
- 两条直线的倾斜角相等 $\Leftrightarrow$ 两条直线平行（或重合）

---

#### 知识点 3.1.2：直线的斜率

**【重要性】A级 —— 核心概念**

**定义**：一条直线的倾斜角$\alpha$的正切值叫做这条直线的斜率，记作$k = \tan\alpha$。斜率反映了直线相对于$x$轴的倾斜程度。

**特性**：
- $k = \tan\alpha$
- $\alpha \in [0°, 90°)$时，$k \geq 0$（斜率为正，直线上升）
- $\alpha \in (90°, 180°)$时，$k < 0$（斜率为负，直线下降）
- $\alpha = 90°$时斜率不存在（直线垂直于$x$轴）
- 斜率越大直线越陡（$\alpha < 90°$时）

**运算方式**：
- 已知倾斜角求斜率：$k = \tan\alpha$（$\alpha \neq 90°$）
- 已知斜率求倾斜角：$\alpha = \arctan k$（$k \geq 0$时$\alpha \in [0°, 90°)$，$k < 0$时$\alpha \in (90°, 180°)$）

**二级结论**：
- 斜率不存在 $\Leftrightarrow$ 直线垂直于$x$轴（$\alpha = 90°$）
- $k = 0$ $\Leftrightarrow$ 水平直线
- $k > 0$时，$|k|$越大直线越陡
- $k < 0$时，$|k|$越大直线越陡（越接近竖直）

---

#### 知识点 3.1.3：斜率公式

**【重要性】A级 —— 核心公式**

**定义**：经过两点$P_1(x_1, y_1)$和$P_2(x_2, y_2)$（$x_1 \neq x_2$）的直线的斜率公式为：

$$k = \frac{y_2 - y_1}{x_2 - x_1}$$

**特性**：
- 斜率公式与两点的顺序无关（分子分母同时变号，比值不变）
- $x_1 = x_2$时斜率不存在（直线垂直于$x$轴）
- 斜率公式是建立直线方程的基础
- 斜率也可表示直线的方向：斜率为$k$的直线的方向向量可以取$(1, k)$

**前提条件**：两点横坐标不相等（$x_1 \neq x_2$）。若$x_1 = x_2$，则直线垂直于$x$轴，倾斜角为$90°$，斜率不存在。

**使用场合**：已知直线上两点求斜率；判断两条直线是否平行或垂直。

**二级结论**：
- 三点共线判定：若$k_{AB} = k_{BC}$，则$A$、$B$、$C$三点共线
- 直线的方向向量为$\vec{d} = (1, k)$或$(x_2-x_1, y_2-y_1)$
- 直线的法向量（垂直于直线的向量）为$\vec{n} = (k, -1)$或$(A, B)$（一般式中）

---

### 3.2 直线的方程（编号3.2）

#### 本节概览

本节推导直线的几种方程形式，包括点斜式、斜截式、两点式、截距式和一般式。五种方程形式各有适用条件，一般式$Ax+By+C=0$是最通用的形式，根据已知条件选择合适的方程形式。

**知识脉络**：点斜式 → 斜截式 → 两点式 → 截距式 → 一般式

**与前后知识联系**：3.1节的斜率是各种方程形式的基础，3.3节的交点坐标和距离公式以一般式为主要工具。

---

#### 知识点 3.2.1：点斜式方程

**【重要性】A级 —— 核心方程**

**定义**：直线过点$P_0(x_0, y_0)$且斜率为$k$，则直线的点斜式方程为：

$$y - y_0 = k(x - x_0)$$

**特性**：
- 已知直线上一点和斜率即可写出方程
- 点斜式是推导其他方程形式的基础

**前提条件**：直线斜率存在（$k \in \mathbb{R}$）。

**限制**：不能表示垂直于$x$轴的直线（此时斜率不存在）。

**使用场合**：已知直线上一点和斜率时使用。

**二级结论**：
- 点斜式中的$(x_0, y_0)$是直线上任意一点，可以替换为直线上其他已知点
- 如果直线过点$(x_0, y_0)$且倾斜角为$\alpha$，则方程为$y - y_0 = \tan\alpha \cdot (x - x_0)$

---

#### 知识点 3.2.2：斜截式方程

**【重要性】A级 —— 核心方程**

**定义**：直线斜率为$k$，在$y$轴上的截距为$b$，则直线的斜截式方程为：

$$y = kx + b$$

**特性**：
- 斜截式是点斜式的特例（过点$(0, b)$）
- $k$为斜率，$b$为$y$轴截距

**前提条件**：直线斜率存在。

**限制**：不能表示垂直于$x$轴的直线。

**使用场合**：已知斜率和$y$轴截距时使用；与一次函数$y = kx + b$直接对应。

**二级结论**：
- 一次函数$y = kx + b$的图像就是斜率为$k$、截距为$b$的直线
- $b > 0$时直线与$y$轴交于正半轴，$b < 0$时交于负半轴，$b = 0$时过原点
- 斜截式是判断两直线平行和垂直最方便的形式

---

#### 知识点 3.2.3：两点式与截距式方程

**【重要性】A级 —— 常用方程**

**两点式定义**：直线过两点$P_1(x_1, y_1)$和$P_2(x_2, y_2)$（$x_1 \neq x_2$，$y_1 \neq y_2$），两点式方程为：

$$\frac{y - y_1}{y_2 - y_1} = \frac{x - x_1}{x_2 - x_1}$$

**截距式定义**：直线在$x$轴、$y$轴上的截距分别为$a$、$b$（$a \neq 0$，$b \neq 0$），截距式方程为：

$$\frac{x}{a} + \frac{y}{b} = 1$$

**特性**：
- 两点式不能表示平行于坐标轴的直线（$x_1 = x_2$或$y_1 = y_2$时无意义）
- 截距式不能表示过原点的直线（$a = 0$或$b = 0$时无意义）和平行于坐标轴的直线
- 截距可正可负

**前提条件**：
- 两点式：$x_1 \neq x_2$且$y_1 \neq y_2$
- 截距式：$a \neq 0$且$b \neq 0$

**使用场合**：
- 两点式：已知直线上两个不同的点时使用
- 截距式：已知直线在两坐标轴上的截距时使用

**二级结论**：
- 截距式中$x$轴截距$a$是直线与$x$轴交点的横坐标（令$y = 0$求得）
- $y$轴截距$b$是直线与$y$轴交点的纵坐标（令$x = 0$求得）
- 已知三角形两边在坐标轴上时，截距式特别方便

---

#### 知识点 3.2.4：一般式方程

**【重要性】A级 —— 最通用的方程形式**

**定义**：任何直线都可以表示为$Ax + By + C = 0$的形式（$A$、$B$不同时为$0$），这就是直线的一般式方程。

**特性**：
- 一般式可以表示所有直线（包括垂直于$x$轴的直线）
- $A = 0$时表示水平直线$y = -\frac{C}{B}$
- $B = 0$时表示竖直直线$x = -\frac{C}{A}$
- 直线的法向量为$\vec{n} = (A, B)$

**各形式与一般式的互化**：

| 形式 | 化为一般式 | 条件 |
|:---:|:---|:---|
| 点斜式 | $kx - y + (y_0 - kx_0) = 0$ | $k$存在 |
| 斜截式 | $kx - y + b = 0$ | $k$存在 |
| 两点式 | 展开整理 | $x_1 \neq x_2$，$y_1 \neq y_2$ |
| 截距式 | $\frac{x}{a} + \frac{y}{b} = 1 \Rightarrow bx + ay - ab = 0$ | $a \neq 0$，$b \neq 0$ |

**公式**：
- 斜率：$k = -\frac{A}{B}$（$B \neq 0$时）
- $x$轴截距：$a = -\frac{C}{A}$（$A \neq 0$时）
- $y$轴截距：$b = -\frac{C}{B}$（$B \neq 0$时）

**二级结论**：
- 法向量$(A, B)$垂直于直线，方向向量可以取$(B, -A)$或$(-B, A)$
- 一般式在距离公式、判断位置关系时使用最方便
- 将方程化为一般式时，通常让$A \geq 0$，且$A$、$B$、$C$没有公因数

---

### 3.3 直线的交点坐标与距离公式（编号3.3）

#### 本节概览

本节研究两条直线的交点坐标的求解方法，推导点到直线的距离公式、两条平行直线间的距离公式，以及两点间的距离公式。交点坐标即联立方程组的解，点到直线距离公式是高考重点。

**知识脉络**：两直线交点 → 两点距离 → 点线距离 → 平行线距离

**与前后知识联系**：以3.2节的各种方程形式为工具，距离公式是后续研究圆与直线位置关系的基础。

---

#### 知识点 3.3.1：两条直线的交点坐标

**【重要性】A级 —— 核心方法**

**定义**：两条直线$l_1: A_1x + B_1y + C_1 = 0$和$l_2: A_2x + B_2y + C_2 = 0$的交点坐标是联立方程组的解。

$$\left\{\begin{array}{l} A_1x + B_1y + C_1 = 0 \\ A_2x + B_2y + C_2 = 0 \end{array}\right.$$

**位置关系判定**：

| 方程组解的情况 | 位置关系 | 代数条件 |
|:---:|:---:|:---|
| 有唯一解 | 相交 | $\frac{A_1}{A_2} \neq \frac{B_1}{B_2}$（即$A_1B_2 \neq A_2B_1$） |
| 无解 | 平行 | $\frac{A_1}{A_2} = \frac{B_1}{B_2} \neq \frac{C_1}{C_2}$ |
| 无数解 | 重合 | $\frac{A_1}{A_2} = \frac{B_1}{B_2} = \frac{C_1}{C_2}$ |

**用斜率判断**（化为斜截式$y = k_1x + b_1$和$y = k_2x + b_2$）：
- $k_1 \neq k_2$：两直线相交
- $k_1 = k_2$且$b_1 \neq b_2$：两直线平行
- $k_1 = k_2$且$b_1 = b_2$：两直线重合

**过两直线交点的直线系方程**：
$$A_1x + B_1y + C_1 + \lambda(A_2x + B_2y + C_2) = 0$$
（$\lambda \in \mathbb{R}$，不包括直线$l_2$本身）

**前提条件**：直线系方程表示过两直线交点的所有直线（除$l_2$外）。

**使用场合**：求过定点的一系列直线时使用直线系方程可简化计算。

**二级结论**：
- 判断两直线是否相交，只需看系数行列式是否为零：$\begin{vmatrix} A_1 & B_1 \\ A_2 & B_2 \end{vmatrix} \neq 0$
- 若两直线垂直：$A_1A_2 + B_1B_2 = 0$（或$k_1 \cdot k_2 = -1$，斜率都存在时）

---

#### 知识点 3.3.2：两点间的距离公式

**【重要性】A级 —— 基础公式**

**定义**：平面上两点$P_1(x_1, y_1)$和$P_2(x_2, y_2)$间的距离公式为：

$$|P_1P_2| = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$$

**特性**：
- 距离公式可由勾股定理推导
- 当两点在$x$轴上时距离为$|x_2 - x_1|$
- 当两点在$y$轴上时距离为$|y_2 - y_1|$

**使用场合**：求两点间距离；判断三角形形状；求圆的方程等。

**公式推导**：

```mermaid
graph LR
    A［P₁［x₁，y₁］］ -->|水平距离| B［x₂−x₁］
    A -->|竖直距离| C［y₂−y₁］
    B -->|勾股定理| D［d＝√［［x₂−x₁］²＋［y₂−y₁］²］］
    C --> D
```

**二级结论**：
- 到原点距离：点$P(x, y)$到原点的距离为$\sqrt{x^2 + y^2}$
- 距离公式是推导点到直线距离公式的基础
- 中点坐标公式：$P_1(x_1, y_1)$和$P_2(x_2, y_2)$的中点为$M\left(\frac{x_1+x_2}{2}, \frac{y_1+y_2}{2}\right)$
- 三角形重心坐标：$G\left(\frac{x_1+x_2+x_3}{3}, \frac{y_1+y_2+y_3}{3}\right)$

---

#### 知识点 3.3.3：点到直线的距离公式

**【重要性】A级 —— 高考重点，核心公式**

**定义**：点$P(x_0, y_0)$到直线$l: Ax + By + C = 0$的距离公式为：

$$d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$$

**特性**：
- 使用距离公式前要将直线方程化为一般式
- 分子取绝对值保证距离非负
- 分母$\sqrt{A^2+B^2}$是直线法向量$(A, B)$的模

**前提条件**：直线方程必须是一般式$Ax + By + C = 0$。

**使用场合**：
- 求点到直线的距离
- 判断点与直线的位置关系（$d = 0$在直线上，$d > 0$在直线外）
- 求三角形面积（面积 = $\frac{1}{2} \times$底$\times$高）
- 判断直线与圆的位置关系（比较圆心到直线的距离与半径）

**公式推导思路**：
1. 过点$P$作直线的垂线，求出垂足坐标
2. 用两点间距离公式求$P$到垂足的距离
3. 化简后得到距离公式

**二级结论**：
- 点到直线距离是求三角形面积等问题的常用工具
- 若点在直线上，则$Ax_0 + By_0 + C = 0$（分子为零）
- 原点$(0, 0)$到直线$Ax + By + C = 0$的距离为$d = \frac{|C|}{\sqrt{A^2 + B^2}}$

---

#### 知识点 3.3.4：两条平行直线间的距离

**【重要性】A级 —— 常用公式**

**定义**：两条平行直线$l_1: Ax + By + C_1 = 0$和$l_2: Ax + By + C_2 = 0$间的距离公式为：

$$d = \frac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}$$

**特性**：
- 使用公式前必须保证两条直线方程中$x$、$y$的系数相同（$A$和$B$分别相等）
- 平行线间距离处处相等

**前提条件**：两条直线平行，即$\frac{A_1}{A_2} = \frac{B_1}{B_2}$（系数成比例）。使用公式时需先将方程化为$x$、$y$系数相同的形式。

**使用场合**：求两条平行直线间的距离；判断两条直线是否等距。

**验证方法**：可在一条直线上任取一点，求该点到另一条直线的距离来验证。

**二级结论**：
- 如果两条平行直线的方程系数不完全相同（如$2x + 3y + 1 = 0$和$4x + 6y - 5 = 0$），需要先将系数统一（化为$4x + 6y + 2 = 0$和$4x + 6y - 5 = 0$），再用公式
- 与两条平行直线等距离的直线方程：$Ax + By + \frac{C_1 + C_2}{2} = 0$

### 4.1 圆的方程（编号4.1）

#### 本节概览

本节推导圆的标准方程和一般方程，研究如何根据已知条件确定圆的方程。标准方程$(x-a)^2+(y-b)^2=r^2$直接反映圆心和半径，一般式$x^2+y^2+Dx+Ey+F=0$需配方转化为标准式。

**知识脉络**：圆的定义 → 标准方程 → 一般方程 → 互化方法

**与前后知识联系**：圆的方程是3.2节直线方程的自然延伸，4.2节直线与圆的位置关系判断需要用到圆的方程和点到直线的距离公式。

---

#### 知识点 4.1.1：圆的标准方程

**【重要性】A级 —— 核心方程**

**定义**：圆心为$(a, b)$，半径为$r$的圆的标准方程为：

$$(x - a)^2 + (y - b)^2 = r^2$$

**特性**：
- 标准方程直接反映圆心$(a, b)$和半径$r$
- 圆心在原点时方程为$x^2 + y^2 = r^2$
- 已知圆心和半径可直接写出方程

**点与圆的位置关系判断**：

设点$P(x_0, y_0)$，圆$(x-a)^2 + (y-b)^2 = r^2$，比较点$P$到圆心的距离$d$与半径$r$：

| 位置关系 | 代数条件 | 几何意义 |
|:---:|:---:|:---|
| 点在圆内 | $(x_0-a)^2 + (y_0-b)^2 < r^2$ | $d < r$ |
| 点在圆上 | $(x_0-a)^2 + (y_0-b)^2 = r^2$ | $d = r$ |
| 点在圆外 | $(x_0-a)^2 + (y_0-b)^2 > r^2$ | $d > r$ |

**运算方式 —— 求圆的方程的方法**：
1. **直接法**：已知圆心和半径，直接写出标准方程
2. **待定系数法**：设圆的标准方程为$(x-a)^2+(y-b)^2=r^2$，根据条件列出方程组求解$a$、$b$、$r$

**使用场合**：已知圆心和半径时使用标准方程。

**二级结论**：
- 以$A(x_1, y_1)$和$B(x_2, y_2)$为直径端点的圆的方程：
  $$(x - x_1)(x - x_2) + (y - y_1)(y - y_2) = 0$$
- 圆的对称性：圆$(x-a)^2+(y-b)^2=r^2$关于圆心$(a,b)$对称，也关于过圆心的任意直线对称
- 圆上一点$P(x_0, y_0)$处的切线垂直于过该点的半径

---

#### 知识点 4.1.2：圆的一般方程

**【重要性】A级 —— 核心方程**

**定义**：方程$x^2 + y^2 + Dx + Ey + F = 0$（$D^2 + E^2 - 4F > 0$）表示圆，其圆心为$\left(-\frac{D}{2}, -\frac{E}{2}\right)$，半径$r = \frac{\sqrt{D^2 + E^2 - 4F}}{2}$。

**特性**：
- 一般式化为标准式需配方
- $D^2 + E^2 - 4F > 0$时表示圆
- $D^2 + E^2 - 4F = 0$时表示一个点$\left(-\frac{D}{2}, -\frac{E}{2}\right)$
- $D^2 + E^2 - 4F < 0$时无图形（虚圆）
- 一般式中$x^2$和$y^2$系数相等且没有$xy$项

**配方过程**：
$$x^2 + y^2 + Dx + Ey + F = 0$$
$$(x^2 + Dx + \frac{D^2}{4}) + (y^2 + Ey + \frac{E^2}{4}) = \frac{D^2}{4} + \frac{E^2}{4} - F$$
$$(x + \frac{D}{2})^2 + (y + \frac{E}{2})^2 = \frac{D^2 + E^2 - 4F}{4}$$

**运算方式 —— 两种方程形式的互化**：

| 标准方程 → 一般方程 | 一般方程 → 标准方程 |
|:---|:---|
| 展开整理 | 配方 |
| $(x-a)^2+(y-b)^2=r^2$ | $x^2+y^2+Dx+Ey+F=0$ |
| $\Rightarrow x^2+y^2-2ax-2by+(a^2+b^2-r^2)=0$ | $\Rightarrow \left(x+\frac{D}{2}\right)^2+\left(y+\frac{E}{2}\right)^2=\frac{D^2+E^2-4F}{4}$ |

对应关系：$D = -2a$，$E = -2b$，$F = a^2+b^2-r^2$

**使用场合**：已知圆上三点坐标时常用一般方程设未知数求解。

**二级结论**：
- 判断$x^2 + y^2 + Dx + Ey + F = 0$是否表示圆：看$x^2$和$y^2$系数是否相等且不为零，以及是否没有$xy$项
- 用待定系数法求圆的方程时，若已知圆心位置关系用标准方程更方便；若已知圆上三点用一般方程更方便

---

### 4.2 直线与圆的位置关系（编号4.2）

#### 本节概览

本节研究直线与圆的三种位置关系（相交、相切、相离）的判断方法，以及圆的切线方程和弦长问题的求解。几何法（比较$d$与$r$）通常更简便。

**知识脉络**：位置关系判断 → 切线方程 → 弦长问题

**与前后知识联系**：以4.1节的圆的方程和3.3.3节的点到直线距离公式为基础，是高考解析几何的核心考点。

---

#### 知识点 4.2.1：直线与圆位置关系的判断

**【重要性】A级 —— 高考必考**

**定义**：直线与圆的位置关系有三种，设圆心到直线的距离为$d$，圆的半径为$r$：

| 位置关系 | 几何条件 | 代数条件（联立方程看判别式） | 公共点个数 |
|:---:|:---:|:---:|:---:|
| 相离 | $d > r$ | $\Delta < 0$ | 0个 |
| 相切 | $d = r$ | $\Delta = 0$ | 1个 |
| 相交 | $d < r$ | $\Delta > 0$ | 2个 |

**两种判断方法**：

**方法一：几何法**（通常更简便）
1. 求圆心到直线的距离$d$
2. 比较$d$与半径$r$的大小
3. $d > r \Rightarrow$ 相离；$d = r \Rightarrow$ 相切；$d < r \Rightarrow$ 相交

**方法二：代数法**
1. 联立直线与圆的方程
2. 消元得到一元二次方程
3. 看判别式$\Delta$：$\Delta > 0$相交、$\Delta = 0$相切、$\Delta < 0$相离

**前提条件**：几何法需要知道圆心坐标和半径；代数法需要能联立方程求解。

**使用场合**：
- 几何法适用于圆心和半径已知的情况
- 代数法适用于参数较多或几何关系不明显的情况

**二级结论**：
- 几何法通常更简便，避免了求解方程的复杂计算
- 直线与圆相切的条件在求切线方程时特别有用
- 过圆外一点可以向圆引两条切线；过圆上一点只能引一条切线；过圆内一点不能引切线

---

#### 知识点 4.2.2：圆的切线方程

**【重要性】A级 —— 高考常考**

**定义**：过圆上一点的切线是与圆只有一个公共点的直线，且该直线垂直于过切点的半径。

**圆的切线方程公式**：

**圆上一点的切线**：

圆$x^2 + y^2 = r^2$上一点$P(x_0, y_0)$的切线方程为：

$$xx_0 + yy_0 = r^2$$

圆$(x-a)^2 + (y-b)^2 = r^2$上一点$P(x_0, y_0)$的切线方程为：

$$(x-a)(x_0-a) + (y-b)(y_0-b) = r^2$$

**特性**：
- 圆上一点的切线垂直于该点与圆心的连线（半径）
- 圆心到切线的距离等于半径

**运算方式 —— 求过圆外一点的切线**：
1. 设切线方程为点斜式$y - y_0 = k(x - x_0)$（注意斜率不存在的情况）
2. 利用圆心到切线的距离等于半径，列方程求$k$
3. 一般得到两个$k$值，对应两条切线

**前提条件**：点在圆上时可直接用公式；点在圆外时需用待定斜率法求解。

**使用场合**：求圆的切线方程；与圆相切的直线问题。

**二级结论**：
- 圆上一点的切线公式记忆技巧：将标准方程中的$x^2 \to xx_0$，$y^2 \to yy_0$，$x \to \frac{x+x_0}{2}$，$y \to \frac{y+y_0}{2}$
- 过圆外一点$P(x_0, y_0)$的切线长（切点到$P$的距离）：
  $$L = \sqrt{(x_0-a)^2 + (y_0-b)^2 - r^2}$$
  即$L = \sqrt{d^2 - r^2}$，其中$d$为点$P$到圆心的距离

---

#### 知识点 4.2.3：圆的弦长问题

**【重要性】A级 —— 高考常考**

**定义**：直线与圆相交时，两个交点之间的线段叫做**弦**。

**弦长公式**：

$$|AB| = 2\sqrt{r^2 - d^2}$$

其中$r$为圆的半径，$d$为圆心到直线的距离。

**特性**：
- 弦长公式可由垂径定理推导
- 过圆内一点的最短弦垂直于该点与圆心的连线
- 最长弦是直径（此时$d = 0$，$|AB| = 2r$）
- 弦的中点与圆心的连线垂直于弦

**定理 —— 垂径定理**：垂直于弦的直径平分这条弦，并且平分弦所对的两条弧。

**前提条件**：直线与圆相交（$d < r$）。

**使用场合**：求直线截圆所得弦的长度；已知弦长求圆心到直线的距离。

**运算方式**：
1. 求圆心到直线的距离$d$
2. 代入弦长公式$|AB| = 2\sqrt{r^2 - d^2}$

**二级结论**：
- 弦的中垂线必过圆心
- 弦越长，圆心到弦的距离越短；弦长最大为直径
- 若弦$AB$的中点为$M$，则$OM \perp AB$（$O$为圆心）
- 过圆内一点$P$的最短弦：与$OP$垂直的弦，弦长$= 2\sqrt{r^2 - |OP|^2}$

### 4.3 空间直角坐标系（编号4.3）

#### 本节概览

本节建立空间直角坐标系，确定空间中点的坐标表示，推导空间两点间的距离公式。空间直角坐标系是沟通数与形的桥梁，为后续学习空间向量奠定基础。

**知识脉络**：空间直角坐标系的建立 → 点的坐标表示 → 空间两点距离公式

**与前后知识联系**：空间直角坐标系是平面直角坐标系的自然推广，空间两点距离公式是平面距离公式的推广，为选修内容中的空间向量奠定基础。

---

#### 知识点 4.3.1：空间直角坐标系

**【重要性】A级 —— 基础概念**

**定义**：以空间中一个定点$O$为原点，以三条两两垂直的直线为坐标轴（$x$轴、$y$轴、$z$轴），建立空间直角坐标系$Oxyz$。空间中任意一点$P$的位置可用有序实数组$(x, y, z)$表示。

**坐标确定方法**：
1. 过点$P$作$xOy$平面的垂线，垂足为$P'$
2. $P'$在$xOy$平面内的坐标为$(x, y)$
3. $P$到$xOy$平面的有向距离为$z$
4. 则点$P$的坐标为$(x, y, z)$

**特性**：
- 三个坐标平面：$xOy$平面（$z = 0$）、$yOz$平面（$x = 0$）、$xOz$平面（$y = 0$）
- 坐标轴上的点有两个坐标为0
- 坐标平面上的点有一个坐标为0

**特殊位置的点的坐标特征**：

| 位置 | 坐标特征 | 示例 |
|:---:|:---:|:---:|
| $x$轴上 | $(x, 0, 0)$ | $(2, 0, 0)$ |
| $y$轴上 | $(0, y, 0)$ | $(0, -1, 0)$ |
| $z$轴上 | $(0, 0, z)$ | $(0, 0, 3)$ |
| $xOy$平面上 | $(x, y, 0)$ | $(1, 2, 0)$ |
| $yOz$平面上 | $(0, y, z)$ | $(0, 1, -2)$ |
| $xOz$平面上 | $(x, 0, z)$ | $(-1, 0, 2)$ |
| 原点 | $(0, 0, 0)$ | $(0, 0, 0)$ |

**点的对称坐标**：

| 对称类型 | 对称点坐标 |
|:---:|:---:|
| 关于$x$轴对称 | $(x, -y, -z)$ |
| 关于$y$轴对称 | $(-x, y, -z)$ |
| 关于$z$轴对称 | $(-x, -y, z)$ |
| 关于$xOy$平面对称 | $(x, y, -z)$ |
| 关于$yOz$平面对称 | $(-x, y, z)$ |
| 关于$xOz$平面对称 | $(x, -y, z)$ |
| 关于原点对称 | $(-x, -y, -z)$ |

**使用场合**：确定空间中点的位置；建立空间图形与代数方程的联系。

**二级结论**：
- 空间直角坐标系是沟通数与形的桥梁
- 右手系规则：$x$轴、$y$轴、$z$轴成右手螺旋关系
- 空间中两点$P(x_1, y_1, z_1)$和$Q(x_2, y_2, z_2)$的中点坐标为$\left(\frac{x_1+x_2}{2}, \frac{y_1+y_2}{2}, \frac{z_1+z_2}{2}\right)$

#### 知识点 4.3.2：空间两点间的距离公式

**【重要性】A级 —— 核心公式**

**定义**：空间中两点$P_1(x_1, y_1, z_1)$和$P_2(x_2, y_2, z_2)$间的距离公式为：

$$|P_1P_2| = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2 + (z_2 - z_1)^2}$$

**特性**：
- 空间距离公式是平面距离公式的推广
- 到原点距离：$|OP| = \sqrt{x^2 + y^2 + z^2}$

**使用场合**：求空间中两点间距离；判断空间图形的形状和大小；求空间中的轨迹方程。

**公式推导**：

```mermaid
graph LR
    A［P₁［x₁，y₁，z₁］］ -->|x方向差| B［Δx＝x₂−x₁］
    A -->|y方向差| C［Δy＝y₂−y₁］
    A -->|z方向差| D［Δz＝z₂−z₁］
    B -->|两次勾股定理| E［d＝√［Δx²＋Δy²＋Δz²］］
    C --> E
    D --> E
```

**推导过程**：
1. 先求$P_1$和$P_2$在$xOy$平面上的射影$P_1'(x_1, y_1, 0)$和$P_2'(x_2, y_2, 0)$之间的距离：
   $$|P_1'P_2'| = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$$
2. 再过$P_1$作$P_2P_2'$的垂线，构造直角三角形
3. 由勾股定理：
   $$|P_1P_2|^2 = |P_1'P_2'|^2 + |z_2-z_1|^2$$
4. 代入得：
   $$|P_1P_2| = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2 + (z_2-z_1)^2}$$

**二级结论**：
- 空间中点$P(x, y, z)$到原点的距离：$|OP| = \sqrt{x^2 + y^2 + z^2}$
- 空间中两点间的中点坐标公式：$M\left(\frac{x_1+x_2}{2}, \frac{y_1+y_2}{2}, \frac{z_1+z_2}{2}\right)$
- 空间中的球面方程：$(x-a)^2 + (y-b)^2 + (z-c)^2 = R^2$，球心$(a, b, c)$，半径$R$
- 空间中三角形重心坐标：$G\left(\frac{x_1+x_2+x_3}{3}, \frac{y_1+y_2+y_3}{3}, \frac{z_1+z_2+z_3}{3}\right)$
